
# ODBC - 调用失败。（错误 3146）

您尝试使用 ODBC 连接对 ODBC 数据源中的数据执行操作。当 ODBC 数据源位于网络驱动器上，而您未连接到网络时，可发生此错误。请确保网络可用，然后再次尝试该操作。
