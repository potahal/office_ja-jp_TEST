
# ListLevel.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ListLevel** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[ListLevel](0cd152cb-6c25-50cb-7c1d-8b6d9734505b.md)** object.
