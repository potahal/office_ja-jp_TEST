
# MailMessage.Delete Method (Word)

Deletes the specified mail message.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents a **[MailMessage](d0109969-27f7-0180-c56d-5b49a3f0171b.md)** object.
