
# SeriesCollection.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[SeriesCollection](785d61ff-96c9-b9b0-ed98-e992d9adeda6.md)** object.
