
# OMathMatCols.Item Method (Word)

Returns an  **OMathMatCol** object that represents the specified item in the collection.


## Syntax

 _表达式_. **Item**( ** _Index_** )

 _表达式_ An expression that returns an **[OMathMatCols](b56ee426-56bd-6588-ebe9-898f4bfbba0c.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Long**|Specifies the ordinal position of the object within the collection.|

### Return Value

OMathMatCol
