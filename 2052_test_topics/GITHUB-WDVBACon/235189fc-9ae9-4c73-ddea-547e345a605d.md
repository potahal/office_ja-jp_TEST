
# Pane.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _表达式_. **Index**

 _表达式_ Required. A variable that represents a **[Pane](4a0c2690-d9d2-4e34-fef4-cc41365f5251.md)** object.
