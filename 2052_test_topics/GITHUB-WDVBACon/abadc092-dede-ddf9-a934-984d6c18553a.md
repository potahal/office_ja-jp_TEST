
# Pane.Next Property (Word)

Returns a  **Pane** object that represents the next document pane in the collection. Read-only.


## Syntax

 _表达式_. **Next**

 _表达式_ A variable that represents a **[Pane](4a0c2690-d9d2-4e34-fef4-cc41365f5251.md)** object.
