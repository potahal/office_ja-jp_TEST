
# LegendKey.Height Property (Word)

Returns the height, in points, of the object. Read-only  **Double**.


## Syntax

 _表达式_. **Height**

 _表达式_ A variable that represents a **[LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.
