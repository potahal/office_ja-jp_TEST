
# ListEntry.Delete Method (Word)

Deletes the specified list entry.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents a **[ListEntry](ea9e8276-45d6-8b11-fd86-4944f582bb80.md)** object.
