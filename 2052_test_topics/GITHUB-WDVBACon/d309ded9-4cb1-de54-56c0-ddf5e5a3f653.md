
# TextInput.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TextInput** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[TextInput](d7f6531a-4da2-ccc4-29b3-ad79ca7b18de.md)** object.
