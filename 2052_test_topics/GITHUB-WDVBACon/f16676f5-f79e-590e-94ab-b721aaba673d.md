
# Footnote.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Footnote** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Footnote](877340c4-14f9-4560-eaf8-2c6482a1ade8.md)** object.
