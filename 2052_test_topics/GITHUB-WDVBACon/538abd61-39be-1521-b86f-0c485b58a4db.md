
# TableOfFigures.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TableOfFigures** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[TableOfFigures](0a93d274-cd2e-3d65-48bc-b8fea84ff268.md)** collection.
