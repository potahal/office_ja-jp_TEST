
# StyleSheet.Move Method (Word)

Moves a style sheet's order of precedence.


## Syntax

 _表达式_. **Move**( ** _Precedence_** )

 _表达式_ Required. A variable that represents a **[StyleSheet](5e576ff8-c458-f5bd-730d-9db827c4f76e.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Precedence_|必需|**WdStyleSheetPrecedence**|The precedence level.|
