
# Index.RightAlignPageNumbers Property (Word)

 **True** if page numbers are aligned with the right margin in an index. Read/write **Boolean**.


## Syntax

 _表达式_. **RightAlignPageNumbers**

 _表达式_ Required. A variable that represents an **[Index](6a2aab98-485b-01c3-8d9b-9e108b455e22.md)** object.
