
# Hyperlinks.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Hyperlinks** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Hyperlinks](25801753-737f-9219-6a14-6531eb2ca699.md)** collection.
