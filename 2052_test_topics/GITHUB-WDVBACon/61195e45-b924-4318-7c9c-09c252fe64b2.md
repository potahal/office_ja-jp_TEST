
# Task.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Task](8802fcd5-0947-2ea0-308a-376077633e34.md)** object.
