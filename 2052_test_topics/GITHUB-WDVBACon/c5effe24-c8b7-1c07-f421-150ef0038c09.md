
# Rows.RelativeHorizontalPosition Property (Word)

Specifies the relative horizontal position of a group of rows. Read/write  **[WdRelativeHorizontalPosition](155eefdf-bbf8-9895-1bc0-64fcda4e2635.md)**.


## Syntax

 _表达式_. **RelativeHorizontalPosition**

 _表达式_ An expression that represents a **[Rows](cd83d0ef-f743-1886-54de-497017c5f542.md)** object.
