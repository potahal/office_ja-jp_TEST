
# Endnotes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Endnotes** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents an **[Endnotes](32676579-dd41-e83d-a305-fcc2b7cb4f64.md)** collection.
