
# Language.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Language** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Language](0acc4a42-b4c2-a415-0e38-a049b085dc86.md)** object.
