
# Subdocument.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Subdocument](ed966369-34f6-ef0c-d6d6-4c86baff4793.md)** object.
