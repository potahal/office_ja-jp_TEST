
# ChartGroup.CategoryCollection Method (Word)

Returns all the visible categories in the chart group, or the specified visible category.


## Syntax

 _表达式_. **CategoryCollection** _(Index)_

 _表达式_ A variable that represents a **ChartGroup** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
|||||
| _Index_|可选|VARIANT|The index number in the visible category collection of the category to return.|

### Return value

 **OBJECT**
