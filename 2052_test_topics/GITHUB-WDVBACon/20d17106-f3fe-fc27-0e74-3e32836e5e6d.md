
# Source.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Source** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ A variable that represents a **[Source](f90108a8-6432-a700-86ce-7b8f9e9c034b.md)** object.
