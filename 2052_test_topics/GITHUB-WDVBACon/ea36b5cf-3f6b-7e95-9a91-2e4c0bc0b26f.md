
# Category.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Category](5485ae39-fbcf-b18f-b1f9-945e220ecd2a.md)** object.
