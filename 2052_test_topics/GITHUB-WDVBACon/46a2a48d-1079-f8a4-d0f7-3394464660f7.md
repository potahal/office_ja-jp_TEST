
# AxisTitle.Height Property (Word)

Returns the height, in points, of the object. Read-only.


## Syntax

 _表达式_. **Height**

 _表达式_ A variable that represents an **[AxisTitle](ec746a05-40df-95cc-c017-40ef150504cf.md)** object.


### Return Value

Double
