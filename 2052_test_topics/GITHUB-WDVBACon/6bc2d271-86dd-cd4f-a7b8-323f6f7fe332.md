
# Chart.SetBackgroundPicture Method (Word)

Sets the background graphic for a chart.


## Syntax

 _表达式_. **SetBackgroundPicture**( ** _FileName_** )

 _表达式_ A variable that represents a **[Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _FileName_|必需|**String**|The name of the file for the graphic.|
