
# Sections.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Sections** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection.
