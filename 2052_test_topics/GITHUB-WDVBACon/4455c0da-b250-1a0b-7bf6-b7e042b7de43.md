
# Language.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Language](0acc4a42-b4c2-a415-0e38-a049b085dc86.md)** object.
