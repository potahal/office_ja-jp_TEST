
# ChartGroup.BinsOverflowEnabled Property (Word)

Specifies whether a bin for values above the [BinsOverflowValue](http://msdn.microsoft.com/library/411856a7-ac17-e9eb-35bd-c851c0cfdfdc%28Office.15%29.aspx) is enabled. Read/write **Boolean**.在 Office 2016 中引入。


## Syntax

 _表达式_. **BinsOverflowEnabled**

 _表达式_ A variable that represents a **ChartGroup** object.
