
# OMathMatRows.Count Property (Word)

Returns the number of items in the  **OMathMatRows** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[OMathMatRows](beeb74aa-23ba-b9da-1f24-65c91fb3dc2c.md)** object.
