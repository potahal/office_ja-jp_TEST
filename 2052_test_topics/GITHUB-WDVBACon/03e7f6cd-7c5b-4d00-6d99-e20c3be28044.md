
# PlotArea.Height Property (Word)

Returns or sets the height, in points, of the object. Read/write  **Double**.


## Syntax

 _表达式_. **Height**

 _表达式_ A variable that represents a **[PlotArea](72d30767-7cfc-3063-0b49-f9fbc129a52c.md)** object.
