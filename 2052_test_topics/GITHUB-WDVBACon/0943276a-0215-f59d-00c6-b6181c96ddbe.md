
# Options.FrenchReform Property (Word)

Returns or sets a  **[WdFrenchSpeller](0942c739-63e8-fe99-64a5-31ce71aed7c0.md)** constant that represents which spelling dictionary to use for regions of text with language formatting set to French. Read/write.


## Syntax

 _表达式_. **FrenchReform**

 _表达式_ An expression that returns a **[Options](873b7b99-3fe1-fd89-9ece-a9355cb827dc.md)** object.
