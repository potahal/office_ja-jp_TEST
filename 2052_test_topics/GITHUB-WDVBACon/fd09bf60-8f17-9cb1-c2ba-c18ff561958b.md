
# Variable.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Variable](e6a75f54-6f91-75b4-7ca0-9be302e8dbe0.md)** object.
