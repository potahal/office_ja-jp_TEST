
# ListEntries.Count Property (Word)

Returns a  **Long** that represents the number of list entries in the collection. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ Required. A variable that represents a **[ListEntries](cfd3c706-5b69-338f-b104-e12526b89f47.md)** collection.
