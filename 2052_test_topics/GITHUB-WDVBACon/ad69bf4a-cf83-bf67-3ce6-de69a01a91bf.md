
# InlineShapes.Count Property (Word)

Returns a  **Long** that represents the number of inline shapes in the collection. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ Required. A variable that represents an **[InlineShapes](88c632b2-80de-c96a-8879-a98461b38bd0.md)** collection.
