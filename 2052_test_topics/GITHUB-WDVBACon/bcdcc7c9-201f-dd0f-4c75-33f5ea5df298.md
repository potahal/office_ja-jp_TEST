
# OMaths.Count Property (Word)

Returns the number of items in the  **OMaths** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[OMaths](5e185b0f-b0c9-16f8-3056-c1114dadd3e0.md)** object.
