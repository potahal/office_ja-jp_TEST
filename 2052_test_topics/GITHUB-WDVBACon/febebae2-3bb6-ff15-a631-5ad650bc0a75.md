
# FillFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **FillFormat** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[FillFormat](39205d07-9e37-1be1-ec4a-93ba8bac2f26.md)** object.
