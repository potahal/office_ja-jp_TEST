
# Chart.HasTitle Property (Word)

 **True** if the axis or chart has a visible title. Read/write **Boolean**.


## Syntax

 _表达式_. **HasTitle**

 _表达式_ A variable that represents a **[Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.


## Remarks

A chart title is represented by a  **[ChartTitle](fc8ca540-0a29-123b-2fdf-b16aaa1f940c.md)** object.
