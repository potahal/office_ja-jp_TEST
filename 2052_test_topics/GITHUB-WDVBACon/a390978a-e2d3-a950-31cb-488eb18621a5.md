
# Bookmark.Copy Method (Word)

Copies a bookmark to the new bookmark specified in the Name argument, and returns a  **Bookmark** object.


## Syntax

 _表达式_. **Copy**( ** _Name_** )

 _表达式_ Required. A variable that represents a **[Bookmark](be6b0c7b-60ca-97e7-ef19-6de335da3197.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Name_|必需|**String**|The name of the new bookmark.|
