
# CanvasShapes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CanvasShapes** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[CanvasShapes](f4b37915-7fde-2a21-0df0-fc3c97983900.md)** collection.
