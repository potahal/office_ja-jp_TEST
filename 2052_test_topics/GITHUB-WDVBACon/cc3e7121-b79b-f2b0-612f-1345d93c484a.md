
# Row.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _表达式_. **Index**

 _表达式_ Required. A variable that represents a **[Row](38a05858-829a-ea5c-ce63-7f7343bf7b88.md)** object.
