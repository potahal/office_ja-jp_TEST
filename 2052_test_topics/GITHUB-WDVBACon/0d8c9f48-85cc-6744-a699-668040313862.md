
# 内置对话框参数列表 (Word)

许多内置的 Word 中的对话框有您可能想要设置的选项。若要设置或返回与 Word 对话框中关联的属性，使用 Visual Basic 的等效属性和方法。例如，如果您想要打印的文档，使用 VBA  **[PrintOut](bad3cd20-39e7-11b8-4a55-244bfcb6df24.md)** 方法。下面的代码将打印当前文档使用 **打印** 对话框中的默认设置。但是，如果您不想使用打印对话框中的默认设置，您可以使用与 **PrintOut** 方法相关的参数。


```
Sub PrintCurrentDocument() 
 ActiveDocument.PrintOut 
End Sub
```


虽然我们推荐您使用 VBA 关键字获取或设置对话框选项的值，但许多内置的 Word 对话框也具有可用于获取或设置对话框值的参数。有关详细信息，请参阅[显示内置 Word 对话框](abe465f9-09a1-72ea-2e2d-9de14fc02434.md)。
