
# ChartGroup.BinWidthValue Property (Word)

Specifies the number of points in each range. Read/write  **Double**.在 Office 2016 中引入。


## Syntax

 _表达式_. **BinWidthValue**

 _表达式_ A variable that represents a **ChartGroup** object.
