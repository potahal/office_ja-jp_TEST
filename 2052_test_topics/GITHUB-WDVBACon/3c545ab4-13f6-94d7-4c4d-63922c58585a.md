
# List.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **List** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[List](2c3dae28-447a-af48-2966-e19ae75ab6c2.md)** object.
