
# Series.Copy Method (Word)

If the series has a picture fill, copies the picture to the Clipboard.


## Syntax

 _表达式_. **Copy**

 _表达式_ A variable that represents a **[Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object.
