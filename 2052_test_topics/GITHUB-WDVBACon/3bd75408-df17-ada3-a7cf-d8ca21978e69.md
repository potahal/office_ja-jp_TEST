
# Task.Height Property (Word)

Returns or sets the height of the specified task window. Read/write  **Long**.


## Syntax

 _表达式_. **Height**

 _表达式_ An expression that represents a **[Task](8802fcd5-0947-2ea0-308a-376077633e34.md)** object.
