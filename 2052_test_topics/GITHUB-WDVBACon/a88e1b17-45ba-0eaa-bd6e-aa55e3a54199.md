
# Font.ThreeD Property (Word)

Returns a [ThreeDFormat](d397e780-a53d-0cc3-7a02-b40397253e91.md) object that contains 3-D effect formatting properties for the specified font. Read-only.


## Syntax

 _表达式_. **ThreeD**

 _表达式_ An expression that returns a **[Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.
