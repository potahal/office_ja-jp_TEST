
# Application.ActiveProtectedViewWindow Property (Word)

Returns a [ProtectedViewWindow](d77e80e7-c54e-5954-1586-dacd3c9f7434.md) object that represents the active protected view window. Read-only.


## Syntax

 _表达式_. **ActiveProtectedViewWindow**

 _表达式_ An expression that returns a **Application** object.


## Example

The following code example displays the caption text for the active protected view window.


```
MsgBox ActiveProtectedViewWindow.Caption 

```
