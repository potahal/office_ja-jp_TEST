
# PlotArea.ClearFormats Method (Word)

Clears the formatting of the object.


## Syntax

 _表达式_. **ClearFormats**

 _表达式_ A variable that represents a **[PlotArea](72d30767-7cfc-3063-0b49-f9fbc129a52c.md)** object.
