
# Axis.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _表达式_. **Parent**

 _表达式_ A variable that represents an **[Axis](3a7ad7d8-d397-a79a-eb6a-a5f0822cbe5d.md)** object.
