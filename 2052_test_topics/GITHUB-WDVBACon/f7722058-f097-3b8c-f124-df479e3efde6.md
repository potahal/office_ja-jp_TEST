
# Application.LoadMasterList Method (Word)

Loads a bibliography source file.


## Syntax

 _表达式_. **LoadMasterList**( ** _FileName_** )

 _表达式_ An expression that returns an **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _FileName_|必需|**String**|The path and file name of the bibliography source file.|
