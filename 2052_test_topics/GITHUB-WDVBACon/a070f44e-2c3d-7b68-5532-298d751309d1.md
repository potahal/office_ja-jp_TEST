
# OMathArgs.Count Property (Word)

Returns the number of items in the  **OMathArgs** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[OMathArgs](5e4d542b-11c3-8cb8-be2a-5b990e777290.md)** object.
