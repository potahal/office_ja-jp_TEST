
# Selection.Sections Property (Word)

Returns a  **[Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection that represents the sections in the specified selection. Read-only.


## Syntax

 _表达式_. **Sections**

 _表达式_ A variable that represents a **[Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](28f76384-f495-9640-a7c8-10ada3fac727.md).
