
# Range.SetListLevel Method (Word)

Sets the list level for one or more items in a numbered list.


## Syntax

 _表达式_. **SetListLevel**( ** _Level_** )

 _表达式_ An expression that returns a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Level_|必需|**Integer**|A number that indicates the new list level.|
