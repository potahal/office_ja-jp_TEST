
# ChartCategory.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ChartCategory** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ A variable that represents a **ChartCategory** object.


## Property value

 **OBJECT**
