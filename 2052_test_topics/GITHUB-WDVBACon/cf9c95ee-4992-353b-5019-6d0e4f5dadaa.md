
# Hyperlink.Delete Method (Word)

Deletes the specified hyperlink.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents a **[Hyperlink](af785a9e-081a-e359-705f-04f490304e2e.md)** object.
