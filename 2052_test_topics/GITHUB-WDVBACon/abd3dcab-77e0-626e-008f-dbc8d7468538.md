
# RecentFile.Delete Method (Word)

Deletes the specified file on the list of recent files.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents a **[RecentFile](c8d7a06d-c340-2d35-d4a9-5d0cd4a07aab.md)** object.
