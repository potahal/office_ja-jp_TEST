
# Window.Application Property (Word)

Returns an  **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object that represents the Microsoft OfficeWord application.


## Syntax

 _表达式_. **Application**

 _表达式_ Required. A variable that represents a **[Window](d92f83f9-ae44-56c0-4584-7a9359253c6d.md)** object.


## Remarks

The Microsoft Visual Basic  **CreateObject** and **GetObject** functions give you access to an OLE Automation object from a Visual Basic for Applications (VBA) project.
