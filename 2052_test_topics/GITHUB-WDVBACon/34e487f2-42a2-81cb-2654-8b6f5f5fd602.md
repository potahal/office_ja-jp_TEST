
# DataLabels.Orientation Property (Word)

Returns or sets the text orientation. Read/write  **Long**.


## Syntax

 _表达式_. **Orientation**

 _表达式_ A variable that represents a **[DataLabels](a7676f18-b1f2-1e11-9489-863cb85c1669.md)** object.


## Remarks

The value of this property can be an integer from -90 through 90 degrees or one of the  **[XlOrientation](28baaaa4-3ac7-350e-512c-e6f8c5651486.md)** constants.
