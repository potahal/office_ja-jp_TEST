
# Series.IsFiltered Property (Word)

Returns or sets a  **Boolean** that determines whether the specified chart series is filtered out from the chart. Read-write.


## Syntax

 _表达式_. **IsFiltered**

 _表达式_ A variable that represents a **Series** object.


## Property value

 **BOOL**
