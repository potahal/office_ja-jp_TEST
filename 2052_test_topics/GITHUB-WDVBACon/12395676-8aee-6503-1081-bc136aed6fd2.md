
# StyleSheet.Delete Method (Word)

Deletes the specified cascading style sheet.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents a **[StyleSheet](5e576ff8-c458-f5bd-730d-9db827c4f76e.md)** object.
