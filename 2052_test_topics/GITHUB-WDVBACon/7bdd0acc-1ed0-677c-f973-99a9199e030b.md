
# Application.Version Property (Word)

Returns the Microsoft Word version number. Read-only  **String**.


## Syntax

 _表达式_. **Version**

 _表达式_ A variable that represents an **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object.


## Example

This example displays the Word version number in a message box.


```
Msgbox "The version of Word is " &amp; Application.Version
```
