
# Frame.Range Property (Word)

Returns a  **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object that represents the portion of a document that's contained within the frame.


## Syntax

 _表达式_. **Range**

 _表达式_ Required. A variable that represents a **[Frame](d36d3361-9e93-7dd9-b8c9-0ce503e03810.md)** object.
