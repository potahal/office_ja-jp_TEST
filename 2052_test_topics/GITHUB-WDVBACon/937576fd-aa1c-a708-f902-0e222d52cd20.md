
# AutoCaptions.Count Property (Word)

Returns the number of items in the  **AutoCaptions** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[AutoCaptions](da4bd001-8f4c-28c9-4f46-a5a6499000a8.md)** object.
