
# ProtectedViewWindow.Application Property (Word)

Returns an [Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md) object that represents the Microsoft Word application. Read-only.


## Syntax

 _表达式_. **Application**

 _表达式_ An expression that returns a **ProtectedViewWindow** object.
