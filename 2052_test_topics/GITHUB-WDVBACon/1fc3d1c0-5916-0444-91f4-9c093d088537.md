
# Broadcast.Pause Method (Word)

Pauses the specified broadcast.


## Syntax

 _表达式_. **Pause**

 _表达式_ A variable that represents a **Broadcast** object.


### Return value

 **VOID**


## Remarks

If the specified broadcast is not in progress,  **Pause** returns runtime error 4702.
