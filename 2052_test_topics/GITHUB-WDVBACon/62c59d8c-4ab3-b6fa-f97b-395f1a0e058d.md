
# Columns.Count Property (Word)

Returns a  **Long** that represents the number of columns in the collection. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ Required. A variable that represents a **[Columns](7c2d1353-cbc4-a162-83a1-6cac1300266f.md)** collection.
