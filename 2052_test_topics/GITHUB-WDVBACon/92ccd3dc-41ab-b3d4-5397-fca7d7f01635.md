
# Selection.Collapse Method (Word)

Collapses a selection to the starting or ending position. After a selection is collapsed, the starting and ending points are equal.


## Syntax

 _表达式_. **Collapse**( ** _Direction_** )

 _表达式_ Required. A variable that represents a **[Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Direction_|可选|**Variant**|The direction in which to collapse the range or selection. Can be either of the following  **WdCollapseDirection** constants: **wdCollapseEnd** or **wdCollapseStart**. The default value is **wdCollapseStart**.|

## Example

This example collapses the selection to an insertion point at the beginning of the previous selection.


```
Selection.Collapse Direction:=wdCollapseStart
```
