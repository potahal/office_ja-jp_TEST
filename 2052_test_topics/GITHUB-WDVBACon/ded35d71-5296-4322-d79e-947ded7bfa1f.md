
# LegendKey.Delete Method (Word)

Deletes the object.


## Syntax

 _表达式_. **Delete**

 _表达式_ A variable that represents a **[LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.


## Remarks

Deleting a  **LegendKey** object deletes the entire series.
