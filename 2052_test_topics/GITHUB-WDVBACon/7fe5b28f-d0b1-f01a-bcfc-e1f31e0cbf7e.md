
# Points.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _表达式_. **Parent**

 _表达式_ A variable that represents a **[Points](d0adc45a-7b31-a25e-d96f-f2a098702501.md)** object.
