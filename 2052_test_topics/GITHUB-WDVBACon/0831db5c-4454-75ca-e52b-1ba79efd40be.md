
# ParagraphFormat.TextboxTightWrap Property (Word)

Returns or sets a  **[WdTextboxTightWrap](6d9e0dcd-816a-e055-b96a-7da5dcea38f7.md)** constant that represents how tightly text wraps around shapes or text boxes. Read/write.


## Syntax

 _表达式_. **TextboxTightWrap**

 _表达式_ An expression that returns a **[ParagraphFormat](712d754a-dc92-f1a3-531d-dfae74a42c23.md)** object.
