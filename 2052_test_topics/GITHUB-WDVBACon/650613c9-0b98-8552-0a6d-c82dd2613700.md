
# PageSetup.ShowGrid Property (Word)

 **True** to display the page grid. **False** to hide it. Read/write **Boolean**.


## Syntax

 _表达式_. **ShowGrid**

 _表达式_ A variable that represents a **PageSetup** object.


## Property value

 **BOOL**
