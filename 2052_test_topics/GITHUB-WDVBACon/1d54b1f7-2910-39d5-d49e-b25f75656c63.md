
# Options.PasteFormatBetweenStyledDocuments Property (Word)

Returns or sets a  **[WdPasteOptions](a5c03792-5662-9fa0-1c77-cb7c77cd32b2.md)** constant that represents how text is pasted when text is copied from a document that uses styles. Read/write.


## Syntax

 _表达式_. **PasteFormatBetweenStyledDocuments**

 _表达式_ An expression that returns an **Options** object.


## Remarks

Corresponds to the  **Pasting between two documents (with styles)** option in the **Advanced** tab of the **Word Options** dialog box.
