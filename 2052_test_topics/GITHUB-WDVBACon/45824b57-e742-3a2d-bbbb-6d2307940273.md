
# AxisTitle.HorizontalAlignment Property (Word)

Returns or sets the horizontal alignment for the specified object. Read/write  **Variant**.


## Syntax

 _表达式_. **HorizontalAlignment**

 _表达式_ A variable that represents an **[AxisTitle](ec746a05-40df-95cc-c017-40ef150504cf.md)** object.


## Remarks

The value of this property can be set to one of the following  **[XlConstants](90d5b59c-8fa6-171c-4f77-01536bc9a1d3.md)** constants:


-  **xlCenter**
    
-  **xlDistributed**
    
-  **xlJustify**
    
-  **xlLeft**
    
-  **xlRight**
    
Some of these constants may not be available to you, depending on the language support (for example, U.S. English) that you have selected or installed.
