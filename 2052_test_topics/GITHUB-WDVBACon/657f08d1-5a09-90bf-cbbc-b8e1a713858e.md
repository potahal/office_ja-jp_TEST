
# Font.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Font** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.
