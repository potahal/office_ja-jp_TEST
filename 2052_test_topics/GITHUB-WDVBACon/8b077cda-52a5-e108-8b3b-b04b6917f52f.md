
# Bookmark.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Bookmark](be6b0c7b-60ca-97e7-ef19-6de335da3197.md)** object.
