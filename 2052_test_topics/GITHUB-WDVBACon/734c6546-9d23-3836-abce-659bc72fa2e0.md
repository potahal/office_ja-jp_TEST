
# LegendEntry.Height Property (Word)

Returns the height, in points, of the object. Read-only  **Double**.


## Syntax

 _表达式_. **Height**

 _表达式_ A variable that represents a **[LegendEntry](9f793578-cb9b-faa1-f0a1-ea0f9e90dc6f.md)** object.
