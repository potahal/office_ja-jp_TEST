
# OMathBorderBox.StrikeV Property (Word)

Returns or sets a  **Boolean** that represents a vertical strikethrough. Read/write.


## Syntax

 _表达式_. **StrikeV**

 _表达式_ An expression that returns an **[OMathBorderBox](a13bd7f4-606a-d483-a36d-944356b13c95.md)** object.
