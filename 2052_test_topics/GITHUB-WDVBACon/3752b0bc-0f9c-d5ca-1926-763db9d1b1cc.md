
# Range.XML Property (Word)

Returns a  **String** that represents the XML text in the specified object. .


## Syntax

 _表达式_. **XML**( ** _DataOnly_** )

 _表达式_ An expression that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _DataOnly_|可选|**Boolean**|**True** returns the text of the XML without the Word XML markup. The default setting is **False**.|
