
# Series.Name Property (Word)

Returns or sets the name of the object. Read/write  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ A variable that represents a **[Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object.
