
# XMLSchemaReference.Reload Method (Word)

Reloads the XML schemas that are referenced in a document.


## Syntax

 _表达式_. **Reload**

 _表达式_ An expression that returns an **[XMLSchemaReference](54142ef1-f731-3f82-2dc0-809d8a041b73.md)** object.


### Return Value

Nothing
