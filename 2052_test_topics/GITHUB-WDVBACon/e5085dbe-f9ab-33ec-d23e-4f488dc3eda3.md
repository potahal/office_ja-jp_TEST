
# Endnotes.Separator Property (Word)

Returns a  **Range** object that represents the endnote separator.


## Syntax

 _表达式_. **Separator**

 _表达式_ Required. A variable that represents an **[Endnotes](32676579-dd41-e83d-a305-fcc2b7cb4f64.md)** collection.
