
# Borders.Count Property (Word)

Returns the number of items in the  **Borders** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[Borders](6dd1d4cc-2dcf-22c7-a299-4721a5543ba3.md)** collection.
