
# Font.Shading Property (Word)

Returns a  **Shading** object that refers to the shading formatting for the specified font.


## Syntax

 _表达式_. **Shading**

 _表达式_ Required. A variable that represents a **[Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.
