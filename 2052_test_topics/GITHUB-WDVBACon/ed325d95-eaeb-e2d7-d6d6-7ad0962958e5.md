
# ChartTitle.Width Property (Word)

Returns the width, in points, of the object. Read-only.


## Syntax

 _表达式_. **Width**

 _表达式_ A variable that represents a **[ChartTitle](fc8ca540-0a29-123b-2fdf-b16aaa1f940c.md)** object.
