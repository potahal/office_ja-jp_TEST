
# ThreeDFormat.SetPresetCamera Method (Word)

Sets the camera presets for a shape.


## Syntax

 _表达式_. **SetPresetCamera**( ** _PresetCamera_** )

 _表达式_ An expression that returns a **[ThreeDFormat](d397e780-a53d-0cc3-7a02-b40397253e91.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _PresetCamera_|必需|**MsoPresetCamera**|Specifies the camera preset type.|
