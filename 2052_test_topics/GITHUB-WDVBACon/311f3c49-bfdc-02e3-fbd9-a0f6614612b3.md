
# Range.Endnotes Property (Word)

Returns an  **[Endnotes](32676579-dd41-e83d-a305-fcc2b7cb4f64.md)** collection that represents all the endnotes in a range. Read-only.


## Syntax

 _表达式_. **Endnotes**

 _表达式_ A variable that represents a **[Range](15a7a1c4-5f3f-5b6e-60e9-29688de3f274.md)** object.


## Remarks

For information about returning a single member of a collection, see [Returning an Object from a Collection](28f76384-f495-9640-a7c8-10ada3fac727.md).
