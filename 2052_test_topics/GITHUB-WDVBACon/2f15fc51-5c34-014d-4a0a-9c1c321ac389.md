
# OtherCorrectionsExceptions.Count Property (Word)

Returns a  **Long** that represents the number of exceptions in the collection. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ Required. A variable that represents an **[OtherCorrectionsExceptions](f72135be-9a82-2c45-1835-8cabb18869de.md)** collection.
