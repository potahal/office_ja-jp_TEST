
# Chart.Select Method (Word)

Selects the object.


## Syntax

 _表达式_. **Select**( ** _Replace_** )

 _表达式_ A variable that represents a **[Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Replace_|可选|**Variant**|**True** to replace the current selection with the specified object. **False** to extend the current selection to include any previously selected objects and the specified object. This parameter is used only with sheets.|
