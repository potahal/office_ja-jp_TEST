
# TabStop.Next Property (Word)

Returns the next tabstop in the collection. Read-only.


## Syntax

 _表达式_. **Next**

 _表达式_ A variable that represents a **[TabStop](5290ae79-f728-24a8-6bb0-267072cd0288.md)** object.
