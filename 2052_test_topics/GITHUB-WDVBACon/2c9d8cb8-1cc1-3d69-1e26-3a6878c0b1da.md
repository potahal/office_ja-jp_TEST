
# Selection.InsertBreak Method (Word)

Inserts a page, column, or section break.


## Syntax

 _表达式_. **InsertBreak**( ** _Type_** )

 _表达式_ Required. A variable that represents a **[Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Type_|必需|**[WdBreakType](537782ce-c41a-b849-4f7b-2b70b0ac1700.md)**|the type of break to insert. The default value is  **wdPageBreak**. Some of the **WdBreakType** constants may not be available to you, depending on the language support (U.S. English, for example) that you have selected or installed.|

## Remarks

When you insert a page or column break, the break replaces the selection. If you don't want to replace the selection, use the  **[Collapse](92ccd3dc-41ab-b3d4-5397-fca7d7f01635.md)** method before using the **InsertBreak** method.


 **注释**  When you insert a section break, the break is inserted immediately preceding the selection.


## Example

This example inserts a continuous section break immediately preceding the selection.


```
Selection.InsertBreak Type:=wdSectionBreakContinuous
```
