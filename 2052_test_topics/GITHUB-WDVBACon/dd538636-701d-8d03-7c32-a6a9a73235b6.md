
# Endnote.Delete Method (Word)

Deletes the specified endnote.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents an **[Endnote](01f29be4-58e7-28f5-5fcb-dae50c33890e.md)** object.
