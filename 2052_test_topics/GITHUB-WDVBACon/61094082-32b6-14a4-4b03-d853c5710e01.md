
# ThreeDFormat.LightAngle Property (Word)

Returns or sets a  **Single** that represents angle of the lighting. Read/write.


## Syntax

 _表达式_. **LightAngle**

 _表达式_ An expression that returns a **[ThreeDFormat](d397e780-a53d-0cc3-7a02-b40397253e91.md)** object.
