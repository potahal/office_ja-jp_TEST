
# Sections.Last Property (Word)

Returns the last item in the  **Sections** collection as a **Section** object.


## Syntax

 _表达式_. **Last**

 _表达式_ Required. A variable that represents a **[Sections](cf6f77ba-9eee-5614-e697-bc031c4c6dcd.md)** collection.
