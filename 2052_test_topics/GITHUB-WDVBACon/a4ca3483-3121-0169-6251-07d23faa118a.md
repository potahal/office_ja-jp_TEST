
# Table.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Table** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Table](996b58dd-ebc6-ee30-5bfe-c5e51a0f71d6.md)** object.
