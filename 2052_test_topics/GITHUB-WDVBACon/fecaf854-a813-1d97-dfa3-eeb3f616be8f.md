
# OMathMatCols.Count Property (Word)

Returns the number of items in the  **OMathMatCols** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[OMathMatCols](b56ee426-56bd-6588-ebe9-898f4bfbba0c.md)** object.
