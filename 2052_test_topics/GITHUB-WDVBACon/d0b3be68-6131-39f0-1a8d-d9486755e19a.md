
# Style.Linked Property (Word)

Returns  **Boolean** that represents whether a style is a linked style that can be used for both paragraph and character formatting. Read-only.


## Syntax

 _表达式_. **Linked**

 _表达式_ An expression that returns a **Style** object.


## Remarks

 **True** indicates that the **Style Type** option in the **Modify Style** dialog box is set to " **Linked (paragraph and character)**".
