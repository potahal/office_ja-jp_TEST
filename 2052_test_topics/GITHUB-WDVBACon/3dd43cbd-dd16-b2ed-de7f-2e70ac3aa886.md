
# HeadingStyles.Item Method (Word)

Returns an individual  **HeadingStyle** object in a collection.


## Syntax

 _表达式_. **Item**( ** _Index_** )

 _表达式_ Required. A variable that represents a **[HeadingStyles](be882a12-1264-8f7e-415b-b8bcbf28e703.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

HeadingStyle
