
# Frames.Delete Method (Word)

Deletes the specified Frames collection.


## Syntax

 _表达式_. **Delete**

 _表达式_ An expression that returns a **[Frames](d0f526b5-ae1d-ad7a-0da3-5a7b30526b55.md)** object.


### Return Value

Nothing
