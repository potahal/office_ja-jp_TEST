
# MailMergeFieldNames.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **MailMergeFieldNames** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ An expression that returns a **MailMergeFieldNames** object.
