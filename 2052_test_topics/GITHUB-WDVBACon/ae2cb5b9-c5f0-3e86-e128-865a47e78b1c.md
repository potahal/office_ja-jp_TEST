
# AddIn.Delete Method (Word)

Deletes the specified add-in.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents an **[AddIn](5615a8a9-1fd6-04fa-1fee-ec16502bd84a.md)** object.
