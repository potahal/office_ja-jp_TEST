
# XMLSchemaReference.Delete Method (Word)

Deletes the specified XML Schema Reference.


## Syntax

 _表达式_. **Delete**

 _表达式_ Required. A variable that represents a **[XMLSchemaReference](54142ef1-f731-3f82-2dc0-809d8a041b73.md)** object.
