
# TablesOfAuthorities.Item Method (Word)

Returns an individual  **TableOfAuthorities** object in a collection.


## Syntax

 _表达式_. **Item**( ** _Index_** )

 _表达式_ Required. A variable that represents a **[TablesOfAuthorities](c0fd88b1-b737-2811-ec4c-1fc274fc3e20.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

TableOfAuthorities
