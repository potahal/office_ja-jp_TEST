
# Dictionary.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Dictionary** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Dictionary](1946d60c-2abd-9ca9-8d0b-7068e9173bb3.md)** object.
