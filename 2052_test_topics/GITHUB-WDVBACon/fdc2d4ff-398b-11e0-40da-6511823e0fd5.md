
# OMathAutoCorrectEntry.Name Property (Word)

Returns or sets a  **String** that represents the name of an equation auto correct entry. Read/write.


## Syntax

 _表达式_. **Name**

 _表达式_ An expression that returns an **OMathAutoCorrectEntry** object.
