
# Editor.Name Property (Word)

Returns or sets the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ An expression that represents a **[Editor](af0c80f5-8c8a-be0e-4475-d3b3b3bacd0d.md)** object.
