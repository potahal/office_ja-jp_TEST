
# Point.Copy Method (Word)

If the point has a picture fill, copies the picture to the Clipboard.


## Syntax

 _表达式_. **Copy**

 _表达式_ A variable that represents a **[Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.
