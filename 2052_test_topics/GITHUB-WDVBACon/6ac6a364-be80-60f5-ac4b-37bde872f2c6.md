
# InlineShape.Glow Property (Word)

Returns a  **[GlowFormat](d177e399-cafc-132f-b135-b8b578e76889.md)** object that represents the formatting properties for a glow effect. Read-only.


## Syntax

 _表达式_. **Glow**

 _表达式_ An expression that returns an **[InlineShape](a8fd110a-4aa7-c4b9-1559-32022787d955.md)** object.
