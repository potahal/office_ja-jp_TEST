
# TableStyle.AllowPageBreaks Property (Word)

Allows Microsoft Word to break the specified table across pages. Read/write  **Boolean**.


## Syntax

 _表达式_. **AllowPageBreaks**

 _表达式_ Required. A variable that represents a **[TableStyle](4f1f4489-0ef7-dff0-8f2a-77f87937f3ad.md)** object.
