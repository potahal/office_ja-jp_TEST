
# DisplayUnitLabel.ReadingOrder Property (Word)

Returns or sets an  **[XlReadingOrder](eb7796d5-8ef5-96a4-dcf4-c86153f9a475.md)** constant that represents the reading order for the specified object. Read/write **Long**.


## Syntax

 _表达式_. **ReadingOrder**

 _表达式_ A variable that represents a **[DisplayUnitLabel](9b028f6c-fd66-f767-f3d1-09de0fbdc148.md)** object.
