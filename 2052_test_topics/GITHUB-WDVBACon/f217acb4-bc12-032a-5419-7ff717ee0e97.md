
# Paragraph.ListNumberOriginal Property (Word)

Returns an  **Integer** that represents the original list level for a paragraph. Read-only.


## Syntax

 _表达式_. **ListNumberOriginal**( ** _Level_** )

 _表达式_ An expression that returns a **[Paragraph](0a704079-a082-4ab1-841b-fc0d49dd26d4.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Level_|必需|**Integer**|Specifies the list level.|
