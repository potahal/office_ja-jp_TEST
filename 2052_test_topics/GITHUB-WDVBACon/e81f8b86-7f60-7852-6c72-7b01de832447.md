
# Font.Shadow Property (Word)

 **True** if the specified font is formatted as shadowed. Read/write **Long**.


## Syntax

 _表达式_. **Shadow**

 _表达式_ Required. A variable that represents a **[Font](bc97f4df-fc81-d6c8-e99a-d50dc793b7ae.md)** object.


## Remarks

This property can be  **True**, **False**, or **wdUndefined**.


## Example

This example applies shadow and bold formatting to the selection.


```
If Selection.Type = wdSelectionNormal Then 
 With Selection.Font 
 .Shadow = True 
 .Bold = True 
 End With 
Else 
 MsgBox "You need to select some text." 
End If
```
