
# Subdocuments.Select Method (Word)

Selects the specified subdocument.


## Syntax

 _表达式_. **Select**

 _表达式_ Required. A variable that represents a **[Subdocuments](8e14fced-19b3-2794-39a3-9e5ec15dd0ad.md)** collection.


## Remarks

After using this method, use the  **Selection** object to work with the selected items. For more information, see[Working with the Selection Object](a1ef7e48-5a0f-d278-4b67-7b96f4e24052.md).
