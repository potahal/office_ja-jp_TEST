
# Shape.RelativeHorizontalSize Property (Word)

Returns or sets a  **[WdRelativeVerticalSize](07a507e4-66fb-b383-fe36-aa5ccadbdcba.md)** constant that represents the object to which a range of shapes is relative. Read/write.


## Syntax

 _表达式_. **RelativeHorizontalSize**

 _表达式_ An expression that returns a **[Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.


## Remarks

Use this property with the  **[WidthRelative](db076311-7ecf-3564-9cf4-400663ae400b.md)** property.
