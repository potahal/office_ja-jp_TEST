
# EmailOptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **EmailOptions** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents an **[EmailOptions](41fefa03-c993-e218-0f92-0cf30c0bfbd4.md)** collection.
