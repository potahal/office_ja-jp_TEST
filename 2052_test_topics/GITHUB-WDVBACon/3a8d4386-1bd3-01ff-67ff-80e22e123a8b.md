
# Shape.Glow Property (Word)

Returns a  **[GlowFormat](d177e399-cafc-132f-b135-b8b578e76889.md)** object that represents the glow formatting for a shape. Read-only.


## Syntax

 _表达式_. **Glow**

 _表达式_ An expression that returns a **[Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.
