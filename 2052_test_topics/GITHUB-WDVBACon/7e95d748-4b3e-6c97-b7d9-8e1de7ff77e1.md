
# Panes.Item Method (Word)

Returns the specified pane as a  **Pane** object.


## Syntax

 _表达式_. **Item**( ** _Index_** )

 _表达式_ Required. A variable that represents a **[Panes](6ed6353c-9134-f47d-a108-13e84eced8ff.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

Pane
