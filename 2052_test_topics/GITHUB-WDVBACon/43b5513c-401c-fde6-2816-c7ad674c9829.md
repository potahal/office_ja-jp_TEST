
# Characters.Item Method (Word)

Returns an individual  **Range** object in a collection.


## Syntax

 _表达式_. **Item**( ** _Index_** )

 _表达式_ Required. A variable that represents a **[Characters](6d22ae7a-128d-134d-9136-1cdd5a8d9941.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

Range
