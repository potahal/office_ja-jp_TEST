
# Global.Help Method (Word)

Displays on-line Help information.


## Syntax

 _表达式_. **Help**( ** _HelpType_** )

 _表达式_ A variable that represents a **[Global](b91e7459-08d5-ea8c-42e0-f7b9bfd1a72c.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _HelpType_|必需|**Variant**|The on-line Help topic or window. Can be any of these  **[WdHelpType](798a78ec-73cc-62aa-32fd-19f08c4c556f.md)** constants.|

## Remarks

Some of the constants listed above may not be available to you, depending on the language support (U.S. English, for example) that you?ve selected or installed.


## Example

This example displays the Help Topics dialog box.


```
Help HelpType:=wdHelp
```

This example displays a list of Help topics that describe how to use Help.




```
Help HelpType:=wdHelpUsingHelp
```
