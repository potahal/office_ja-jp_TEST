
# Points.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[Points](d0adc45a-7b31-a25e-d96f-f2a098702501.md)** object.
