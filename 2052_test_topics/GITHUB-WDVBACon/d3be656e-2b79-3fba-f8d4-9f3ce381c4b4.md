
# DisplayUnitLabel.Position Property (Word)

Returns or sets the position of the unit label on an axis in the chart. Read/write  **[XlChartElementPosition](96be8751-d642-dddf-3fd0-588d6cfe7410.md)**.


## Syntax

 _表达式_. **Position**

 _表达式_ A variable that represents a **[DisplayUnitLabel](9b028f6c-fd66-f767-f3d1-09de0fbdc148.md)** object.
