
# SeriesCollection.Add Method (Word)

Adds one or more new series to the collection.


## Syntax

 _表达式_. **Add**( ** _Source_**, ** _Rowcol_**, ** _SeriesLabels_**, ** _CategoryLabels_**, ** _Replace_** )

 _表达式_ A variable that represents a **[SeriesCollection](785d61ff-96c9-b9b0-ed98-e992d9adeda6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Source_|必需|**Variant**|The new data as a string representation of a range contained in the  **[Workbook](2295d653-7a36-b258-dfb8-f48844331705.md)** property of the **[ChartData](323ee62c-9b70-8280-d448-79cf4d2b6953.md)** object for the chart.|
| _Rowcol_|可选|**[XlRowCol](d9c9303d-155f-155c-6012-c7df2e1e47e4.md)**|One of the enumeration values that specifies whether the new values are in the rows or columns of the specified range.|
| _SeriesLabels_|可选|**Variant**|**True** if the first row or column contains the name of the data series. **False** if the first row or column contains the first data point of the series. If this argument is omitted, Microsoft Word attempts to determine the location of the series name from the contents of the first row or column.|
| _CategoryLabels_|可选|**Variant**|**True** if the first row or column contains the name of the category labels. **False** if the first row or column contains the first data point of the series. If this argument is omitted, Word attempts to determine the location of the category label from the contents of the first row or column.|
| _Replace_|可选|**Variant**|If CategoryLabels is  **True** and Replace is **True**, the specified categories replace the categories that currently exist for the series. If Replace is **False**, the existing categories will not be replaced. The default is **False**.|

### Return Value

A  **[Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object that represents the new series.


## Remarks

This method does not actually return a  **Series** object as stated in the Object Browser.


## Example

The following example creates a new series for the first chart in the active document. The data source for the new series is range  `B1:B10` on the workbook associated with the chart.


```
With ActiveDocument.InlineShapes(1) 
 If .HasChart Then 
 .Chart.SeriesCollection.Add _ 
 Source:="Sheet1!B1:B10" 
 End If 
End With
```
