
# Shapes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Shapes** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Shapes](0907eed3-886e-8e73-0e5e-71f4b37ddd5b.md)** collection.
