
# Shape.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Shape** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Shape](604029ce-9b2f-9748-5d4e-b458796fa2f0.md)** object.
