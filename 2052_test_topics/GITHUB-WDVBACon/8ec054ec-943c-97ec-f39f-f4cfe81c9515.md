
# Row.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Row** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Row](38a05858-829a-ea5c-ce63-7f7343bf7b88.md)** object.
