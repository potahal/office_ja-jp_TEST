
# ContentControl.Color Property (Word)

Returns or sets the color of the content control. Read/write [WdColor](11a5b2ba-279d-9654-cff6-e430ae27666d.md).


## Syntax

 _表达式_. **Color**

 _表达式_ A variable that represents a **ContentControl** object.


## Property value

 **WDCOLOR**
