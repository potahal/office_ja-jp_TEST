
# Conflict.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _表达式_. **Index**

 _表达式_ An expression that returns a **[Conflict](e9fe0318-d3e3-7589-0c15-64210ac5b709.md)** object.
