
# Selection.InsertNewPage Method (Word)

Inserts a new page at the position of the Insertion Point.


## Syntax

 _表达式_. **InsertNewPage**

 _表达式_ An expression that returns a **[Selection](7b574a91-c33e-ecfd-6783-6b7528b2ed8f.md)** object.
