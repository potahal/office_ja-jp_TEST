
# CaptionLabel.Position Property (Word)

Returns or sets the position of caption label text. Read/write  **WdCaptionPosition**.


## Syntax

 _表达式_. **Position**

 _表达式_ Required. A variable that represents a **[CaptionLabel](71c82dfd-6a66-e0f4-e30f-ae453c764864.md)** object.
