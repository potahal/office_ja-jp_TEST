
# TablesOfContents.Count Property (Word)

Returns a  **Long** that represents the number of tables of contents in the collection. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ Required. A variable that represents a **[TablesOfContents](d0d0e5fc-e443-31ae-e1a9-15b945f1e318.md)** collection.
