
# Legend.Height Property (Word)

Returns or sets the height, in points, of the object. Read/write  **Double**.


## Syntax

 _表达式_. **Height**

 _表达式_ A variable that represents a **[Legend](f0122074-87b7-0225-3c6c-406103fa4c29.md)** object.
