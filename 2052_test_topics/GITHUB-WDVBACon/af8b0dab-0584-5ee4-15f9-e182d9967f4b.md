
# DataLabels.NumberFormat Property (Word)

Returns or sets the format code for the object. Read/write  **String**.


## Syntax

 _表达式_. **NumberFormat**

 _表达式_ A variable that represents a **[DataLabels](a7676f18-b1f2-1e11-9489-863cb85c1669.md)** object.


## Remarks

The format code is the same string as the  **Format Codes** option in the **Format Cells** dialog box. The **Format** function uses different format code strings than do the **NumberFormat** and **[NumberFormatLocal](28b0544e-e5cc-81bd-bf0b-48b6a8015fbb.md)** properties.
