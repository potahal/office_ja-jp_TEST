
# ShadowFormat.Size Property (Word)

Returns or sets a  **Single** that represents the width of the shadow. Read/write.


## Syntax

 _表达式_. **Size**

 _表达式_ An expression that returns a **[ShadowFormat](2a179f0b-ec18-c3dd-dd73-51b18f42e0e2.md)** object.
