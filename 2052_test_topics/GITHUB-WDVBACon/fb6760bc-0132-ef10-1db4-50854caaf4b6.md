
# Dictionary.Name Property (Word)

Returns the name of the specified object. Read-only  **String**.


## Syntax

 _表达式_. **Name**

 _表达式_ Required. A variable that represents a **[Dictionary](1946d60c-2abd-9ca9-8d0b-7068e9173bb3.md)** object.
