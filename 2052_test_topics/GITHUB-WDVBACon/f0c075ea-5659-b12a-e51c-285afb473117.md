
# Frame.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Frame** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Frame](d36d3361-9e93-7dd9-b8c9-0ce503e03810.md)** object.
