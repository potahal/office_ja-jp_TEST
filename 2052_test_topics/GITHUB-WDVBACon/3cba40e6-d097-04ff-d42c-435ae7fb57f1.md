
# Task.Width Property (Word)

Returns or sets the width of the specified  **Task** object, in points. Read/write **Long**.


## Syntax

 _表达式_. **Width**

 _表达式_ A variable that represents a **[Task](8802fcd5-0947-2ea0-308a-376077633e34.md)** object.
