
# AxisTitle.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _表达式_. **Parent**

 _表达式_ A variable that represents an **[AxisTitle](ec746a05-40df-95cc-c017-40ef150504cf.md)** object.
