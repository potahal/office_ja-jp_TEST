
# ReflectionFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ReflectionFormat** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ An expression that returns a **ReflectionFormat** object.
