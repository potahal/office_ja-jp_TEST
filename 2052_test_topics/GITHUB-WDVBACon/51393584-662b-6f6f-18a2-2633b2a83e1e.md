
# XMLNode.Cut Method (Word)

Removes the specified XML element from the document and places it on the Clipboard.


## Syntax

 _表达式_. **Cut**

 _表达式_ Required. A variable that represents a **[XMLNode](fe305ba9-7375-ad4f-6036-155add17a9d0.md)** object.
