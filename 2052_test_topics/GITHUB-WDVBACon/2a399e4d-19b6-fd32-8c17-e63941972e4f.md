
# Bookmarks.Count Property (Word)

Returns the number of items in the  **Bookmarks** collection. Read-only **Long**.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[Bookmarks](827bed64-3034-0eb4-401d-f117cdb98898.md)** object.
