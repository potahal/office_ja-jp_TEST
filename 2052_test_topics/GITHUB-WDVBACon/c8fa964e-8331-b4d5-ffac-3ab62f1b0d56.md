
# Options.MoveFromTextMark Property (Word)

 Returns or sets a **[WdMoveFromTextMark](343005e2-535f-5de2-34b6-efa7353d44ff.md)** constant that represents the type of revision mark to use for moved text. Read/write.


## Syntax

 _表达式_. **MoveFromTextMark**

 _表达式_ An expression that returns an **[Options](873b7b99-3fe1-fd89-9ece-a9355cb827dc.md)** object.
