
# Dialog.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Dialog** object.


## Syntax

 _表达式_. **Parent**

 _表达式_ Required. A variable that represents a **[Dialog](f90f6e6d-aaa0-c127-ab37-ca074144eff1.md)** object.
