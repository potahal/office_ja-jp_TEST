
# ListObject.Slicers Property (Excel)

Returns a list of the table slicers associated with a  **ListObject**. Read-only


## Syntax

 _表达式_. **Slicers**

 _表达式_ A variable that represents a **ListObject** object.


## Remarks

A  **ListObject** is essentially a table.


## Property value

 **SLICERS**
