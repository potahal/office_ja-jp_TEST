
# Range.Comment Property (Excel)

Returns a  **[Comment](3627e9be-2a28-9dc5-c822-ad42857134e3.md)** object that represents the comment associated with the cell in the upper-left corner of the range.


## Syntax

 _表达式_. **Comment**

 _表达式_ A variable that represents a **Range** object.
