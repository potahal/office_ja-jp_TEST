
# Application.SheetTableUpdate Event (Excel)

Occurs when a table on a worksheet is updated.


## Syntax

 _表达式_. **SheetTableUpdate** _(Sh,_ _Target)_

 _表达式_ A variable that represents a **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Sh_|必需|OBJECT|The worksheet.|
| _Target_|必需|TABLEOBJECT|The table.|
