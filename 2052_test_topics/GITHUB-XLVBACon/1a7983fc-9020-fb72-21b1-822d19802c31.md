
# Name.WorkbookParameter Property (Excel)

Returns or sets the specified  **Name** object as a workbook parameter. Read/write


## Syntax

 _表达式_. **WorkbookParameter**

 _表达式_ An expression that returns a **Name** object.


### Return Value

Boolean
