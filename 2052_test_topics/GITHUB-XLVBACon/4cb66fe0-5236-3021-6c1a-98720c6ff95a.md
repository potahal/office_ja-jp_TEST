
# CustomProperty.Delete Method (Excel)

Deletes the object.


## Syntax

 _表达式_. **Delete**

 _表达式_ A variable that represents a **CustomProperty** object.


## Remarks

You can delete custom document properties, but you cannot delete a built-in document property.
