
# Application.SharePointVersion Method (Excel)

Returns the version number of SharePoint Foundation instances running at site for the specified URL.


## Syntax

 _表达式_. **SharePointVersion**( ** _bstrUrl_** )

 _表达式_ A variable that returns an **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _bstrUrl_|必需|**String**|The URL of site to check.|

### Return Value

Long
