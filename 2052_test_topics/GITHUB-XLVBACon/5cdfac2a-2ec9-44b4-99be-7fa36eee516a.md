
# ModelChanges.Source Property (Excel)

Returns the source of the data model. Read-only. 


## Syntax

 _表达式_. **Source**

 _表达式_ A variable that represents a **ModelChanges** object.


## Remarks

The source can be either Excel or a PowerPivot Add-in.


## Property value

 **XlModelChangeSource**
