
# WorksheetFunction.Munit Method (Excel)

Returns the unit matrix for the specified dimension.


## Syntax

 _表达式_. **Munit** _(Arg1)_

 _表达式_ A variable that represents a[WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|DOUBLE|Integer specifying the dimension of the unit matrix that you want to return.|

### Return value

 **VARIANT**
