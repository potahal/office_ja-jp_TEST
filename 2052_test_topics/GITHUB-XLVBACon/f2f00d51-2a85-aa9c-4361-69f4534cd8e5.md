
# Offset 属性

返回或设置每一层标签间的距离以及第一层标签与轴线间的距离。其默认值为 100，它代表了坐标轴标签之间以及标签与轴线之间的间距。此值可以是从 0 到 1000 之间的一个整数，其大小与坐标轴标签的字体大小相关。 **Long** 类型，可读写。


## 示例

如果 myChart 中数值轴的现有刻度间距小于 500，则以下示例将该间距乘以 2。


```
With myChart.Axes(xlCategory).TickLabels 
 If .Offset < 500 then 
 .Offset = .Offset * 2 
 End If 
End With 

```
