
# Pane.PointsToScreenPixelsX Method (Excel)

Returns or sets a pixel point on the screen.


## Syntax

 _表达式_. **PointsToScreenPixelsX**( ** _Points_** )

 _表达式_ A variable that represents a **Pane** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Points_|必需|**Long**|Location of the pixel on the screen.|

### Return Value

Long
