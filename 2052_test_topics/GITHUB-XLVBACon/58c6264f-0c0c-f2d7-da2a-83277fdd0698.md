
# Sort.Orientation Property (Excel)

Specifies the orientation for the sort. Read/write  **[XlSortOrientation](d63a94e5-ff58-69b6-e553-4c485948ee62.md)**.


## Syntax

 _表达式_. **Orientation**

 _表达式_ A variable that represents a **Sort** object.
