
# Chart.ApplyChartTemplate Method (Excel)

Applies a standard or custom chart type to a chart.


## Syntax

 _表达式_. **ApplyChartTemplate**( ** _Filename_** )

 _表达式_ A variable that represents a **Chart** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Filename_|必需|**String**|The file name for a chart template.|

## Remarks

This method does not support taking constants for native or combination charts.
