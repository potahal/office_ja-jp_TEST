
# Range.CopyPicture Method (Excel)

Copies the selected object to the Clipboard as a picture.  **Variant**.


## Syntax

 _表达式_. **CopyPicture**( ** _Appearance_**, ** _Format_** )

 _表达式_ A variable that represents a **Range** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Appearance_|可选|**[XlPictureAppearance](5fd97feb-a976-2c1c-4051-f2c99d0fa793.md)**|. Specifies how the picture should be copied.|
| _Format_|可选|**[XlCopyPictureFormat](a764bd52-1c2b-9395-7774-1daceb6a8415.md)**|. The format of the picture.|

### Return Value

Variant
