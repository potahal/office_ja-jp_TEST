
# Font 对象

包含指定对象的字体属性（字体名称、字号、颜色等）。


## Font 对象用法

可用  **Font** 属性返回 **Font** 对象。下例设置数值轴标题的文字，并将字体设置为 10 磅的 Bookman，然后将单词"millions"设置为倾斜。


```
With myChart.Axes(xlValue) 
 .HasTitle = True 
 With .AxisTitle 
 .Caption = "Revenue (millions)" 
 .Font.Name = "bookman" 
 .Font.Size = 10 
 .Characters(10, 8).Font.Italic = True
 End With 
End With
```
