
# PivotFilter.Value1 Property (Excel)

This property is a user-supplied parameter to define a filter for a PivotField. Read/write  **Variant**.


## Syntax

 _表达式_. **Value1**

 _表达式_ A variable that represents a **PivotFilter** object.
