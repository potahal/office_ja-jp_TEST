
# Slicer.Height Property (Excel)

Returns or sets the height of the specified slicer, in points. Read/write.


## Syntax

 _表达式_. **Height**

 _表达式_ A variable that represents a **[Slicer](577be0f6-4eda-0093-8899-097f3c900383.md)** object.


### Return Value

Double
