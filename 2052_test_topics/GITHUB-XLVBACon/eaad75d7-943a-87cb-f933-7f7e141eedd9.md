
# SlicerCacheLevel.Name Property (Excel)

Returns the MDX unique name of OLAP hierarchy level represented by the object.


## Syntax

 _表达式_. **Name**

 _表达式_ A variable that returns a **[SlicerCacheLevel](d73ff7ab-4d7a-6a73-3716-11dc6716688d.md)** object.


### Return Value

 **String**
