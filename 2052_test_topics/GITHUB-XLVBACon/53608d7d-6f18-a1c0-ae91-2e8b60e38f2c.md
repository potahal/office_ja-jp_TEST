
# ThreeDFormat.SetPresetCamera Method (Excel)

Sets the camera for the specified  **ThreeDFormat** object.


## Syntax

 _表达式_. **SetPresetCamera**( ** _PresetCamera_** )

 _表达式_ A variable that represents a **ThreeDFormat** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _PresetCamera_|必需|**MsoPresetCamera**|Specifies the preset camera.|
