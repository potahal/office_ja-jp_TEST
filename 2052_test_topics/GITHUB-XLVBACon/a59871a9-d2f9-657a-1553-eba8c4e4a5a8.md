## 示例

当图表为二维气泡图时，本示例将第一个图表组中的气泡大小设置为默认大小的 200%。


```
With myChart 
 If .ChartType = xlBubble Then 
 .ChartGroups(1).BubbleScale = 200 
 End If 
End With
```
