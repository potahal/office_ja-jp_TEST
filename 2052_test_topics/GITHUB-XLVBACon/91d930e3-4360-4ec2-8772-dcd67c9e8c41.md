
# QueryTable.AfterRefresh Event (Excel)

Occurs after a query is completed or canceled.


## Syntax

 _表达式_. **AfterRefresh**( ** _Success_** )

 _表达式_ A variable that represents a **QueryTable** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Success_|必需|**Boolean**|**True** if the query was completed successfully.|

### Return Value

Nothing


## Example

This example uses the  `Success` argument to determine which section of code to run.


```
Private Sub QueryTable_AfterRefresh(Success As Boolean) 
 If Success Then 
 ' Query completed successfully 
 Else 
 ' Query failed or was cancelled 
 End If 
End Sub
```
