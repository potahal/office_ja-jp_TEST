
# ChartGroup.BinsOverflowValue Property (Excel)

If an [BinsOverflowEnabled](3af8d552-94e1-6f15-df2b-38fb7d3a0be1.md) is **True**, specifies the value above which an overflow bin is displayed. Read/write  **Double**.在 Office 2016 中引入。


## Syntax

 _表达式_. **BinsOverflowValue**

 _表达式_ A variable that represents a **ChartGroup** object.
