
# Databar.PTCondition Property (Excel)

Returns a  **Boolean** value indicating if the conditional format is being applied to a PivotTable. Read-only.


## Syntax

 _表达式_. **PTCondition**

 _表达式_ A variable that represents a **Databar** object.
