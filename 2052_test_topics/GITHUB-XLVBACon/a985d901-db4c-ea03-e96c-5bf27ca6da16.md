
# SparklineGroups.Clear Method (Excel)

Clears the selected sparklines.


## Syntax

 _表达式_. **Clear**

 _表达式_ A variable that represents a **[SparklineGroups](9bc6be34-fa2e-8652-ca92-fa9630b4d7a6.md)** object.


### Return Value
