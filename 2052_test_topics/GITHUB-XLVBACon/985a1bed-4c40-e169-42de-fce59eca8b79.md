
# ShapeNodes.Delete Method (Excel)

Deletes the object.


## Syntax

 _表达式_. **Delete**( ** _Index_** )

 _表达式_ A variable that represents a **ShapeNodes** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Integer**|The index of the object to delete.|
