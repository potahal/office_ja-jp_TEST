
# LineFormat.Pattern Property (Excel)

Returns or sets an  **[MsoPatternType](2485a32b-422c-fc99-df19-05e23925579c.md)** value that represents the fill pattern.


## Syntax

 _表达式_. **Pattern**

 _表达式_ A variable that represents a **LineFormat** object.
