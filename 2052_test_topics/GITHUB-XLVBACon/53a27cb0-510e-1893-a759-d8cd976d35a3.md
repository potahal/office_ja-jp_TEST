
# HeaderFooter.Picture Property (Excel)

Returns a  **Picture** object that represents a picture field included in the specified header or footer. Read-only.


## Syntax

 _表达式_. **Picture**

 _表达式_ A variable that represents a **HeaderFooter** object.
