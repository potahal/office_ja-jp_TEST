
# WorksheetFunction.Log10 Method (Excel)

Returns the base-10 logarithm of a number.


## Syntax

 _表达式_. **Log10**( ** _Arg1_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Double**|Number - the positive real number for which you want the base-10 logarithm.|

### Return Value

Double
