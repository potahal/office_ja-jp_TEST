
# SlicerItem.Name Property (Excel)

Returns the name of the slicer item. Read-only.


## Syntax

 _表达式_. **Name**

 _表达式_ A variable that represents a **[SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object.


### Return Value

String
