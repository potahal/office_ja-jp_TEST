
# WorksheetFunction.Rept Method (Excel)

Repeats text a given number of times. Use REPT to fill a cell with a number of instances of a text string.


## Syntax

 _表达式_. **Rept**( ** _Arg1_**, ** _Arg2_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**String**|Text - the text you want to repeat.|
| _Arg2_|必需|**Double**|Number_times - a positive number specifying the number of times to repeat text.|

### Return Value

String


## Remarks




- If number_times is 0 (zero), REPT returns "" (empty text).
    
- If number_times is not an integer, it is truncated.
    
- The result of the REPT function cannot be longer than 32,767 characters, or REPT returns #VALUE!.
