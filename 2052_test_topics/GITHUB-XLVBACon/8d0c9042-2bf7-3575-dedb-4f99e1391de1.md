
# Application.SheetCalculate Event (Excel)

Occurs after any worksheet is recalculated or after any changed data is plotted on a chart.


## Syntax

 _表达式_. **SheetCalculate**( ** _Sh_**, )

 _表达式_ An expression that returns a **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Sh_|必需|**Object**|Can be a  **[Chart](179c32ce-49bd-6f36-ea12-89fb5443f3ea.md)** or **[Worksheet](182b705e-854a-81cc-a4b0-59b942de55ae.md)** object.|
