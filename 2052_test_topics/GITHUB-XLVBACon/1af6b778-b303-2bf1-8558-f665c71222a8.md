
# Text 属性

返回或设置指定对象中的文本。String 类型，可读写。

 _expression_. **Text**

 _表达式_ 必需。该表达式返回"应用于"列表中的一个对象。


## 示例

以下示例设置图表的标题文本。


```
With myChart 
 .HasTitle = True 
 .ChartTitle.Text = "First Quarter Sales" 
End With
```

以下示例设置分类轴的轴标题文本。




```
With myChart.Axes(xlCategory) 
 .HasTitle = True 
 .AxisTitle.Text = "Month" 
End With
```
