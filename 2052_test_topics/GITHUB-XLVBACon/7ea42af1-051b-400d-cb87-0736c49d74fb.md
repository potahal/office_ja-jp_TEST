
# Workbook.PurgeChangeHistoryNow Method (Excel)

Removes entries from the change log for the specified workbook.


## Syntax

 _表达式_. **PurgeChangeHistoryNow**( ** _Days_**, ** _SharingPassword_** )

 _表达式_ A variable that represents a **Workbook** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Days_|必需|**Long**|The number of days that changes in the change log are to be retained.|
| _SharingPassword_|可选|**Variant**|The password that unprotects the workbook for sharing. If the workbook is protected for sharing with a password and this argument is omitted, the user is prompted for the password.|

## Example

This example removes all changes that are more than one day old from the change log for the active workbook.


```
ActiveWorkbook.PurgeChangeHistoryNow Days:=1
```
