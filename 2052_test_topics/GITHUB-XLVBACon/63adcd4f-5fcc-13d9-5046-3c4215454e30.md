
# SlicerItems.Count Property (Excel)

Returns the number of objects in the collection. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[SlicerItems](80bbbbab-711a-cefb-255b-94fe2994d3c8.md)** object.


### Return Value

 **Long**
