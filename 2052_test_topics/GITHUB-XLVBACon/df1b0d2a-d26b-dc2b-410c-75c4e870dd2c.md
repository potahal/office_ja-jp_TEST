
# WorksheetFunction.Radians Method (Excel)

Converts degrees to radians.


## Syntax

 _表达式_. **Radians**( ** _Arg1_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Double**|Angle - an angle in degrees that you want to convert.|

### Return Value

Double
