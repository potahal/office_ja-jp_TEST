
# Style.ShrinkToFit Property (Excel)

Returns or sets a  **Boolean** value that indicates if text automatically shrinks to fit in the available column width.


## Syntax

 _表达式_. **ShrinkToFit**

 _表达式_ A variable that represents a **Style** object.
