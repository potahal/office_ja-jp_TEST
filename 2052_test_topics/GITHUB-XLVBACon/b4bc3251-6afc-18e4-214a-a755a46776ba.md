
# Chart.ChartStyle Property (Excel)

Returns or sets the chart style for the chart. Read/write  **Variant**.


## Syntax

 _表达式_. **ChartStyle**

 _表达式_ A variable that represents a **Chart** object.


## Remarks

You can use a number from 1 to 48 to set the chart style.
