
# PivotTable.DrillTo Method (Excel)

Enables you to drill to a location within an OLAP or PowerPivot based cube hierarchy.


## Syntax

 _表达式_. **DrillTo** _(PivotItem,_ _CubeField,_ _PivotLine)_

 _表达式_ A variable that represents a[PivotTable Object (Excel)](a9c1d4a0-78a9-f9a6-6daf-91cb63e45842.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _PivotItem_|必需|PIVOTITEM|The member from which the drill operation is performed.|
| _CubeField_|必需|CUBEFIELD|The target hierarchy being drilled to.|
| _PivotLine_|可选|VARIANT|Specifies the line in the PivotTable where the operation starting member resides. In cases where PivotLine is not specified, defaults to the top PivotLine where the member appears.|

### Return value

 **VOID**
