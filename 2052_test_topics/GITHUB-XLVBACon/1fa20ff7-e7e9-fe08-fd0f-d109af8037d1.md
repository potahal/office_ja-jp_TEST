
# WorksheetFunction.ChiSq_Inv Method (Excel)

Returns the inverse of the left-tailed probability of the chi-squared distribution.


## Syntax

 _表达式_. **ChiSq_Inv**( ** _Arg1_**, ** _Arg2_** )

 _表达式_ A variable that represents a **[WorksheetFunction](7b1d5639-363d-632c-2cf0-2232562646b6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Double**|Probability - A probability associated with the chi-squared distribution.|
| _Arg2_|必需|**Double**|Deg_freedom - The number of degrees of freedom.|

### Return Value

Double


## Remarks




- If any argument is nonnumeric, CHISQ_INV returns the #VALUE! error value. 
    
- If probability < 0 or probability > 1, CHISQ_INV returns the #NUM! error value. 
    
- If deg_freedom is not an integer, it is truncated.
