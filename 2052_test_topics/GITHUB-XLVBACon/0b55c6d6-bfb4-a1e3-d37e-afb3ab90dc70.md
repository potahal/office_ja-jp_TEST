
# WorksheetFunction.RandBetween Method (Excel)

Returns a random integer number between the numbers you specify. A new random integer number is returned every time the worksheet is calculated.


## Syntax

 _表达式_. **RandBetween**( ** _Arg1_**, ** _Arg2_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Variant**|Bottom - the smallest integer RANDBETWEEN will return.|
| _Arg2_|必需|**Variant**|Top - the largest integer RANDBETWEEN will return.|

### Return Value

Double
