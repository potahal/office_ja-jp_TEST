
# Chart.Name Property (Excel)

Returns or sets a  **String** value representing the name of the object.


## Syntax

 _表达式_. **Name**

 _表达式_ A variable that represents a **Chart** object.


## Remarks

This property is read-only for chart objects (embedded charts).
