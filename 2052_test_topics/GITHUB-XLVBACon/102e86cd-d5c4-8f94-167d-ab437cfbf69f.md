
# Parameter.Type Property (Excel)

Returns a  **[XlParameterType](f6774f89-4992-2b7c-2dce-791fecafc1df.md)** value that represents the parameter type.


## Syntax

 _表达式_. **Type**

 _表达式_ A variable that represents a **Parameter** object.
