
# WorksheetFunction.ImCos Method (Excel)

Returns the cosine of a complex number in x + yi or x + yj text format.


## Syntax

 _表达式_. **ImCos**( ** _Arg1_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Variant**|Inumber - a complex number for which you want the cosine.|

### Return Value

String


## Remarks




- Use COMPLEX to convert real and imaginary coefficients into a complex number.
    
- If inumber is a logical value, IMCOS returns the #VALUE! error value.
