
# ShapeRange.Select Method (Excel)

Selects the object.


## Syntax

 _表达式_. **Select**( ** _Replace_** )

 _表达式_ A variable that represents a **ShapeRange** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Replace_|可选|**Variant**| (used only with sheets). **True** to replace the current selection with the specified object. **False** to extend the current selection to include any previously selected objects and the specified object.|
