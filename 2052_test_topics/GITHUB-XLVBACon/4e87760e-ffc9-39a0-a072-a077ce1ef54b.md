
# WorksheetFunction.Find Method (Excel)

Finds specific information in a worksheet.


## Syntax

 _表达式_. **Find**( ** _Arg1_**, ** _Arg2_**, ** _Arg3_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**String**|The name of the worksheet.|
| _Arg2_|必需|**String**|The name of the range.|
| _Arg3_|可选|**Variant**|The name of an argument to refine the search.|

### Return Value

Double
