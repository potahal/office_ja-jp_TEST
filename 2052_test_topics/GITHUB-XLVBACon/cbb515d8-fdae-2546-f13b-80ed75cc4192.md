
# MinorTickMark 属性

返回或设置指定坐标轴的次要刻度线的类型。读/写 XlTickMark。


||
|:-----|
|XlTickMark 可为以下 XlTickMark 常量之一。|
|**xlTickMarkCross**|
|**xlTickMarkInside**|
|**xlTickMarkNone**|
|**xlTickMarkOutside**|

 _expression_. **MinorTickMark**

 _表达式_ 必需。该表达式返回"应用于"列表中的一个对象。

## 示例

以下示例使数值轴的次要刻度线处于坐标轴的内侧。


```
myChart.Axes(xlValue).MinorTickMark = xlTickMarkInside
```
