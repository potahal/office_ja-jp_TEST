
# WorksheetFunction.Quotient Method (Excel)

Returns the integer portion of a division. Use this function when you want to discard the remainder of a division.


## Syntax

 _表达式_. **Quotient**( ** _Arg1_**, ** _Arg2_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Variant**|Numerator - the dividend.|
| _Arg2_|必需|**Variant**|Denominator - the divisor.|

### Return Value

Double


## Remarks

If either argument is nonnumeric, QUOTIENT returns the #VALUE! error value.
