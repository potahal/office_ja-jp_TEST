
# ReadingOrder 属性

返回或设置指定对象的阅读次序。可以是下列 **常量** 之一。读/写 **长** 。


||
|:-----|
|**xlContext**|
|**xlLTR** （从左到右）|
|**xlRTL** （从右到左）|

 _expression_. **CharacterType**

 _表达式_ 必需。该表达式返回"应用于"列表中的一个对象。

## 说明

某些常量可能不可用，这取决于所选择或安装的语言支持（例如，美国英语）。


## 示例

以下示例将图表标题的阅读次序设置为从右到左。


```
myChart.ChartTitle.ReadingOrder = xlRTL
```
