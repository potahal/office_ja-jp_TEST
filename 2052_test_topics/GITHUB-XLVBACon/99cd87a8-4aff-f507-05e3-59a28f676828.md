
# ModelFormatCurrency.DecimalPlaces Property (Excel)

Specifies the number of decimal places after the dot. Read/write  **Long**.在 Office 2016 中引入。


## Syntax

 _表达式_. **DecimalPlaces**

 _表达式_ A variable that represents a **ModelFormatCurrency** object.
