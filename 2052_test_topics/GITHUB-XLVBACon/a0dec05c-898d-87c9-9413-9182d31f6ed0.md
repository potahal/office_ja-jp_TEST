
# FormatCondition.Modify Method (Excel)

Modifies an existing conditional format.


## Syntax

 _表达式_. **Modify**( ** _Type_**, ** _Operator_**, ** _Formula1_**, ** _Formula2_** )

 _表达式_ A variable that represents a **FormatCondition** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Type_|必需|**[XlFormatConditionType](ae97c695-f56a-c9ee-91b0-dac413c93428.md)**|Specifies whether the conditional format is based on a cell value or an expression.|
| _Operator_|可选|**Variant**|An  **[XlFormatConditionOperator](f395f440-0870-513d-a1ae-0a15262f1a7a.md)** value that represents the conditional format operator. This parameter is ignored if _Type_ is set to **xlExpression**.|
| _Formula1_|可选|**Variant**|The value or expression associated with the conditional format. Can be a constant value, a string value, a cell reference, or a formula.|
| _Formula2_|可选|**Variant**|The value or expression associated with the conditional format. Can be a constant value, a string value, a cell reference, or a formula..|

## Example

This example modifies an existing conditional format for cells E1:E10.


```
Worksheets(1).Range("e1:e10").FormatConditions(1) _ 
 .Modify xlCellValue, xlLess, "=$a$1"
```
