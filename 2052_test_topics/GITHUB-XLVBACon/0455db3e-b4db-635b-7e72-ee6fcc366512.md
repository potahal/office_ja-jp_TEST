
# ModelFormatWholeNumber.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.在 Office 2016 中引入。


## Syntax

 _表达式_. **Parent**

 _表达式_ A variable that represents a **ModelFormatWholeNumber** object.
