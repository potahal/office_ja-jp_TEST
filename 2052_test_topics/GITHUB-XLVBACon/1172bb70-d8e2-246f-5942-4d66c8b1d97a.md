
# PivotAxis.PivotLines Property (Excel)

Returns the PivotLines attached to the specified  **PivotAxis** object. Read-only.


## Syntax

 _表达式_. **PivotLines**

 _表达式_ A variable that represents a **PivotAxis** object.
