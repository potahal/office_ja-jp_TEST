
# DataFeedConnection.RefreshOnFileOpen Property (Excel)

 **True** if the connection is automatically updated each time the workbook is opened. The default value is **False**.


## Syntax

 _表达式_. **RefreshOnFileOpen**

 _表达式_ A variable that represents a[DataFeedConnection Object (Excel)](2ccb242b-28d5-3baf-78be-aa8f7478f4b6.md) object.


## Property value

 **BOOL**
