
# ScaleType 属性

返回或设置数值轴的刻度类型。仅应用于数值轴。读/写 XlScaleType。


||
|:-----|
|XlScaleType 可为以下 XlScaleType 常量之一。|
|**xlScaleLinear**|
|**xlScaleLogarithmic**|

 _expression_. **ScaleType**

 _表达式_ 必需。该表达式返回"应用于"列表中的一个对象。

## 说明

对数刻度使用以 10 为底的对数。


## 示例

以下示例使数值轴使用对数刻度。


```
myChart.Axes(xlValue).ScaleType = xlScaleLogarithmic
```
