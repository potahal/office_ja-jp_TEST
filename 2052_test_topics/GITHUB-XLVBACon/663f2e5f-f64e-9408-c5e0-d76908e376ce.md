
# WorksheetFunction.Arabic Method (Excel)

Converts a Roman numeral to an Arabic numeral.


## Syntax

 _表达式_. **Arabic** _(Arg1)_

 _表达式_ A variable that represents a[WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|STRING|The Roman numeral you want to convert.|

### Return value

 **DOUBLE**
