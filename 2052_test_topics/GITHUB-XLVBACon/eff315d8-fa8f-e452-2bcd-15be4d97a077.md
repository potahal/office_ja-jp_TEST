
# Range.Merge Method (Excel)

Creates a merged cell from the specified  **[Range](b8207778-0dcc-4570-1234-f130532cc8cd.md)** object.


## Syntax

 _表达式_. **Merge**( ** _Across_** )

 _表达式_ A variable that represents a **Range** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Across_|可选|**Variant**|**True** to merge cells in each row of the specified range as separate merged cells. The default value is **False**.|

## Remarks

The value of a merged range is specified in the cell of the range's upper-left corner.
