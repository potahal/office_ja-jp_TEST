
# WorksheetFunction.Phonetic Method (Excel)

Extracts the phonetic (furigana) characters from a text string.


## Syntax

 _表达式_. **Phonetic**( ** _Arg1_** )

 _表达式_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Range**|Reference - a text string or a reference to a single cell or a range of cells that contain a furigana text string.|

### Return Value

String


## Remarks




- If reference is a range of cells, the furigana text string in the upper-left corner cell of the range is returned.
    
- If the reference is a range of nonadjacent cells, the #N/A error value is returned.
