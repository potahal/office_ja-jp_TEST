
# Range.Hyperlinks Property (Excel)

Returns a  **[Hyperlinks](de28e0af-7a4c-56c3-5fe5-ac47d1654628.md)** collection that represents the hyperlinks for the range.


## Syntax

 _表达式_. **Hyperlinks**

 _表达式_ A variable that represents a **Range** object.
