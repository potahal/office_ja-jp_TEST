
# Application.WorkbookModelChange Event (Excel)

Occurs when the data model is updated.


## Syntax

 _表达式_. **WorkbookModelChange** _(Wb,_ _Changes)_

 _表达式_ A variable that represents a **Application** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Wb_|必需|WORKBOOK|The workbook|
| _Changes_|必需|MODELCHANGES|The changes to the data model.|
