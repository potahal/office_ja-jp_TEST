
# WorksheetFunction.Sech Method (Excel)

Returns the hyperbolic secant of an angle.


## Syntax

 _表达式_. **Sech** _(Arg1)_

 _表达式_ A variable that represents a[WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|DOUBLE|Angle in radians for which you want the hyperbolic secant.|

### Return value

 **DOUBLE**
