
# ShapeRange.TextFrame2 Property (Excel)

Returns a  **[TextFrame2](66ba23e5-9b15-b954-a1db-1bd19b4eb90d.md)** object that contains text formatting for the specified shape range. Read-only.


## Syntax

 _表达式_. **TextFrame2**

 _表达式_ A variable that represents a **ShapeRange** object.
