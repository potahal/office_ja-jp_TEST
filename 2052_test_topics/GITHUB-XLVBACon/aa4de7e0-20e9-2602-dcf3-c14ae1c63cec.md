
# SparklineGroup.Count Property (Excel)

Returns the number of sparklines in the sparkline group. Read-only.


## Syntax

 _表达式_. **Count**

 _表达式_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

Long
