
# WorksheetFunction.NumberValue Method (Excel)

Converts text to number in a locale-independent manner.


## Syntax

 _表达式_. **NumberValue** _(Arg1,_ _Arg2,_ _Arg3)_

 _表达式_ A variable that represents a[WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|STRING|String representing the number you want to convert.|
| _Arg2_|必需|STRING|Character used as the decimal separator in the string.|
| _Arg3_|必需|STRING|Character used as the group separator in the string.|

### Return value

 **DOUBLE**
