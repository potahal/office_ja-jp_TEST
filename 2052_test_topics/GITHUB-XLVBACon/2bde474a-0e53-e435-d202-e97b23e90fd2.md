
# Series.Has3DEffect Property (Excel)

 **True** if the series has a three-dimensional appearance. Read/write **Boolean**.


## Syntax

 _表达式_. **Has3DEffect**

 _表达式_ A variable that represents a **Series** object.


## Remarks

This property applies to bubble charts only.
