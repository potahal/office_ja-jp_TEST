
# SetEchoOn 方法

返回 Chart 对象。

 _expression_. **SetEchoOn** ( ** _EchoOn_** )

 _表达式_ 所必需。返回一个 **Chart** 对象表达式。

 **EchoOn** **Variant** 类型，可选。

## 示例

以下示例设置应用程序中第一个对象的回响。


```
Sub UseEchoOn() 
 
 Dim grpOne As Graph.Chart 
 
 Set grpOne = Application.ActiveSheet.OLEObjects(1).Object 
 
 grpOne.SetEchoOn 
 
End Sub
```
