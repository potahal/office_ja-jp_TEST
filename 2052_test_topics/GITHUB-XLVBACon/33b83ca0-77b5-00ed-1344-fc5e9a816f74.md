
# AutoCorrect.AddReplacement Method (Excel)

Adds an entry to the array of AutoCorrect replacements.


## Syntax

 _表达式_. **AddReplacement**( ** _What_**, ** _Replacement_** )

 _表达式_ A variable that represents an **AutoCorrect** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _What_|必需|**String**|The text to be replaced. If this string already exists in the array of AutoCorrect replacements, the existing substitute text is replaced by the new text.|
| _Replacement_|必需|**String**|The replacement text.|

### Return Value

Variant


## Example

This example substitutes the word "Temp." for the word "Temperature" in the array of AutoCorrect replacements.


```
With Application.AutoCorrect 
 .AddReplacement "Temperature", "Temp." 
End With
```
