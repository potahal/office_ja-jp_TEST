
# WorksheetFunction.LogNormDist Method (Excel)

Returns the cumulative lognormal distribution of x, where ln(x) is normally distributed with parameters mean and standard_dev. Use this function to analyze data that has been logarithmically transformed.


 **重要信息**  This function has been replaced with one or more new functions that may provide improved accuracy and whose names better reflect their usage. This function is still available for compatibility with earlier versions of Excel. However, if backward compatibility is not required, you should consider using the new functions from now on, because they more accurately describe their functionality.

For more information about the new functions, see the [LogNorm_Dist](df3510f3-0518-9e65-f9e9-af393c3113e1.md) method.

## Syntax

 _表达式_. **LogNormDist**( ** _Arg1_**, ** _Arg2_**, ** _Arg3_** )

 _表达式_ A variable that represents a **[WorksheetFunction](7b1d5639-363d-632c-2cf0-2232562646b6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|必需|**Double**|X - the value at which to evaluate the function.|
| _Arg2_|必需|**Double**|Mean - the mean of ln(x).|
| _Arg3_|必需|**Double**|Standard_dev - the standard deviation of ln(x).|

### Return Value

Double


## Remarks




- If any argument is nonnumeric, LOGNORMDIST returns the #VALUE! error value.
    
- If x ? 0 or if standard_dev ? 0, LOGNORMDIST returns the #NUM! error value.
