
# AllowEditRange.Title Property (Excel)

Returns or sets the title of the range of cells that can edited on a protected sheet. Read/write  **String**.


## Syntax

 _表达式_. **Title**

 _表达式_ A variable that represents an **AllowEditRange** object.
