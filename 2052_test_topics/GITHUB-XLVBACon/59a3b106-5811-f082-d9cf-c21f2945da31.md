
# Weight 属性

返回或设置边框的粗细。读/写
 **XlBorderWeight**
 。


||
|:-----|
|XlBorderWeight 可为以下 XlBorderWeight 常量之一。|
|**xlHairline**|
|**xlThin** **xlMedium** **xlThick**|

 _expression_. **Weight**

 _表达式_ 必需。该表达式返回"应用于"列表中的一个对象。

## 示例

以下示例设置图表区的边框粗细。


```
myChart.ChartArea.Border.Weight = xlMedium
```
