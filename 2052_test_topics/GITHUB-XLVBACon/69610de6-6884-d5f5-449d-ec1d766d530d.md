
# Worksheet.TableUpdate Event (Excel)

Occurs after a Query table connected to the Data Model is updated on a worksheet.


## Syntax

 _表达式_. **TableUpdate** _(Target)_

 _表达式_ A variable that represents a **Worksheet** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Target_|必需|TABLEOBJECT|The Query table with data from the Data Model.|
