
# SparklineGroup.Points Property (Excel)

Returns the associated  **[SparkPoints](382bf292-7824-179f-e254-1b72dfb557b2.md)** object for the sparkline group. Read-only.


## Syntax

 _表达式_. **Points**

 _表达式_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

SparkPoints


## Property value

 **SPARKPOINTS**
