
# HasUpDownBars 属性

如果指定的折线图有涨跌柱线，则该属性值为  **True** 。仅适用于折线图。 **Boolean** 类型，可读写。


## 示例

本示例打开第一个图表组的涨跌柱线，并设置其颜色。本示例应在包含两个数据系列的二维折线图上运行，这两个数据系列应在一个或多个数据点处相交。


```
With myChart.ChartGroups(1) 
 .HasUpDownBars = True 
 .DownBars.Interior.ColorIndex = 3 
 .UpBars.Interior.ColorIndex = 5 
End With
```
