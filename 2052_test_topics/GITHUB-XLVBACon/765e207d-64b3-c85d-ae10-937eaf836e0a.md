
# AutoCorrect.DeleteReplacement Method (Excel)

Deletes an entry from the array of AutoCorrect replacements.


## Syntax

 _表达式_. **DeleteReplacement**( ** _What_** )

 _表达式_ A variable that represents an **AutoCorrect** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _What_|必需|**String**|The text to be replaced, as it appears in the row to be deleted from the array of AutoCorrect replacements. If this string doesn't exist in the array of AutoCorrect replacements, this method fails.|

### Return Value

Variant


## Example

This example removes the word "Temperature" from the array of AutoCorrect replacements.


```
With Application.AutoCorrect 
 .DeleteReplacement "Temperature" 
End With
```
