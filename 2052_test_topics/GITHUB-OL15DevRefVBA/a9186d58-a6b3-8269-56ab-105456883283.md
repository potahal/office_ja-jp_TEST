
# TaskItem.InternetCodepage 属性 (Outlook)

返回或设置一个 **Long** ，确定项目所用的互联网代码页。读/写。


## 语法

 _表达式_. **InternetCodepage**

 _表达式_ 一个表示 **TaskItem** 对象的变量。


## 注解

Internet 代码页定义了项目使用的文本编码方案。
