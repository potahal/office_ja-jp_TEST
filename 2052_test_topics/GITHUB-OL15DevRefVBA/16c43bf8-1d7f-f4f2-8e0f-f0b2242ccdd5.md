
# JournalItem.Duration 属性 (Outlook)

返回或设置 **Long** 值，表明 **[JournalItem](6e850295-39f9-47b8-e866-9622e9958c69.md)** 写的持续时间 (以分钟为单位)。


## 语法

 _表达式_. **Duration**

 _表达式_ 一个代表 **JournalItem** 对象的变量。
