
# OlkTextBox.Enter 事件 (Outlook)

当控件获得焦点时前, 一个控件的 **Exit** 事件之后立即发生。


## 语法

 _表达式_. **Enter**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

这是类似于 **GotFocus** 事件公开由托管代码控件或 Microsoft Visual Basic 6 中。
