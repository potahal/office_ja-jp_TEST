
# Pages.Item 方法 （Outlook）

从集合中返回一页。


## 语法

 _表达式_. **Item**( ** _Index_** )

 _表达式_ 一个代表 **Pages** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _varg_|必需|**Variant**|或者是对象的索引号，或者是用于匹配集合中某个对象的默认属性的值。|

### 返回值

一个 **Object**值，表示指定的对象。
