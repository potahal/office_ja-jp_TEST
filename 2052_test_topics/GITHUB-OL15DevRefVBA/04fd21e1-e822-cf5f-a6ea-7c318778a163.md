
# OlkComboBox.ListCount 属性 (Outlook)

返回一个 **Long** ，组合的下拉列表中指定的元素数框控件。只读的。


## 语法

 _表达式_. **ListCount**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

默认值为零，表示列表中没有项目。
