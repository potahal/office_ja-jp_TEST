
# ReportItem.Subject 属性 (Outlook)

返回或设置一个 **字符串** ，表示 Outlook 项目的主题。读/写。


## 语法

 _表达式_. **Subject**

 _表达式_ 一个表示 **ReportItem** 对象的变量。


## 注解

此属性对应于 **PidTagSubject** 的 MAPI 属性。 **主题** 属性是 Outlook 项目的默认属性。
