
# PeopleView.SaveOption 属性 (Outlook)
返回一个 [OlViewSaveOption](c08bab4d-ecdd-a2ac-1cdc-fa910f9585e0.md) 常量，该常量指定所指定的视图在哪些文件夹中可用以及附加到该视图的读取权限。只读。

## 版本信息

添加的版本： Outlook 2013


## 语法

 _表达式_. **SaveOption**

 _表达式_ 一个表示 **PeopleView**对象的变量。


## 属性值

 **OLVIEWSAVEOPTION**
