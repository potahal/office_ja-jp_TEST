
# MailItem.CreationTime 属性 (Outlook)

返回 **日期** 的 Outlook 项目的创建时间。只读的。


## 语法

 _表达式_. **CreationTime**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性对应于 **PidTagCreationTime** 的 MAPI 属性。
