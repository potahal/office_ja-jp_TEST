
# Selection.Count 属性 (Outlook)

返回一个 **Long** ，指示数项目或所选内容中的对话头。只读的。


## 语法

 _表达式_. **Count**

 _表达式_ 一个代表 **[Selection](0b06a3ce-0445-db8f-e6e8-bb7bd469c50f.md)** 对象的变量。
