
# RemoteItem.Companies 属性 (Outlook)

返回或设置一个 **字符串** ，表示与 Outlook 项关联的公司名称。读/写。


## 语法

 _表达式_. **Companies**

 _表达式_ 一个代表 **RemoteItem** 对象的变量。


## 注解

它是自由格式的文本字段。
