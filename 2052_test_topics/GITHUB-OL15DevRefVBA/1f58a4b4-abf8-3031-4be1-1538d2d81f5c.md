
# MailItem.SentOnBehalfOfName 属性 (Outlook)

返回一个 **字符串** ，指示对电子邮件的预定发件人显示名称。读/写。


## 语法

 _表达式_. **SentOnBehalfOfName**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性对应于 **PidTagSentRepresentingName** 的 MAPI 属性。
