
# ContactItem.SelectedMailingAddress 属性 (Outlook)

返回或设置一个  **[OlMailingAddress](1a0822be-3c4c-f162-8ab7-12edb20a0949.md)** 常量，该常量指示联系人的通讯地址类型。可读/写。


## 语法

 _表达式_. **SelectedMailingAddress**

 _表达式_ 一个代表 **ContactItem** 对象的变量。
