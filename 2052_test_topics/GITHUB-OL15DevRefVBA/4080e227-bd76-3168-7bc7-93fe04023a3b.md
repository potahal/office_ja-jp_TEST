
# NameSpace.DefaultStore 属性 (Outlook)

返回一个  **[Store](1eb22fe9-8849-7476-5388-2515b48591b9.md)** 对象，该对象代表配置文件的默认存储区。只读。


## 语法

 _表达式_. **DefaultStore**

 _表达式_ 一个表示 **NameSpace** 对象的变量。


## 说明

如果当前配置文件没有存储区，则返回错误。
