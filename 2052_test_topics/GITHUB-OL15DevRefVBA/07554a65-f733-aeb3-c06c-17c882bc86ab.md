
# NoteItem.Save 方法 （Outlook）

将 Microsoft Outlook项目保存到当前文件夹中，如果是新项目，则保存到该项目类型的 Outlook 默认文件夹中。


## 语法

 _表达式_. **Save**

 _表达式_ 一个代表 **NoteItem** 对象的变量。
