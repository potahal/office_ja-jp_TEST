
# MailItem.ReplyRecipients 属性 (Outlook)

返回表示 Outlook 项目的所有答复收件人对象的 **[收件人](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** 集合。只读的。


## 语法

 _表达式_. **ReplyRecipients**

 _表达式_ 一个代表 **MailItem** 对象的变量。
