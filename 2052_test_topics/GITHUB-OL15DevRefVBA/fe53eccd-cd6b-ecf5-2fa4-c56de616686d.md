
# MeetingItem.PrintOut 方法 （Outlook）

打印 Outlook 项目使用所有默认设置。 **PrintOut** 方法是唯一可用于打印的 Outlook 方法。


## 语法

 _表达式_. **PrintOut**

 _表达式_ 一个代表 **MeetingItem** 对象的变量。
