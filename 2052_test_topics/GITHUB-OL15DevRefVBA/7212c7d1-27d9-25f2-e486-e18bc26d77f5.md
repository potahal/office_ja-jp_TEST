
# OlkCommandButton.AutoSize 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，自动调整大小以显示整个内容的控件。读/写。


## 语法

 _表达式_. **AutoSize**

 _表达式_ 一个代表 **OlkCommandButton** 对象的变量。


## 说明

此属性的默认值为 **False** 。

对于按钮控件，此属性指定该控件是否自动调整以显示整个标题值。
