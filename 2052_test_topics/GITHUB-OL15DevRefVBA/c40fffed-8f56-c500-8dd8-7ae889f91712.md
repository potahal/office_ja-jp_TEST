
# SharingItem.Reply 方法 （Outlook）

从原始  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 创建答复，预先将地址填写为原始发件人。


## 语法

 _表达式_. **Reply**

 _表达式_ 一个代表 **SharingItem** 对象的变量。


### 返回值

一个代表答复的  **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** 对象。
