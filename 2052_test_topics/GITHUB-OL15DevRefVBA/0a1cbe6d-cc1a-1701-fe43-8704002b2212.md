
# BusinessCardView.CardSize 属性 (Outlook)

返回或设置一个 **Long** 类型，表示的大小，以百分比表示，电子名片 (EBC) 视图中。读/写。


## 语法

 _表达式_. **CardSize**

 _表达式_ 一个返回 **BusinessCardView** 对象的表达式。


## 说明

此属性可以设置为 20 和 100 之间的值。如果此属性设置为小于 20 的值，则会将此属性设置为 20。如果此属性设置为大于 100 的值，则会将此属性设置为 100。
