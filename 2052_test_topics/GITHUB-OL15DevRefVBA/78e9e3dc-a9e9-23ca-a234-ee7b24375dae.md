
# TaskRequestAcceptItem.OutlookInternalVersion 属性 (Outlook)

返回一个 **Long** 表示 Outlook 项的 Outlook 应用程序内部版本号。只读的。


## 语法

 _表达式_. **OutlookInternalVersion**

 _表达式_ 一个代表 **TaskRequestAcceptItem** 对象的变量。
