
# OlkTimeZoneControl.DropButtonClick 事件 (Outlook)

用户在时区控件中单击下拉按钮以展开下拉列表时，或者以编程方式调用  **[DropDown](47b8d231-d3dc-4f10-a8ab-ead1853bad0f.md)** 方法时发生。


## 语法

 _表达式_. **DropButtonClick**

 _表达式_ 一个代表 **OlkTimeZoneControl** 对象的变量。
