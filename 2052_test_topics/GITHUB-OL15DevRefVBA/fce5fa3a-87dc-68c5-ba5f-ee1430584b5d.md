
# Store.StoreID 属性 (Outlook)

返回标识此 **[存储区](1eb22fe9-8849-7476-5388-2515b48591b9.md)** 的 **字符串** 。只读的。


## 语法

 _表达式_. **StoreID**

 _表达式_ 一个代表 **Store** 对象的变量。


## 说明

 **StoreID** 是唯一的配置文件。它相当于 **PidTagStoreEntryId** 的 MAPI 属性。
