
# Inspector.BeforeMinimize 事件 (Outlook)

当用户最小化活动检查器时发生。


## 语法

 _表达式_. **BeforeMinimize**( ** _Cancel_** )

 _表达式_ 一个代表 **Inspector** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Cancel_|必需|**Boolean**|**假** 的事件发生时。如果该事件过程将此参数设置为 **True** ，则不完成该操作，并将浏览器或检查器不最小化。|

## 注解

此事件在开始后可以取消。
