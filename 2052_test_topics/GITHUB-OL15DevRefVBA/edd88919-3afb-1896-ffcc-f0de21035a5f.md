
# TimeZones.CurrentTimeZone 属性 (Outlook)

返回一个  **[TimeZone](b27da70d-e545-cc13-9529-cfd327ab7a7c.md)** 类型的值，该值代表当前 Windows 系统本地时区。只读。


## 语法

 _表达式_. **CurrentTimeZone**

 _表达式_ 一个代表 **TimeZones** 对象的变量。
