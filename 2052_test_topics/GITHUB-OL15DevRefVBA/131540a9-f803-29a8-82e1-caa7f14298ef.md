
# Store.RefreshQuotaDisplay 方法 （Outlook）

刷新资源管理器窗口的状态栏中显示的存储配额信息。


## 语法

 _表达式_. **RefreshQuotaDisplay**

 _表达式_ 一个代表 **[Store](1eb22fe9-8849-7476-5388-2515b48591b9.md)** 对象的变量。


## 说明

如果存储区不支持配额信息的显示，则此方法将不起任何作用。
