
# OlAlign 枚举 （Outlook）

指定  **[ColumnFormat](acbbdd97-e695-d1e7-c7ba-24f75efbf22c.md)** 对象的 **[Label](cf104506-3eca-6695-3d3b-05022ce6fba4.md)** 属性的对齐方式。



|**名称**|**值**|**说明**|
|:-----|:-----|:-----|
|**olAlignCenter**|1|表示指定列的标签应该居中。|
|**olAlignLeft**|0|表示指定列的标签应该左对齐。|
|**olAlignRight**|2|表示指定列的标签应该右对齐。|
