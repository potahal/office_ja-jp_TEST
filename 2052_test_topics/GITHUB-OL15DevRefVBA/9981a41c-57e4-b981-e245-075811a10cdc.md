
# 所枚举 （Outlook）

指定响应风格。



|**名称**|**值**|**说明**|
|:-----|:-----|:-----|
|**olOpen**|0|指示将打开一个窗体。|
|**olPrompt**|2|指示将提示用户打开或发送窗体。|
|**olSend**|1|指示将立即发送窗体。|

## 注解

由 [Action](22bd8d4a-9cf4-bd37-011b-8da3dfadf761.md) 的[ResponseStyle](6c20276c-51c1-3164-a28f-ac415c911cbb.md) 属性用来指定执行 **Action** 时将使用的响应风格。
