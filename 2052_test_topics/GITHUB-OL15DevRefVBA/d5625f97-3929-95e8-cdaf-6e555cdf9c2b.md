
# MailItem.ConversationTopic 属性 (Outlook)

返回一个 **字符串** ，表示的 Outlook 项目的会话线索的主题。只读的。


## 语法

 _表达式_. **ConversationTopic**

 _表达式_ 一个代表 **MailItem** 对象的变量。
