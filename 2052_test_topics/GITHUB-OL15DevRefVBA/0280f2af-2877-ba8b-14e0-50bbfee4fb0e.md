
# RuleAction.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **RuleAction** 对象的变量。


## 说明

 **[RuleAction](6451788f-e5ed-239c-a34d-b564b52d8955.md)** 对象的父对象是 **[RuleActions](82ba76cd-86a4-3372-cb51-2df1d58c8b71.md)** 对象。
