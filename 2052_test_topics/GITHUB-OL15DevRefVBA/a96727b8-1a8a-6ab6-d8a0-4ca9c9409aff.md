
# AppointmentItem.ResponseRequested 属性 (Outlook)

返回一个 **boolean 类型的值** ，指示如果发件人想要约会的会议请求的响应 **，则返回 True** 。读/写。


## 语法

 _表达式_. **ResponseRequested**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
