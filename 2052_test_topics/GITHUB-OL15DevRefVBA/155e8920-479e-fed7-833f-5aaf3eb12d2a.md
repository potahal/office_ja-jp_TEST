
# CategoryRuleCondition.ConditionType 属性 (Outlook)

返回常量，  **olConditionCategory** ，从 **[OlRuleConditionType](35c2f965-0f9d-8cc8-2f05-60522268574f.md)** 枚举，指示规则条件的类型。只读的。


## 语法

 _表达式_. **ConditionType**

 _表达式_ 一个代表 **CategoryRuleCondition** 对象的变量。


## 说明

类别规则条件受邮件接收规则和邮件发送规则支持。有关详细信息，请参阅[指定规则条件](http://msdn.microsoft.com/library/812c131a-fe23-1b8b-5e2d-9459d7102630%28Office.15%29.aspx)。
