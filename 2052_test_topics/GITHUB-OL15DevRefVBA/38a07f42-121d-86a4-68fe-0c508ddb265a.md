
# AppointmentItem.BusyStatus 属性 (Outlook)

返回或设置一个  **[OlBusyStatus](4391ccb4-a035-30d1-9693-61b83050b31f.md)** 常量，该常量指示参加约会的用户的忙碌状态。可读/写。


## 语法

 _表达式_. **BusyStatus**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
