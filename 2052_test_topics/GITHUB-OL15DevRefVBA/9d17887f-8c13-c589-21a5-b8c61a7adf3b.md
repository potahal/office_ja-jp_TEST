
# Accounts.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **Accounts** 对象的变量。


## 说明

 **[Accounts](2510b7d7-5062-8ea3-dda4-b544d2882a2b.md)** 对象的父对象是 **[NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** 对象。
