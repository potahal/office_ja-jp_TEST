
# AddressEntry.Name 属性 (Outlook)

返回或设置一个 **String** 值，该值代表对象的显示名称。读/写。


## 语法

 _表达式_. **Name**

 _表达式_ 一个代表 **AddressEntry** 对象的变量。


## 注解

有必要为 **Name** 属性在调用 **[AddressEntry.Details](85457da6-c97a-387d-6c7e-40eb005b25aa.md)** 方法之前设置。
