
# CalendarView.BoldDatesWithItems 属性 (Outlook)

返回或设置一个 **布尔** 值，该值指示是否将包含 Outlook 项目的日期将显示在 **[日历视图](37e078b9-9fc6-5894-b043-06d7257666a8.md)** 对象的日期选择区中加粗。读/写。


## 语法

 _表达式_. **BoldDatesWithItems**

 _表达式_ 一个代表 **CalendarView** 对象的变量。
