
# TaskItem.CancelResponseState 方法 （Outlook）

将未发送的任务要求响应重置为简单任务。


## 语法

 _表达式_. **CancelResponseState**

 _表达式_ 一个代表 **TaskItem** 对象的变量。


## 注解

在接收和响应任务要求之后和发送响应之前的这段时间，可使用此方法将任务回复到响应前的状态。
