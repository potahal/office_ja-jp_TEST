
# OlkComboBox.AutoSize 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，自动调整大小以显示整个内容的控件。 读/写。


## 语法

 _表达式_. **AutoSize**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

此属性的默认值为 **False** 。

 **调整** 设置为 **True** 可自动设置显示区域的宽度的文本长度在组合框中。此行为是类似于单行文本框中。
