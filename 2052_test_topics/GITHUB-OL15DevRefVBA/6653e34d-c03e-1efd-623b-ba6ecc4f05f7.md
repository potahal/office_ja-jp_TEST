
# OlkCategory.BackStyle 属性 (Outlook)

返回或设置一个  **[olBackStyle](54ed2253-fe39-9e91-e15a-8e9072d0c257.md)** 常量，该常量使控件的背景样式在透明和不透明之间更改。可读/写。


## 语法

 _表达式_. **BackStyle**

 _表达式_ 一个代表 **OlkCategory** 对象的变量。


## 说明

默认值是 **olBackStyleOpaque** 。
