
# MailItem.CC 属性 (Outlook)

返回一个 **字符串** ，表示 **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** 的抄送 (CC) 名称的显示列表。读/写。


## 语法

 _表达式_. **CC**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性包含的显示名称。 **[收件人](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** 集合用于修改抄送收件人。
