
# ContactItem.IsMarkedAsTask 属性 (Outlook)

返回一个 **布尔** 值，该值指示是否将 **[联系人](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** 标记为一项任务。只读的。


## 语法

 _表达式_. **IsMarkedAsTask**

 _表达式_ 一个返回 **ContactItem** 对象的表达式。
