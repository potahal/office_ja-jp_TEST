
# Rule.RuleType 属性 (Outlook)

从  **[OlRuleType](a9ef16ad-78cf-8c26-0897-39a0bf1a25b1.md)** 枚举中返回一个常量，该常量指示该规则是应用于发送的邮件，还是应用于接收的邮件。只读。


## 语法

 _表达式_. **RuleType**

 _表达式_ 一个代表 **Rule** 对象的变量。
