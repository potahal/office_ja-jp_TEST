
# FormDescription.Template 属性 (Outlook)

返回或设置一个 **字符串** ，表示窗体模板的名称。读/写。


## 语法

 _表达式_. **Template**

 _表达式_ 一个代表 **FormDescription** 对象的变量。


## 注解

此属性是指 Microsoft Office Word。点指示模板文件，因此它是适用于窗体与 **[UseWordMail](fcc3623d-a9d3-db1e-61cf-8a7f0b5dca6c.md)** = **True** 。
