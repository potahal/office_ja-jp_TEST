
# DistListItem.Saved 属性 (Outlook)

返回一个 **布尔** 值， **如果不自上次保存以来修改 Outlook 项目** 。只读的。


## 语法

 _表达式_. **Saved**

 _表达式_ 一个表示 **DistListItem** 对象的变量。
