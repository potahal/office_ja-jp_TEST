
# Exceptions.Item 方法 （Outlook）

从集合中返回一个  **[Exception](010552b0-9ba6-c81b-1e3a-fd6a681e5163.md)** 对象。


## 语法

 _表达式_. **Item**( ** _Index_** )

 _表达式_ 一个代表 **Exceptions** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Variant**|或者是对象的索引号，或者是用于匹配集合中某个对象的默认属性的值。|

### 返回值

 **异常** 对象，该对象代表指定的对象。
