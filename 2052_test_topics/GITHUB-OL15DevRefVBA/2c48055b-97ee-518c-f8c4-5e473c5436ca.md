
# OlkTimeControl.DropButtonClick 事件 (Outlook)

当用户在时间控件中单击下拉按钮展开时间下拉列表时发生。


## 语法

 _表达式_. **DropButtonClick**

 _表达式_ 一个代表 **OlkTimeControl** 对象的变量。
