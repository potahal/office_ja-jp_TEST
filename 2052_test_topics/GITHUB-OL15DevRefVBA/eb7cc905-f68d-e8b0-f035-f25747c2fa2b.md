
# DistListItem.Importance 属性 (Outlook)

返回或设置一个  **[OlImportance](71e04f9a-fab6-153f-b046-11f7ec50e8e4.md)** 常量，该常量指示 Outlook 项目的相对重要性级别。可读/写。


## 语法

 _表达式_. **Importance**

 _表达式_ 一个代表 **DistListItem** 对象的变量。


## 注解

此属性对应于 **PidTagImportance** 的 MAPI 属性。
