
# MailModule.Visible 属性 (Outlook)

返回或设置一个 **布尔** 值，确定是否 **[MailModule](df20efe5-be5c-952d-c6b7-20c20a83fda0.md)** 对象将显示在导航窗格中。读/写。


## 语法

 _表达式_. **Visible**

 _表达式_ 一个代表 **MailModule** 对象的变量。
