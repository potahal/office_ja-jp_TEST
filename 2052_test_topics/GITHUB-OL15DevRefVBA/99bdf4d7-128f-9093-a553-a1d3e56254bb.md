
# PostItem.SentOn 属性 (Outlook)

返回 **日期** 的日期和的时间已发送的 Outlook 项目。只读的。


## 语法

 _表达式_. **SentOn**

 _表达式_ 一个表示 **PostItem** 对象的变量。


## 注解

此属性对应于 **PidTagClientSubmitTime** 的 MAPI 属性。
