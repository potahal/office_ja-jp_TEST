
# TaskRequestUpdateItem.IsConflict 属性 (Outlook)

返回一个 **boolean 类型的值** ，用于确定该项目是否冲突。只读的。


## 语法

 _表达式_. **IsConflict**

 _表达式_ 一个表示 **TaskRequestUpdateItem** 对象的变量。


## 注解

某一项处于冲突取决于应用程序的状态。例如，如果用户在脱机时试图访问联机文件夹，该操作将失败。在此方案中， **操作** 属性将返回 **True** 。

如果 **True** ，指定的项有冲突。
