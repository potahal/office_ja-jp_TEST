
# OlkControl.MinimumWidth 属性 (Outlook)

返回或设置一个 **Long** ，指定该控件可以自动调整到的最小宽度。 读/写。


## 语法

 _表达式_. **MinimumWidth**

 _表达式_ 一个代表 **OlkControl** 对象的变量。


## 说明

 **[EnableAutoLayout](929f07f1-db56-9b31-910e-e0c83b57afde.md)** 必须设置为 **True** ，为了使 **MinimumWidth** 成为有效。
