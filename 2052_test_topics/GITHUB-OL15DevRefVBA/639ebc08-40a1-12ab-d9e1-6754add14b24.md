
# Attachment.Index 属性 (Outlook)

返回一个 **Long** ，该值指示集合中对象的位置。 只读的。


## 语法

 _表达式_. **Index**

 _表达式_ 一个代表 **Attachment** 对象的变量。


## 注解

 **Index** 属性仅在当前会话期间有效，并且在可以更改对象添加到集合和从集合中删除。集合中的第一个对象的 **索引** 值为 1。
