
# RuleCondition.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **RuleCondition** 对象的变量。


## 注解

 **[RuleCondition](e03f91c2-2c08-b036-104a-d6246f28bc2d.md)** 对象的父对象是 **[Rule](ea2ddbcc-fd65-a636-c6da-79950033f385.md)** 对象。
