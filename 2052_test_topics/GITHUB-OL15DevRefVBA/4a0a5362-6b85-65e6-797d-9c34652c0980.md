
# OlkComboBox.Cut 方法 （Outlook）

删除控件的内容并将内容复制到剪贴板。


## 语法

 _表达式_. **Cut**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

数据以无格式文本格式复制到剪贴板，用于替换剪贴板的现有内容。

如果控件样式是 **olComboBoxStyleListBox** ，然后此方法将不剪切任何内容。
