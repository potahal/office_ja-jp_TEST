
# OlkDateControl.Date 属性 (Outlook)

返回或设置当前选定的 **日期** 值，通过将时间设置为 12:00 AM。 读/写。


## 语法

 _表达式_. **Date**

 _表达式_ 一个代表 **OlkDateControl** 对象的变量。


## 说明

获取此属性时，默认值是时间设置为 12:00 AM 的当前日期。设置此属性时，任何时间信息都将被忽略；时间将被设置为 12:00 AM。
