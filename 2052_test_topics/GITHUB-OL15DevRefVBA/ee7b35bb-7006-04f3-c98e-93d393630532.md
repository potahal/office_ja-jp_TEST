
# ContactItem.NetMeetingAlias 属性 (Outlook)

返回或设置一个 **字符串** ，指示用户的 Microsoft 网络会议 ID 或别名。读/写。


## 语法

 _表达式_. **NetMeetingAlias**

 _表达式_ 一个代表 **ContactItem** 对象的变量。
