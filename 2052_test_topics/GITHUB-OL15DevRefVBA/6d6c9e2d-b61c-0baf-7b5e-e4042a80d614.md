
# DistListItem.BillingInformation 属性 (Outlook)

返回或设置一个 **字符串** ，表示 Outlook 项目与相关的计费信息。读/写。


## 语法

 _表达式_. **BillingInformation**

 _表达式_ 一个代表 **DistListItem** 对象的变量。


## 注解

它是自由格式的文本字段。
