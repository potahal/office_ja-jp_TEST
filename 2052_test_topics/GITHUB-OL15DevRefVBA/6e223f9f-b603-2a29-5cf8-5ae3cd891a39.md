
# CardView.ViewFields 属性 (Outlook)

返回一个  **[ViewFields](2516faed-ed11-6cb3-ce9c-b6afa788e909.md)** 对象，该对象代表一组用来在 **[CardView](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** 对象中显示 Outlook 项目的字段。只读。


## 语法

 _表达式_. **ViewFields**

 _表达式_ 一个代表 **CardView** 对象的变量。
