
# JournalItem.Type 属性 (Outlook)

返回或设置一个 **字符串** 表示自由格式字段中，通常会包含显示名称的日记帐分录的应用程序 (例如，"MSWord"。)读/写。


## 语法

 _表达式_. **Type**

 _表达式_ 一个代表 **JournalItem** 对象的变量。
