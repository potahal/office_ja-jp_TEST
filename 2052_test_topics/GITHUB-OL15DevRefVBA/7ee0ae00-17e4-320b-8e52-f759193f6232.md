
# OlUnifiedGroupFolderType 枚举 (Outlook)

指定的文件夹，以获得统一的组。由于组具有邮件和日历文件夹，您可以指定的 **olGroupMailFolder** 或 **olGroupCalendarFolder** 。在 Office 2016 中引入。


## 成员


||||
|:-----|:-----|:-----|
|**Name**|**值**|**说明**|
|**olGroupCalendarFolder**|1|分配日历文件夹。|
|**olGroupMailFolder**|0|分配的邮件文件夹。|
