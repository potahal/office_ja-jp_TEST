
# OlkControl.ItemProperty 属性 (Outlook)

返回或设置一个 **字符串** ，指定该控件绑定到 Outlook 对象模型中的属性的名称。读/写。


## 语法

 _表达式_. **ItemProperty**

 _表达式_ 一个代表 **OlkControl** 对象的变量。
