
# FormRegion.Close 事件 (Outlook)

代表窗体区域框架的关闭，此事件发生在与 Outlook 项目关联的检查器的  **[Close](5a83b3d3-6096-9e37-88b1-00f97c0bf8bd.md)** 事件之前。


## 语法

 _表达式_. **Close**

 _表达式_ 一个代表 **FormRegion** 对象的变量。
