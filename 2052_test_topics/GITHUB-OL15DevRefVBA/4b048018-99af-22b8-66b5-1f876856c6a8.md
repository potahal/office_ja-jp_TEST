
# AppointmentItem.AttachmentAdd 事件 (Outlook)

向父对象的实例中添加附件时发生。


## 语法

 _表达式_. **AttachmentAdd**( ** _Attachment_** )

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Attachment_|必需|**[Attachment](3e11582b-ac90-0948-bc37-506570bb287b.md)**|**附件** 已添加到该项目。|
