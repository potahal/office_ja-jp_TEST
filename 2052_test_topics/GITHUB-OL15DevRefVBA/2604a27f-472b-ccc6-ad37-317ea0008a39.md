
# OlkCheckBox.Accelerator 属性 (Outlook)

返回或设置一个 **String** 值，它表示的加速键或热键控件。读/写。


## 语法

 _表达式_. **Accelerator**

 _表达式_ 一个代表 **OlkCheckBox** 对象的变量。


## 说明

默认值为空字符串，表示没有热键。如果该属性设置为包含多个字符的字符串，则该值将被截断为只包含第一个字符。

不能在加速键中使用数字。
