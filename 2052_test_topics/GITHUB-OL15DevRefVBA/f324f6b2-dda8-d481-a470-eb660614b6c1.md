
# TaskRequestUpdateItem.Read 事件 (Outlook)

当用户打开父对象的实例进行编辑时发生。


## 语法

 _表达式_. **Read**

 _表达式_ 一个代表 **TaskRequestUpdateItem** 对象的变量。


## 注解

 **读取** 事件与 **[Open](6a92ed2f-44a7-79ad-9b7d-caba455fa27a.md)** 事件不同，每当用户在支持单元格内编辑或者在 **[检查器](d7384756-669c-0549-1032-c3b864187994.md)** 中打开该项目时的视图中选择项时，即发生 **读取** 。
