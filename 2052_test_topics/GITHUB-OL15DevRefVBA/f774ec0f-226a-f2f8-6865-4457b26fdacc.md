
# SharingItem.Subject 属性 (Outlook)

返回或设置一个 **字符串** ，表示 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 的主题。读/写。


## 语法

 _表达式_. **Subject**

 _表达式_ 一个表示 **SharingItem** 对象的变量。


## 说明

此属性对应于 **PidTagSubject** 的 MAPI 属性。 **主题** 属性是 Outlook 项目的默认属性。
