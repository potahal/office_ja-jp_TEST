
# SharingItem.BeforeDelete 事件 (Outlook)

在删除项目（父对象的一个实例）前发生。


## 语法

 _表达式_. **BeforeDelete**( ** _Item_**, ** _Cancel_** )

 _表达式_ 一个返回 **SharingItem** 对象的表达式。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Item_|必需|**对象**|正在删除的项目。|
| _Cancel_|必需|**Boolean**|**假** 的事件发生时。如果事件过程将此参数设置为 **True** ，则不完成该操作，不删除该项目。|

## 说明

为了在通过某项操作删除共享邮件时触发该事件，必须打开一个检查器。

每删除一个项目，该事件发生一次。
