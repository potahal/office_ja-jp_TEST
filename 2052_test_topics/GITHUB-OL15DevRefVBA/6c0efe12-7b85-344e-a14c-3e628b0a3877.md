
# OlkTextBox.Value 属性 (Outlook)

返回或设置一个 **variant 类型的值** ，该值表示控件的内容。读/写。


## 语法

 _表达式_. **Value**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

值相等，则 **[文本](1dc8c0ec-00d1-5066-f204-68f7e6b30a42.md)** ，只是为 **variant 类型的值** ，并且 **文本** 被强类型化为 **字符串** 。
