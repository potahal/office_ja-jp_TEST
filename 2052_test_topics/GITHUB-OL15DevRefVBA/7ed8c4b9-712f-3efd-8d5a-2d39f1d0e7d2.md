
# BusinessCardView.SortFields 属性 (Outlook)

返回一个  **[OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** 集合对象，该对象代表作为视图排序依据的所有字段。只读。


## 语法

 _表达式_. **SortFields**

 _表达式_ 一个返回 **BusinessCardView** 对象的表达式。
