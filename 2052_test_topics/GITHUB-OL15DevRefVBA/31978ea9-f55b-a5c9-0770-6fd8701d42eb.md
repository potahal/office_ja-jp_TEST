
# OrderFields.Count 属性 (Outlook)

返回一个 **Long** 值，指示指定集合中的 **[OrderField](4ae32270-bde9-3178-bca3-f8d145779d3d.md)** 对象的计数。只读的。


## 语法

 _表达式_. **Count**

 _表达式_ 一个代表 **OrderFields** 对象的变量。
