
# OlkTextBox.IntegralHeight 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，指定该控件是否显示全部文本行。读/写。


## 语法

 _表达式_. **IntegralHeight**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

如果此属性为 **True** 时，控件将显示填充行文本;否则，它显示部分行。 默认值为 **True** 。
