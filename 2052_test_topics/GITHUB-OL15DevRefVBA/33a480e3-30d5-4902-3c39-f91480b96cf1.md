
# MarkAsTaskRuleAction.FlagTo 属性 (Outlook)

返回或设置一个 **字符串** ，表示为邮件标记的标签。读/写


## 语法

 _表达式_. **FlagTo**

 _表达式_ 一个代表 **MarkAsTaskRuleAction** 对象的变量。


## 说明

如果 **FlagTo** 是一个空字符串，则返回错误。
