
# NewItemAlertRuleAction.Text 属性 (Outlook)

返回或设置一个 **字符串** ，表示在新邮件通知对话框中显示的文本。读/写。


## 语法

 _表达式_. **Text**

 _表达式_ 一个代表 **NewItemAlertRuleAction** 对象的变量。


## 说明

如果文本为空字符串，则返回错误。
