
# FormRegion.Detail 属性 (Outlook)

返回或设置一个表示后相邻窗体区域的显示名称在运行时，标头中显示的文本的 **字符串** 。读/写。


## 语法

 _表达式_. **Detail**

 _表达式_ 一个代表 **FormRegion** 对象的变量。


## 说明

对于相邻窗体区域只显示 **详细信息** 属性。对于独立窗体区域，它将被忽略。

 **详细信息** 属性是可选的相邻窗体区域。
