
# Account.DisplayName 属性 (Outlook)

返回一个 **字符串** ，表示电子邮件 **[帐户](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** 的显示名称。只读的。


## 语法

 _表达式_. **DisplayName**

 _表达式_ 一个代表 **Account** 对象的变量。


## 说明

 **显示名称** 是此 **帐户** 对象的默认属性。
