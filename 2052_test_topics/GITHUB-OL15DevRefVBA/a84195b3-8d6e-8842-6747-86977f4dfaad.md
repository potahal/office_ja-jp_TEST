
# PostItem.IsMarkedAsTask 属性 (Outlook)

返回一个 **布尔** 值，该值指示是否将 **[PostItem](de44065d-4e93-315a-279f-7b92f09c0465.md)** 标记为一项任务。只读的。


## 语法

 _表达式_. **IsMarkedAsTask**

 _表达式_ 一个返回 **PostItem** 对象的表达式。
