
# PeopleView.Standard 属性 (Outlook)
返回一个 **Boolean**值，指示[PeopleView](7b569709-5da8-a950-a0fb-9d64b520a21b.md)对象是内置的 Outlook 视图。只读的。

## 版本信息

添加的版本： Outlook 2013


## 语法

 _表达式_. **Standard**

 _表达式_ 一个表示 **PeopleView**对象的变量。


## 属性值

 **BOOL**
