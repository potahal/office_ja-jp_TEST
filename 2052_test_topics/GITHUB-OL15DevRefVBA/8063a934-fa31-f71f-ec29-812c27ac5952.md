
# PeopleView.ViewType 属性 (Outlook)
返回一个 [OlViewType](f2fec9d0-55c2-0991-0e1b-4dd653fdf09d.md) 常量，该常量指示视图的类型。只读。

## 版本信息

添加的版本： Outlook 2013


## 语法

 _表达式_. **ViewType**

 _表达式_ 一个表示 **PeopleView**对象的变量。


## 属性值

 **OLVIEWTYPE**
