
# TaskItem.PropertyAccessor 属性 (Outlook)

返回一个  **[PropertyAccessor](2fc91e13-703c-3ec9-9066-ffee7144306c.md)** 对象，该对象支持创建、获取、设置和删除父 **[TaskItem](5df8cfa5-5460-a5a1-a130-ba5bca1a0091.md)** 对象的属性。只读。


## 语法

 _表达式_. **PropertyAccessor**

 _表达式_ 一个表示 **TaskItem** 对象的变量。
