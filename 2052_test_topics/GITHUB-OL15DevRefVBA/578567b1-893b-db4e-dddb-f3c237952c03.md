
# MailItem.Body 属性 (Outlook)

返回或设置一个 **字符串** ，表示纯文本正文的 Outlook 项目。读/写。


## 语法

 _表达式_. **Body**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

 **[MailItem.BodyFormat](f635a0bc-20b7-206c-f558-a4ca2519670f.md)** 属性允许您以编程方式更改项目正文所使用的编辑器。
