
# MoveOrCopyRuleAction.ActionType 属性 (Outlook)

返回常量，  **olRuleActionMoveToFolder** ，从 **[OlRuleActionType](d6a39ac2-00e7-73e7-3890-ea658211eae9.md)** 枚举指示规则采取的操作的类型。只读的。


## 语法

 _表达式_. **ActionType**

 _表达式_ 一个代表 **MoveOrCopyRuleAction** 对象的变量。


## 说明

移动或复制规则操作只受邮件接收规则的支持。有关详细信息，请参阅[指定规则操作](http://msdn.microsoft.com/library/c5f83c81-0e01-38aa-5ec7-3932b4443e43%28Office.15%29.aspx)。
