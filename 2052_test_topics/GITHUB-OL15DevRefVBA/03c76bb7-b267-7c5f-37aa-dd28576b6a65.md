
# TaskRequestAcceptItem.BeforeAutoSave 事件 (Outlook)

Outlook 自动保存项目前发生。


## 语法

 _表达式_. **BeforeAutoSave**( ** _Cancel_** )

 _表达式_ 一个表示 **TaskRequestAcceptItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Cancel_|必需|**Boolean**|设置为 **True** 以取消操作;否则，设置为 **False** 以允许 **[TaskRequestAcceptItem](a2905f72-0a67-b07d-7f85-84fe4de17c25.md)** 保存。|
