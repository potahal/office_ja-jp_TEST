
# OlkTimeZoneControl.DropDown 方法 （Outlook）

展开时区控件的下拉部分。


## 语法

 _表达式_. **DropDown**

 _表达式_ 一个代表 **OlkTimeZoneControl** 对象的变量。


## 说明

模拟用户单击控件上的下拉按钮。此方法导致  **[DropButtonClick](4b0026d4-fb18-e72f-a369-6e14f4cfe8dd.md)** 事件被触发。
