
# MailItem.IsMarkedAsTask 属性 (Outlook)

返回一个 **布尔** 值，该值指示是否将 **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** 标记为一项任务。只读的。


## 语法

 _表达式_. **IsMarkedAsTask**

 _表达式_ 一个返回 **MailItem** 对象的表达式。
