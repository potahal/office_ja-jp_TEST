
# ViewFields.Remove 方法 （Outlook）

从集合中删除对象。


## 语法

 _表达式_. **Remove**( ** _Index_** )

 _表达式_ 一个代表 **ViewFields** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Variant**|或者是对象的索引号，或者是用来与集合中某个对象的  **[ViewXMLSchemaName](69490353-b470-6092-0b8e-b0f1c1549f7a.md)** 属性值相匹配的值。|
