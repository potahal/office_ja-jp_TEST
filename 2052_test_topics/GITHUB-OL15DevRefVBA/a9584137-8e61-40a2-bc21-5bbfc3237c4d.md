
# SharingItem.DeferredDeliveryTime 属性 (Outlook)

## 语法

 _表达式_. **DeferredDeliveryTime**

 _表达式_ 一个表示 **SharingItem** 对象的变量。


## 说明

此属性对应于 **PidTagDeferredDeliveryTime** 的 MAPI 属性。
