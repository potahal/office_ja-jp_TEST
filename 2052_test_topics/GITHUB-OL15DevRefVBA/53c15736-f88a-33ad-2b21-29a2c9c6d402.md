
# Store.DeleteUnifiedGroup 方法 (Outlook)

启用统一的组被删除。在 Office 2016 中引入。


## 语法

 _表达式_. **DeleteUnifiedGroup**

 _表达式_ 一个表示 **Store**对象的变量。


### 参数


||||
|:-----|:-----|:-----|
|**Name**|**值**|**说明**|
| _GroupSmtpAddress_|String|组的 Smtp 地址。|

### 返回值

用于删除组的 Smtp 地址。
