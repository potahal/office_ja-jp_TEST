
# OlkTextBox.DragBehavior 属性 (Outlook)

返回或设置一个  **[OlDragBehavior 枚举](1e8c29d4-7800-663f-fb5f-aebc2a6b89fe.md)** 常量，该常量指示系统是否为此控件启用拖放功能。可读/写。


## 语法

 _表达式_. **DragBehavior**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

默认值是 **olDragDropDisabled** 。
