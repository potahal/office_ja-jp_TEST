
# TimeZone.Application 属性 (Outlook)

返回一个代表对象的父 Outlook 应用程序的  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** 对象。只读。


## 语法

 _表达式_. **Application**

 _表达式_ 一个代表 **TimeZone** 对象的变量。
