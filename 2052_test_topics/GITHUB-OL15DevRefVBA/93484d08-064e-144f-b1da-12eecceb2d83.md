
# Store.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **Store** 对象的变量。


## 说明

 **[Store](1eb22fe9-8849-7476-5388-2515b48591b9.md)** 对象的父对象是 **[NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** 对象。
