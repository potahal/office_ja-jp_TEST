
# OlkLabel.Caption 属性 (Outlook)

返回或设置用于标识或描述该对象上显示的 **字符串** 。 读/写。


## 语法

 _表达式_. **Caption**

 _表达式_ 一个代表 **OlkLabel** 对象的变量。


## 说明

默认值为空字符串。窗体设计器将此属性设置为控件的默认名称。
