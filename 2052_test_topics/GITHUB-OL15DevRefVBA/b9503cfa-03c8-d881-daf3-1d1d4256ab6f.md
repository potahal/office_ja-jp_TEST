
# DistListItem.UnRead 属性 (Outlook)

返回或设置一个 **布尔** 值，如果的 Outlook 项目尚未打开 (只读)，则为 **True** 。读/写。


## 语法

 _表达式_. **UnRead**

 _表达式_ 一个表示 **DistListItem** 对象的变量。
