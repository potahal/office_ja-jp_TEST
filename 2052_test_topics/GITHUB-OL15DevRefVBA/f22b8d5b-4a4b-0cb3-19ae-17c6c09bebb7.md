
# SharingItem.Importance 属性 (Outlook)

返回或设置一个  **[OlImportance](71e04f9a-fab6-153f-b046-11f7ec50e8e4.md)** 常量，该常量指示 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 的相对重要性级别。可读/写。


## 语法

 _表达式_. **Importance**

 _表达式_ 一个表示 **SharingItem** 对象的变量。


## 说明

此属性对应于 **PidTagImportance** 的 MAPI 属性。
