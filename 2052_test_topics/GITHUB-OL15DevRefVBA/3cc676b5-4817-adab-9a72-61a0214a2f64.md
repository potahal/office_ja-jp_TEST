
# TaskItem.Mileage 属性 (Outlook)

返回或设置一个 **字符串** ，表示项目的里程。读/写。


## 语法

 _表达式_. **Mileage**

 _表达式_ 一个代表 **TaskItem** 对象的变量。


## 注解

这是自由格式的字符串字段，可用于存储与项目的经费报销相关的里程信息（例如，里程为 100 英里的约会、联系人或任务）。
