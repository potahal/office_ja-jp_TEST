
# MailItem.RecipientReassignmentProhibited 属性 (Outlook)

返回一个 **boolean 类型的值** ，指示如果收件人不能转发此邮件 **，则返回 True** 。读/写。


## 语法

 _表达式_. **RecipientReassignmentProhibited**

 _表达式_ 一个代表 **MailItem** 对象的变量。
