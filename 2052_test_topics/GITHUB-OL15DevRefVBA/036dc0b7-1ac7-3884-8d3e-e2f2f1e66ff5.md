
# MailItem.To 属性 (Outlook)

返回或设置 Outlook 项目 **对**收件人的显示名称的分号分隔的 **字符串** 列表。读/写。


## 语法

 _表达式_. **To**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性包含的显示名称。 **To**属性对应于 **PidTagDisplayTo** 的 MAPI 属性。若要修改此属性，应使用 **[收件人](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** 集合。
