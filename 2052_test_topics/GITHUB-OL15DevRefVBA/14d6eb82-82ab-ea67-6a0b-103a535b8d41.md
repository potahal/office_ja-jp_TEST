
# Application.Assistance 属性 (Outlook)

返回一个用于调用帮助的  **[IAssistance](http://msdn.microsoft.com/library/c8327d45-a6a2-dc4c-67f0-d02598eb60ba%28Office.15%29.aspx)** 对象。只读。


## 语法

 _表达式_. **Assistance**

 _表达式_ 一个表示 **Application** 对象的变量。
