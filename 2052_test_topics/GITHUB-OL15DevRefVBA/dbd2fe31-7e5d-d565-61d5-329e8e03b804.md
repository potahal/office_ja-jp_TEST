
# MailItem.DeferredDeliveryTime 属性 (Outlook)

返回或设置一个指示的日期和时间，则邮件消息将被传送的 **日期** 。 读/写。


## 语法

 _表达式_. **DeferredDeliveryTime**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性对应于 **PidTagDeferredDeliveryTime** 的 MAPI 属性。
