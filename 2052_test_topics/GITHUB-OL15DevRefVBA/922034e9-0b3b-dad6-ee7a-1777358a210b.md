
# OlkTimeZoneControl.MouseIcon 属性 (Outlook)

返回或设置 **StdPicture** 表示此控件的鼠标光标的自定义图片。读/写。


## 语法

 _表达式_. **MouseIcon**

 _表达式_ 一个代表 **OlkTimeZoneControl** 对象的变量。


## 说明

默认值为 **Null** ( **没有** 在 Visual Basic 中)。
