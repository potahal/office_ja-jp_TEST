
# TaskRequestDeclineItem.Conflicts 属性 (Outlook)

返回  **[Conflicts](c4e1c060-519a-a6d1-8fb2-c7dfa1e3e66f.md)** 对象，该对象代表对于任何 Outlook 项目对象存在冲突的项目。只读。


## 语法

 _表达式_. **Conflicts**

 _表达式_ 一个代表 **TaskRequestDeclineItem** 对象的变量。
