
# NavigationModules.Application 属性 (Outlook)

返回 **[应用程序](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** 对象表示父对象的 Outlook 应用程序。只读的。


## 语法

 _表达式_. **Application**

 _表达式_ 一个返回 **NavigationModules** 对象的表达式。
