
# DistListItem.ConversationIndex 属性 (Outlook)

返回一个 **字符串** ，指示会话线程中的项的相对位置。只读的。


## 语法

 _表达式_. **ConversationIndex**

 _表达式_ 一个代表 **DistListItem** 对象的变量。


## 注解

此属性对应于 **PidTagConversationIndex** 的 MAPI 属性。
