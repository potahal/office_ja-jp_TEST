
# PreviewPane.Class 属性 (Outlook)

返回指示[PreviewPane](fd4f497b-7085-6e0f-018b-17845f4dfe61.md)对象的类[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)枚举常量。只读的。在 Office 2016 中引入。


## 语法

 _表达式_. **Class**

 _表达式_ 一个表示 **PreviewPane**对象的变量。
