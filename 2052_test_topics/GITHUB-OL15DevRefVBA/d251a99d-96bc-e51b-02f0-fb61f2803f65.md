
# Conversation.Application 属性 (Outlook)

返回一个代表  **[Conversation](2705d38a-ebc0-e5a7-208b-ffe1f5446b1b.md)** 对象的父级 Microsoft Outlook 应用程序的 **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** 对象。只读。


## 语法

 _表达式_. **Application**

 _表达式_ 一个代表 **Conversation** 对象的变量。
