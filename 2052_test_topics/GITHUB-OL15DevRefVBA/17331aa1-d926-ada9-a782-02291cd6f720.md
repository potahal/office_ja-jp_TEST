
# ExchangeUser.Application 属性 (Outlook)

返回一个代表  **[ExchangeUser](6ec117d1-7fdb-aa36-b567-1242f8238df0.md)** 对象的父级应用程序 (Outlook) 的 **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** 对象。只读。


## 语法

 _表达式_. **Application**

 _表达式_ 一个代表 **ExchangeUser** 对象的变量。
