
# Items.Remove 方法 （Outlook）

从集合中删除对象。


## 语法

 _表达式_. **Remove** ** _Index_**

 _表达式_ 一个代表 **Items** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Long**|对象在集合中的索引值，索引从 1 开始编号。|
