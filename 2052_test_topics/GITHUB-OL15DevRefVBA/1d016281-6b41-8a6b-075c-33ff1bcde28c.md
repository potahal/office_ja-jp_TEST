
# OlkComboBox.ListIndex 属性 (Outlook)

返回或设置一个 **Long** ，该值指示组合框的列表中当前选定的元素的位置控制。读/写。


## 语法

 _表达式_. **ListIndex**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

 **ListIndex** 是值列表中的项的数组的索引从零开始。 默认值为-1，表示未选择任何项。
