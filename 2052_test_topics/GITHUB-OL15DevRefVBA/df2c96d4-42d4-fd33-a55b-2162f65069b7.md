
# OlkDateControl.Value 属性 (Outlook)

返回或设置一个 **variant 类型的值** ，该值表示控件的内容。读/写。


## 语法

 _表达式_. **Value**

 _表达式_ 一个代表 **OlkDateControl** 对象的变量。


## 说明

值相等，则 **[文本](fda479bc-c613-171f-4e81-efe9c548fd81.md)** ，只是为 **variant 类型的值** ，并且 **文本** 被强类型化为 **字符串** 。
