
# SharingItem.Save 方法 （Outlook）

将  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 保存到当前文件夹中，如果是新项目，则保存到该项目类型的 Outlook 默认文件夹中。


## 语法

 _表达式_. **Save**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
