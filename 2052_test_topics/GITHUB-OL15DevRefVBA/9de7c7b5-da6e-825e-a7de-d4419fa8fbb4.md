
# OlkComboBox.DropButtonClick 事件 (Outlook)

用户在组合框控件中单击下拉按钮以展开下拉列表时或者以编程方式调用  **[DropDown](0414a3d2-2c3b-1ce7-e897-f667b252a521.md)** 方法时发生。


## 语法

 _表达式_. **DropButtonClick**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。
