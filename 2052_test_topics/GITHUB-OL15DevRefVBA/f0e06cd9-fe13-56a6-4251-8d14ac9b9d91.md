
# OlkTimeZoneControl.BorderStyle 属性 (Outlook)

返回或设置一个 **[OlBorderStyle](fd0a6be8-8d4b-be9f-639c-cd1ea5de9c97.md)** 常量，用于定义控件边框的样式。 读/写。


## 语法

 _表达式_. **BorderStyle**

 _表达式_ 一个代表 **OlkTimeZoneControl** 对象的变量。


## 说明

默认值是 **olBorderStyleSingle** 。
