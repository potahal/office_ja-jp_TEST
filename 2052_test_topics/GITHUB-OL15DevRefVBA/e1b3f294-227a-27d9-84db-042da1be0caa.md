
# Conversation.Parent 属性 (Outlook)

返回父 **对象** 的指定的 **[对话](2705d38a-ebc0-e5a7-208b-ffe1f5446b1b.md)** 对象。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **Conversation** 对象的变量。


## 说明

 **会话** 对象的父级是对话的第一项。

如果对话中的所有项被都删除，在获取 **会话** 对象后， **父** 属性返回 **Null** ( **不** 在 Visual Basic 中)。
