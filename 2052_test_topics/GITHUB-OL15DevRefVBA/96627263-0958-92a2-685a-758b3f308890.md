
# NotesModule.Name 属性 (Outlook)

返回一个 **字符串** 值，表示本地化的显示名称 **[NotesModule](cdbdde08-0773-a78d-3809-a3811975bcc1.md)** 对象。只读的。


## 语法

 _表达式_. **Name**

 _表达式_ 一个代表 **NotesModule** 对象的变量。
