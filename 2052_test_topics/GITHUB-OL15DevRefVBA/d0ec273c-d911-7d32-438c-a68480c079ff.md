
# SharingItem.Size 属性 (Outlook)

返回 **Long** 值，表明 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 的大小 (以字节为单位)。只读的。


## 语法

 _表达式_. **Size**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
