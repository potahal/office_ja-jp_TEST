
# Row.Class 属性 (Outlook)

返回  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** 枚举中的一个常量，该常量指示 **[Row](06db3fa4-1649-48bf-3b86-ffdf99a47305.md)** 对象的类。只读。


## 语法

 _表达式_. **Class**

 _表达式_ 一个代表 **Row** 对象的变量。
