
# PostItem.AttachmentRemove 事件 (Outlook)

从父对象的实例中删除附件时发生。


## 语法

 _表达式_. **AttachmentRemove**( ** _Attachment_** )

 _表达式_ 一个代表 **PostItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Attachment_|必需|**Attachment**|已从项目中删除的  **[Attachment](3e11582b-ac90-0948-bc37-506570bb287b.md)** 。|
