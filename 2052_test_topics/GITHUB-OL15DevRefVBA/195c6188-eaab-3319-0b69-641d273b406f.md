
# MeetingItem.Submitted 属性 (Outlook)

返回一个 **布尔** 值， **如果该项目已提交** 。只读的。


## 语法

 _表达式_. **Submitted**

 _表达式_ 一个表示 **MeetingItem** 对象的变量。


## 注解

邮件总是在文件夹（通常为 **"发件箱"**）中创建和提交。
