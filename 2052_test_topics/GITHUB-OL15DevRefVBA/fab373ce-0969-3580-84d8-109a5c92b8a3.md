
# SharingItem.Class 属性 (Outlook)

返回一个指示对象的类的  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** 常量。只读。


## 语法

 _表达式_. **Class**

 _表达式_ 一个表示 **SharingItem** 对象的变量。


## 说明

此属性总是返回对 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 对象的 **olSharing** 。
