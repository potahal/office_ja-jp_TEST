
# NavigationPane.Modules 属性 (Outlook)

返回一个  **[NavigationModules](4b0743d3-0a21-488c-27b2-31ae07129a61.md)** 对象，该对象代表导航窗格包含的导航模块集合。只读。


## 语法

 _表达式_. **Modules**

 _表达式_ 一个代表 **NavigationPane** 对象的变量。
