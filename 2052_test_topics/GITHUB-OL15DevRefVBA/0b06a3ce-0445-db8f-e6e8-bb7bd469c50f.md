
# 选择对象 (Outlook)

包含当前在浏览器中选定的一组 Outlook 项目。


## 注解

使用 **[所选内容](11002043-9dab-a5ad-b36e-52ddb04c1859.md)** 从 **[资源管理器](026591e5-049f-503a-4166-34e6dbc225fb.md)** 返回的 **选择** 集合的属性 对象。


## 示例

下面的示例返回从对象 **资源管理器中** **选择** 对象。


```
Set mySelectedItems = myExplorer.Selection
```
