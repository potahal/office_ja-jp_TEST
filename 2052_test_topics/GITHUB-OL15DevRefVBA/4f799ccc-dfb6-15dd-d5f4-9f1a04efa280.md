
# BusinessCardView.Filter 属性 (Outlook)

返回或设置一个 **String** 值，该值代表视图的筛选器。读/写。


## 语法

 _表达式_. **Filter**

 _表达式_ 一个返回 **BusinessCardView** 对象的表达式。


## 说明

此属性的值是一个 DAV 搜索和定位 (DASL) 字符串，该字符串代表当前的视图筛选器。
