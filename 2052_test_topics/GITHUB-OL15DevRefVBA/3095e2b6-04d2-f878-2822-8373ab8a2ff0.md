
# StorageItem.LastModificationTime 属性 (Outlook)

返回指定的日期和时间的 Outlook 项目的上次修改 **日期时间** 值。 只读的。


## 语法

 _表达式_. **LastModificationTime**

 _表达式_ 一个代表 **StorageItem** 对象的变量。


## 说明

此属性对应于 **PidTagLastModificationTime** 的 MAPI 属性。
