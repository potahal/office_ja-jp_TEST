
# NameSpace.Folders 属性 (Outlook)

返回表示指定 **[命名空间](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** 中包含的所有文件夹的 **[文件夹](0c814c3c-74fc-414c-982d-a0097fcb35c2.md)** 集合。只读的。


## 语法

 _表达式_. **Folders**

 _表达式_ 一个代表 **NameSpace** 对象的变量。


## 注解

 **命名空间** 是给定命名空间的所有文件夹的根目录。
