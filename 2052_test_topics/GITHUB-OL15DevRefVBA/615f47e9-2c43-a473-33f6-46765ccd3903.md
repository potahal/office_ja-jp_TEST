
# TextRuleCondition.Text 属性 (Outlook)

返回或设置代表要由规则条件计算的文本的 **String** 元素数组。读/写。


## 语法

 _表达式_. **Text**

 _表达式_ 一个代表 **TextRuleCondition** 对象的变量。


## 说明

可以分配一个具有一个或多个字符串的数组进行计算。在一个数组中分配的多个文本字符串是使用逻辑或运算求出的。
