
# DistListItem.ReminderPlaySound 属性 (Outlook)

返回或设置一个 **布尔** 值， **如果此项时，提醒要发出声音** 。读/写。


## 语法

 _表达式_. **ReminderPlaySound**

 _表达式_ 一个代表 **DistListItem** 对象的变量。


## 注解

为了验证 **[ReminderSoundFile](3d0d0543-97f5-cecd-a838-c36b5c35b8aa.md)** 属性必须设置 **ReminderPlaySound** 属性。
