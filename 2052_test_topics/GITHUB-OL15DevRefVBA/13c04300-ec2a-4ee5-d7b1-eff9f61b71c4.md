
# MailItem.FlagRequest 属性 (Outlook)

返回或设置一个 **字符串** ，表示为邮件项目请求的操作。读/写。


## 语法

 _表达式_. **FlagRequest**

 _表达式_ 一个表示 **MailItem** 对象的变量。


## 说明

默认情况下，使用任何标志未标记的邮件项目，此属性的默认值为空字符串。您可以通过用户界面或以编程方式设置 **FlagRequest** 的值。当您将标记一个标志，该标志通过用户界面， **FlagRequest** 的默认值的邮件项目是"跟进"。
