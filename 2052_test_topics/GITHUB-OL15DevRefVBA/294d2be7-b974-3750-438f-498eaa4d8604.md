
# ViewFont.Color 属性 (Outlook)

返回或设置一个  **[OlColor](b2a457b5-d331-58c4-f9cc-2d56c8edd5e4.md)** 常量，该常量指示 **[ViewFont](cbd7c6ce-f49a-1627-0ad9-a019911fb47b.md)** 对象中字体的颜色。可读/写。


## 语法

 _表达式_. **Color**

 _表达式_ 一个代表 **ViewFont** 对象的变量。
