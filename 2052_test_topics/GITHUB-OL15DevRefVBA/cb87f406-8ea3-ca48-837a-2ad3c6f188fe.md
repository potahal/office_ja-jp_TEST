
# SharingItem.Attachments 属性 (Outlook)

返回表示指定 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 的所有附件的 **[附件](4cc96a5f-a822-8ad5-6f61-e996bee8ba22.md)** 对象。只读的。


## 语法

 _表达式_. **Attachments**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
