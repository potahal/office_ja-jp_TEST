
# JournalItem.ContactNames 属性 (Outlook)

返回或设置一个 **字符串** ，表示与 Outlook 项目相关联的联系人姓名。读/写。


## 语法

 _表达式_. **ContactNames**

 _表达式_ 一个代表 **JournalItem** 对象的变量。
