
# SharingItem.Read 事件 (Outlook)

当用户打开父对象的实例进行编辑时发生。


## 语法

 _表达式_. **Read**

 _表达式_ 一个返回 **SharingItem** 对象的表达式。


## 说明

 **读取** 事件与 **[Open](b795dbfa-2d47-0ee4-98ef-0c44bb6a0bec.md)** 事件不同，每当用户在支持单元格内编辑或者在 **[检查器](d7384756-669c-0549-1032-c3b864187994.md)** 中打开该项目时的视图中选择项时，即发生 **读取** 。
