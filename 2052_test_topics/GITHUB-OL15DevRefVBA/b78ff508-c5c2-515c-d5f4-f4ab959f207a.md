
# Categories.Count 属性 (Outlook)

返回一个 **Long** 值，指示指定集合中的 **[类别](143ef095-54b0-cbe2-e356-632029061ac2.md)** 对象的计数。只读的。


## 语法

 _表达式_. **Count**

 _表达式_ 一个代表 **Categories** 对象的变量。
