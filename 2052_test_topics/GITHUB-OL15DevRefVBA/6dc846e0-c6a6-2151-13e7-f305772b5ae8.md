
# DistListItem.IsMarkedAsTask 属性 (Outlook)

返回一个 **布尔** 值，该值指示是否将 **[DistListItem](027c3986-abff-d9b1-ecc2-26d60805e952.md)** 标记为一项任务。只读的。


## 语法

 _表达式_. **IsMarkedAsTask**

 _表达式_ 一个返回 **DistListItem** 对象的表达式。
