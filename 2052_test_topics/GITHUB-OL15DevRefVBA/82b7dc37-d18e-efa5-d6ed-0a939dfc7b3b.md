
# OlkTextBox.ForeColor 属性 (Outlook)

返回或设置一个 **Long** ，该值指示控件的前景色。读/写。


## 语法

 _表达式_. **ForeColor**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

此属性为 **OLE_COLOR** 是 **Long** 类型的值的类型。此属性的默认值是 0x80000008，"窗口型"。
