
# Results.DefaultItemType 属性 (Outlook)

返回一个  **[OlItemType](d42959b8-6c91-4d9e-98db-7226780f9995.md)** 常量，该常量指示文件夹中包含的默认 Outlook 项目类型。可读/写。


## 语法

 _表达式_. **DefaultItemType**

 _表达式_ 一个代表 **Results** 对象的变量。
