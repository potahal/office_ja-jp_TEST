
# AppointmentItem.ReminderMinutesBeforeStart 属性 (Outlook)

返回或设置 **Long** 值，表明提醒应在约会开始前的分钟数。读/写。


## 语法

 _表达式_. **ReminderMinutesBeforeStart**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
