
# Action.ShowOn 属性 (Outlook)

返回或设置一个  **[OlActionShowOn](6a6e4156-d593-b5c7-8ed1-e133d61332df.md)** 常量，该常量代表将显示操作的位置。可读/写。


## 语法

 _表达式_. **ShowOn**

 _表达式_ 一个代表 **Action** 对象的变量。
