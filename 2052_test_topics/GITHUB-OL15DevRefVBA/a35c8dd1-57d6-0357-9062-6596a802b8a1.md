
# MailItem.VotingResponse 属性 (Outlook)

返回或设置一个 **字符串** ，指定邮件的投票响应。读/写。


## 语法

 _表达式_. **VotingResponse**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性通常设置为一个由 **[VotingOptions](696b6dfe-1840-d43b-e6ec-e410a387665c.md)** 属性返回原始邮件的回复上的分隔值。
