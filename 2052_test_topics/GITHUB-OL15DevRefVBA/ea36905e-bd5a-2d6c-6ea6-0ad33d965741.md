
# OlkTextBox.Exit 事件 (Outlook)

在焦点从该控件移到同一窗体上的另一个控件后就发生。


## 语法

 _表达式_. **Exit**( ** _Cancel_** )

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Cancel_|必需|**Boolean**|**假** 的事件发生时。如果该事件过程将此参数设置为 **True** ，则不完成退出操作，并且焦点仍保留在该控件中。|
