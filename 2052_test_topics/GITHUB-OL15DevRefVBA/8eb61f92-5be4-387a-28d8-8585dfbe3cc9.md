
# OlkTextBox.Locked 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，指定控件处于锁定状态而无法进行更改。 读/写。


## 语法

 _表达式_. **Locked**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

默认值为  **False** 。

当控件处于锁定状态时，该控件不具有"禁用"外观，但是用户不能编辑控件中的内容。
