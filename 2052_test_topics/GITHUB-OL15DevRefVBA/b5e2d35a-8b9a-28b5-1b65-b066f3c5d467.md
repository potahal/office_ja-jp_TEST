
# TaskRequestDeclineItem.FormDescription 属性 (Outlook)

返回表示指定的 Outlook 项目的窗体说明的 **[答复](c88f92c4-4cac-84b3-6118-1150d42d7cff.md)** 对象。只读的。


## 语法

 _表达式_. **FormDescription**

 _表达式_ 一个代表 **TaskRequestDeclineItem** 对象的变量。
