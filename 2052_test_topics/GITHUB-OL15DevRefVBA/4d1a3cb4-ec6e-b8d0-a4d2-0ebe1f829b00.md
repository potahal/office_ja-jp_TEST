
# ToOrFromRuleCondition.Recipients 属性 (Outlook)

返回表示规则条件计算的收件人列表的 **[收件人](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** 集合。只读的。


## 语法

 _表达式_. **Recipients**

 _表达式_ 一个代表 **ToOrFromRuleCondition** 对象的变量。


## 说明

使用逻辑或运算求出多个收件人。
