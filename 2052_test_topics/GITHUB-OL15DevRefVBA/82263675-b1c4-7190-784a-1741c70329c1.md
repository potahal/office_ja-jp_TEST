
# Action.Prefix 属性 (Outlook)

返回或设置指定的前缀 (例如"Re") 的 **字符串** 执行操作时使用该项目的主题。读/写。


## 语法

 _表达式_. **Prefix**

 _表达式_ 一个代表 **Action** 对象的变量。


## 注解

Outlook 自动将冒号 (:) 添加到 **前缀** 属性的值设置项的主题时。
