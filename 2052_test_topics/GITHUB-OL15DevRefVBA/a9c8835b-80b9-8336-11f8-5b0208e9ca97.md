
# RemoteItem.HasAttachment 属性 (Outlook)

返回一个 **boolean 类型的值** ， **则返回 True** (默认值) 如果远程项目具有与它关联的附件。只读的。


## 语法

 _表达式_. **HasAttachment**

 _表达式_ 一个表示 **RemoteItem** 对象的变量。
