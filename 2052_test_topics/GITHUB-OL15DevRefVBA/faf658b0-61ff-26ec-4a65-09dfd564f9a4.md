
# ContactItem.OfficeLocation 属性 (Outlook)

返回或设置一个 **字符串** ，指定联系人的具体办公地点 (例如，楼 1 室或套间 123)。读/写。


## 语法

 _表达式_. **OfficeLocation**

 _表达式_ 一个表示 **ContactItem** 对象的变量。


## 注解

此属性对应于 **PidTagOfficeLocation** 的 MAPI 属性。
