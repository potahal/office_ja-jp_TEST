
# SimpleItems.Parent 属性 (Outlook)

返回父 **对象** 的 **[SimpleItems](b929ae28-fe5f-607e-37b5-ed6a304d4896.md)** 对象。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **SimpleItems** 对象的变量。


## 说明

 **SimpleItems** 对象的父对象是 **[对话](2705d38a-ebc0-e5a7-208b-ffe1f5446b1b.md)** 对象。
