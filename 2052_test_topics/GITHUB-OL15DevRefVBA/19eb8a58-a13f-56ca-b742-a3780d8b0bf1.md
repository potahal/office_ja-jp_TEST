
# TaskItem.SkipRecurrence 方法 （Outlook）

清除定期任务的当前实例并将重复周期设置为该任务的下一个实例。


## 语法

 _表达式_. **SkipRecurrence**

 _表达式_ 一个代表 **TaskItem** 对象的变量。


### 返回值

 **False** 指示任务的重复周期，在最后一项任务，因此没有任何设置重复执行的任务。 **为 true** ，则表示重复执行已成功设置该任务的下一个实例。
