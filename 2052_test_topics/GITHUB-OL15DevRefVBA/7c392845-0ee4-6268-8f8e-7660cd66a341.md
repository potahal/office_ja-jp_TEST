
# OlkTextBox.Font 属性 (Outlook)

返回 **StdFont** 表示用来呈现控件中的文本的字体。 只读的。


## 语法

 _表达式_. **Font**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

作为 Microsoft Windows 类型 **StdFont** 表示该字体。
