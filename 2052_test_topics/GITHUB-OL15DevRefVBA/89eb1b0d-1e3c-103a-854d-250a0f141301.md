
# AccountSelector.Application 属性 (Outlook)

返回一个  **[Application](797003e7-ecd1-eccb-eaaf-32d6ddde8348.md)** 对象，该对象代表对象的父级 Outlook 应用程序。只读。


## 语法

 _表达式_. **Application**

 _表达式_ 一个代表 **[AccountSelector](846f176e-5680-a214-7624-75f3a524c989.md)** 对象的变量。
