
# ContactItem.NetMeetingServer 属性 (Outlook)

返回或设置一个 **字符串** ，指定用于联机会议 Microsoft 网络会议服务器的名称。读/写。


## 语法

 _表达式_. **NetMeetingServer**

 _表达式_ 一个表示 **ContactItem** 对象的变量。
