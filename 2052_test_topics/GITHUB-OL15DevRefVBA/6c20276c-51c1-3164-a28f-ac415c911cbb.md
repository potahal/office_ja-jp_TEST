
# Action.ResponseStyle 属性 (Outlook)

返回或设置一个  **[OlActionResponseStyle](9981a41c-57e4-b981-e245-075811a10cdc.md)** 常量，该常量指示执行指定操作时所使用的响应样式。可读/写。


## 语法

 _表达式_. **ResponseStyle**

 _表达式_ 一个代表 **Action** 对象的变量。
