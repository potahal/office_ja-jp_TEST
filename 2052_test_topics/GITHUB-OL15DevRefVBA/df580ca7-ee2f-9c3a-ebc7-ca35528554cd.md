
# RuleConditions.FromAnyRSSFeed 属性 (Outlook)

返回与 **olConditionFromAnyRssFeed** **[RuleCondition.ConditionType](d695339d-5b64-dffb-652e-ac993fca4489.md)** 的 **[RuleCondition](e03f91c2-2c08-b036-104a-d6246f28bc2d.md)** 对象。只读的。


## 语法

 _表达式_. **FromAnyRSSFeed**

 _表达式_ 一个代表 **RuleConditions** 对象的变量。


## 说明

枚举的规则条件或例外条件的现有规则，或者在创建一个规则来指定的条件或例外条件时，该消息是从 RSS 订阅，请使用返回的 **RuleCondition** 对象。

始终 **[RuleConditions](e8e9a05a-b36b-add2-b294-8cdc5a97e119.md)** 集合的该属性返回的 **RuleCondition** 对象，而不论是否与此 **RuleConditions** 集合关联的规则已定义规则条件。如果该规则已定义并启用此类规则条件，则 **[RuleCondition.Enabled](43a6aa5f-18da-1b6c-a481-f30718725bd8.md)** 将 **真** 。
