
# SharingItem.Saved 属性 (Outlook)

返回一个 **布尔** 值，如果自上次保存以来没有被修改 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 为 **true** 。只读的。


## 语法

 _表达式_. **Saved**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
