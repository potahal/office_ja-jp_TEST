
# SharingItem.DownloadState 属性 (Outlook)

返回一个指示  **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 下载状态的常量，该常量属于 **[OlDownloadState](ff5e00db-ad06-ddf1-6e3a-536c0ae4ef34.md)** 枚举。只读。


## 语法

 _表达式_. **DownloadState**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
