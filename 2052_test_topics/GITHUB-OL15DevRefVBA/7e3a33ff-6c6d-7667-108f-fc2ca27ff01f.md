
# OlkListBox.ListCount 属性 (Outlook)

返回一个 **长** 的列表框控件的下拉列表中指定的元素数。只读的。


## 语法

 _表达式_. **ListCount**

 _表达式_ 一个代表 **OlkListBox** 对象的变量。


## 说明

默认值为零，表示列表中没有项目。
