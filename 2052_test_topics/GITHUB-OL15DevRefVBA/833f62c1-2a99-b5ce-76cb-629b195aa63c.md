
# MailItem.ClearTaskFlag 方法 （Outlook）

清除任务形式的  **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** 对象。


## 语法

 _表达式_. **ClearTaskFlag**

 _表达式_ 一个返回 **MailItem** 对象的表达式。


## 说明

调用此方法将 **[IsMarkedAsTask](6cc4530d-fa74-916b-654d-db995d9a989f.md)** 属性设置为 **False** 。
