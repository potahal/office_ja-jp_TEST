
# Column.Class 属性 (Outlook)

返回  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** 枚举中的一个常量，该常量指示 **[Column](b7eb6916-2d80-57c3-2077-47a2a4c73185.md)** 对象的类。只读。


## 语法

 _表达式_. **Class**

 _表达式_ 一个代表 **Column** 对象的变量。
