
# MailItem.ReceivedOnBehalfOfName 属性 (Outlook)

返回一个 **字符串** ，表示代理邮件收件人的用户显示名称。只读的。


## 语法

 _表达式_. **ReceivedOnBehalfOfName**

 _表达式_ 一个代表 **MailItem** 对象的变量。


## 注解

此属性对应于 **PidTagReceivedRepresentingName** 的 MAPI 属性。
