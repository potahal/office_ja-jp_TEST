
# NameSpace.ExchangeMailboxServerName 属性 (Outlook)

返回一个 **字符串** 值，表示承载主 Exchange 帐户邮箱的 Exchange 服务器的名称。只读的。


## 语法

 _表达式_. **ExchangeMailboxServerName**

 _表达式_ 一个返回 **[NameSpace](f0dcaa19-07f5-5d42-a3bf-2e42b7885644.md)** 对象的表达式。


## 说明

如果 Exchange 邮箱在命名空间中不存在，则此属性返回一个空字符串。
