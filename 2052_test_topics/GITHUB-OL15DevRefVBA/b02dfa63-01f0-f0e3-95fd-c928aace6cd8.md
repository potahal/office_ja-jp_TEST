
# NavigationModule.Name 属性 (Outlook)

返回一个 **字符串** 值，表示本地化的显示名称 **[NavigationModule](76565eaf-1e64-f5d4-b90f-ba156863802c.md)** 对象。只读的。


## 语法

 _表达式_. **Name**

 _表达式_ 一个返回 **NavigationModule** 对象的表达式。
