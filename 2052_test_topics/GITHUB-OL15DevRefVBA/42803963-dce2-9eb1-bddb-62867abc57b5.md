
# AppointmentItem.AllDayEvent 属性 (Outlook)

如果约会是全天事件 (不是指定的时间)，则返回 **True** 。读/写。


## 语法

 _表达式_. **AllDayEvent**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
