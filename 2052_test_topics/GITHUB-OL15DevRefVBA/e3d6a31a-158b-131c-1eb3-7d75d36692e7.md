
# Inspector.WindowState 属性 (Outlook)

返回或设置带有  **[OlWindowState](cc9860c4-9de6-4dd3-05e5-5b87e6b4bd88.md)** 枚举中的常量的属性，该常量指定浏览器或检查器窗口的窗口状态。可读/写。


## 语法

 _表达式_. **WindowState**

 _表达式_ 一个代表 **Inspector** 对象的变量。
