
# Exception.Deleted 属性 (Outlook)

返回一个 **布尔** 值，该值指示是否从定期模式中删除 **[AppointmentItem](204a409d-654e-27aa-643a-8344c631b82d.md)** 。只读的


## 语法

 _表达式_. **Deleted**

 _表达式_ 一个代表 **Exception** 对象的变量。


## 说明

如果此属性返回 **True** ， **AppointmentItem** 已从重复执行模式。
