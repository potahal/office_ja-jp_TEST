
# ViewFields.Count 属性 (Outlook)

返回一个 **Long** 值，指示指定集合中的 **[ViewField](997319f0-7ff3-a712-8484-2e442965e187.md)** 对象的计数。只读的。


## 语法

 _表达式_. **Count**

 _表达式_ 一个代表 **ViewFields** 对象的变量。
