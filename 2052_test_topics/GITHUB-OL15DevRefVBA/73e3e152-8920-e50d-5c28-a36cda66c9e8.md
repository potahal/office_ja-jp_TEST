
# UserDefinedProperty.Name 属性 (Outlook)

返回一个 **字符串** 值，表示显示名称对象。只读的。


## 语法

 _表达式_. **Name**

 _表达式_ 一个代表 **UserDefinedProperty** 对象的变量。
