
# MailItem.Reply 方法 （Outlook）

从原始邮件创建答复并预先将地址设为原始发件人。


## 语法

 _表达式_. **Reply**

 _表达式_ 一个代表 **MailItem** 对象的变量。


### 返回值

一个代表答复的  **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** 对象。
