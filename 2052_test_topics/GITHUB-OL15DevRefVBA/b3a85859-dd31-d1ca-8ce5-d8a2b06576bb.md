
# MeetingItem.ReplyAll 方法 （Outlook）

从原始邮件创建给所有原始收件人的答复。


## 语法

 _表达式_. **ReplyAll**

 _表达式_ 一个代表 **MeetingItem** 对象的变量。


### 返回值

一个代表答复的  **[MailItem](14197346-05d2-0250-fa4c-4a6b07daf25f.md)** 。
