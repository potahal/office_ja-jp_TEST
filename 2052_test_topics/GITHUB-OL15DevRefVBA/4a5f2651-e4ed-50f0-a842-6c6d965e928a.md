
# OlkTimeControl.Style 属性 (Outlook)

返回或设置一个  **[OlTimeStyle](82c4d063-29f2-d7c8-44ff-8b4aca912855.md)** 常量，该常量指定时间控件的样式。可读/写。


## 语法

 _表达式_. **Style**

 _表达式_ 一个代表 **OlkTimeControl** 对象的变量。


## 说明

默认值是 **olTimeStyleTimeOnly** 。
