
# OlkComboBox.Copy 方法 （Outlook）

将控件的内容复制到剪贴板上。


## 语法

 _表达式_. **Copy**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

数据以无格式文本格式复制，用于替换剪贴板的现有内容。

如果控件样式是 **olComboBoxStyleListBox** ，然后此方法复制当前所选内容的下拉列表中到剪贴板。
