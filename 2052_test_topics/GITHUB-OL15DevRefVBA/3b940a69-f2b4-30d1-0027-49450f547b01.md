
# TaskItem.TotalWork 属性 (Outlook)

返回或设置一个 **Long** ，该值指示任务的工作总量。读/写。


## 语法

 _表达式_. **TotalWork**

 _表达式_ 一个代表 **TaskItem** 对象的变量。


## 注解

 **TotalWork** 与任务项的 **详细信息**选项卡上 **工作总量**字段相对应。它是以分钟为单位存储中。标准任务窗体上的 **工作总量**字段绑定到您的 **TotalWork** 属性。默认情况下该字段假定一个工作日 8 小时和每周 40 小时。
