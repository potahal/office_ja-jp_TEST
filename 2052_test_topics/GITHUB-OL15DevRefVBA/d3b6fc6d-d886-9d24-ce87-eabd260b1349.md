
# JournalItem.DocPosted 属性 (Outlook)

返回一个 **布尔** 值，指示是否发布日记中记录的项目作为所记录会话的一部分。读/写。


## 语法

 _表达式_. **DocPosted**

 _表达式_ 一个代表 **JournalItem** 对象的变量。


## 说明

如果此属性返回 **True** ，然后日记中记录的项目已作为所记录会话的一部分过帐。
