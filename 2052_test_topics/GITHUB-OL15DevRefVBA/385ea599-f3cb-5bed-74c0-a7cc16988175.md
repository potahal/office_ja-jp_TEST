
# OlkComboBox.Text 属性 (Outlook)

返回或设置一个控件中显示的文本的 **字符串** 。读/写。


## 语法

 _表达式_. **Text**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

 **文本** 属性的默认值为空字符串。

 **文本** 属性提供值相同的 **[Value](742dd2a3-d3ef-46f9-4aca-5ebe8af17356.md)** 属性中，但值强类型化为 **字符串** 。
