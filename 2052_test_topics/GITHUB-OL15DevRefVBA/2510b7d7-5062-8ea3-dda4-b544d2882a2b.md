
# 客户对象 (Outlook)

 **帐户** 集合对象包含一组表示可用于当前配置文件的帐户的 **[帐户](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** 对象。


## 备注

 **帐户** 集合对象和 **帐户** 对象的用途是允许在给定配置文件中标识类型的 **帐户** ，并使用特定的 **帐户** 对象发送邮件的 **帐户** 对象的枚举。
