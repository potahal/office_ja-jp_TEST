
# TaskItem.ToDoTaskOrdinal 属性 (Outlook)

返回或设置一个 **Date** 值，表示任务的序号值 **[TaskItem](5df8cfa5-5460-a5a1-a130-ba5bca1a0091.md)** 。读/写。


## 语法

 _表达式_. **ToDoTaskOrdinal**

 _表达式_ 一个返回 **TaskItem** 对象的表达式。


## 说明

此属性用于指明该任务在父组（如 **"待办事项栏"**的 **"今天"**组或 **"明天"**组）中应如何排序。
