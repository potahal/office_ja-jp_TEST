
# OlkControl.EnableAutoLayout 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，可以将控件与窗体的其余部分的布局的自动调整。读/写。


## 语法

 _表达式_. **EnableAutoLayout**

 _表达式_ 一个代表 **OlkControl** 对象的变量。


## 说明

控件布局的自动调整包含将控件与其他控件垂直或水平对齐，以及按照窗体的高度和宽度调整控件的高度和宽度。
