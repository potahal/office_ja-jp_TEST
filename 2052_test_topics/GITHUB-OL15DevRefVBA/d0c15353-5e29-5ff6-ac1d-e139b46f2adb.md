
# NavigationModule.Visible 属性 (Outlook)

返回或设置一个 **布尔** 值，确定是否在导航窗格中显示的 **[NavigationModule](76565eaf-1e64-f5d4-b90f-ba156863802c.md)** 对象。读/写。


## 语法

 _表达式_. **Visible**

 _表达式_ 一个返回 **NavigationModule** 对象的表达式。
