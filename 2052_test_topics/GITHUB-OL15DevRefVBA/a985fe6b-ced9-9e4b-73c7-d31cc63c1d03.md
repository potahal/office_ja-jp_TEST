
# SharingItem.OutlookInternalVersion 属性 (Outlook)

返回一个 **Long** 表示 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 的 Outlook 应用程序内部版本号。只读的。


## 语法

 _表达式_. **OutlookInternalVersion**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
