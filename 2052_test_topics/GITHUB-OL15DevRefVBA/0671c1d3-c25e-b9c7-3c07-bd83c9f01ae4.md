
# Folder.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **Folder** 对象的变量。


## 说明

 **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** 对象的父对象是 **[Folders](0c814c3c-74fc-414c-982d-a0097fcb35c2.md)** 集合对象。
