
# Recipient.Address 属性 (Outlook)

返回一个表示 **[收件人](8cee4d79-ec55-52a4-710b-6456944ca86d.md)** 的电子邮件地址的 **字符串** 。只读的。


## 语法

 _表达式_. **Address**

 _表达式_ 一个代表 **Recipient** 对象的变量。
