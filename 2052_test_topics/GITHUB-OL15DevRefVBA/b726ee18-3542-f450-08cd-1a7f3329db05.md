
# OlkTextBox.Scrollbars 属性 (Outlook)

返回或设置一个  **[olScrollBars](22832728-5d27-ff6b-9ef2-ed6a8e1c9d5c.md)** 常量，该常量指定控件是具有垂直滚动条、水平滚动条还是同时具有这两者。可读/写。


## 语法

 _表达式_. **Scrollbars**

 _表达式_ 一个代表 **OlkTextBox** 对象的变量。


## 说明

默认值是 **olScrollBarsNone** 。
