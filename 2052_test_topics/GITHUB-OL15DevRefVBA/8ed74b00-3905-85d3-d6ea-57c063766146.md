
# Attachment.FileName 属性 (Outlook)

返回一个 **字符串** ，表示附件的文件名。 只读的。


## 语法

 _表达式_. **FileName**

 _表达式_ 一个代表 **Attachment** 对象的变量。


## 注解

 **[路径名](19ecbb05-11f0-2e35-eb3d-f60bce6b1357.md)** 属性结合使用此属性。
