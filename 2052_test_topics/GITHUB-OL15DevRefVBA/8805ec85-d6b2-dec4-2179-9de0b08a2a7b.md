
# ExplorerEvents_10.DisplayModeChange 方法 (Outlook)

当用户执行更改显示模式操作时发生。可能的模式包括普通、 纵向视图和纵向阅读窗格。在 Office 2016 中引入。


## 语法

 _表达式_. **DisplayModeChange**

 _表达式_ 一个表示 **ExplorerEvents_10**对象的变量。


### 参数


||||
|:-----|:-----|:-----|
| _DisplayMode_|必需|指明显示模式|

### 返回值

 **HRESULT**
