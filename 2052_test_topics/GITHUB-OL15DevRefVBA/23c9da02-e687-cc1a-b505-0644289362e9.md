
# AddressEntry.Type 属性 (Outlook)

返回或设置一个 **字符串** ，表示此地址，如 Internet 地址、 MacMail 地址或 Microsoft 邮件地址的条目的类型。读/写。


## 语法

 _表达式_. **Type**

 _表达式_ 一个代表 **AddressEntry** 对象的变量。
