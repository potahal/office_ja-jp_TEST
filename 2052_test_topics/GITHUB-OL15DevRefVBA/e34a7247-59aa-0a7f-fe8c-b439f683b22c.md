
# CalendarModule.Visible 属性 (Outlook)

返回或设置一个 **布尔** 值，确定是否在导航窗格中显示的 **[CalendarModule](9203024d-9cef-75e0-600f-f3899e24761a.md)** 对象。读/写。


## 语法

 _表达式_. **Visible**

 _表达式_ 一个代表 **CalendarModule** 对象的变量。
