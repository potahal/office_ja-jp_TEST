
# IconView.GoToDate 方法 （Outlook）

更改当前视图用来显示信息的日期。


## 语法

 _表达式_. **GoToDate**( ** _Date_** )

 _表达式_ 一个代表 **IconView** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Date_|必需|**日期**|应当将视图更改到的日期。|
