
# Folder.Description 属性 (Outlook)

返回或设置一个 **字符串** ，表示文件夹的描述。 读/写。


## 语法

 _表达式_. **Description**

 _表达式_ 一个代表 **Folder** 对象的变量。


## 说明

此属性对应于 **PidTagComment** 的 MAPI 属性。
