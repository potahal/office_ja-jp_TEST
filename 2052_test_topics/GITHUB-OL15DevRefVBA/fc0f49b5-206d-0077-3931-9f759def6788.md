
# OlkPageControl.Page 属性 (Outlook)

返回或设置一个  **[OlPageType](e4392bf7-5e61-c67b-e564-191e9c62e407.md)** 常量，该常量指定将显示在此控件中的页面的类型。可读/写。


## 语法

 _表达式_. **Page**

 _表达式_ 一个代表 **OlkPageControl** 对象的变量。


## 说明

默认值是 **olPageTypePlanner** 。
