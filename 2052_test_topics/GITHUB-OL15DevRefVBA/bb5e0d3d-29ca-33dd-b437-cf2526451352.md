
# Action.ReplyStyle 属性 (Outlook)

返回或设置一个  **[OlActionReplyStyle](730f9712-a2bb-f698-d210-9dc94da373e8.md)** 常量，该常量指示指定操作的文本格式答复样式。可读/写。


## 语法

 _表达式_. **ReplyStyle**

 _表达式_ 一个代表 **Action** 对象的变量。
