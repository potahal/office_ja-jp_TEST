
# OlkCommandButton.PictureAlignment 属性 (Outlook)

返回或设置一个  **[OlPictureAlignment](69b9407f-a86a-393d-11ac-5272be88de6d.md)** 常量，该常量指定按钮上的图像与文本的对齐方式。可读/写。


## 语法

 _表达式_. **PictureAlignment**

 _表达式_ 一个代表 **OlkCommandButton** 对象的变量。
