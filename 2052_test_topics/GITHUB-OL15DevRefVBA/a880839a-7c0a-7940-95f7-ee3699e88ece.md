
# AppointmentItem.ClearRecurrencePattern 方法 （Outlook）

删除约会（或任务）的定期设置，恢复为一次性状态。


## 语法

 _表达式_. **ClearRecurrencePattern**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
