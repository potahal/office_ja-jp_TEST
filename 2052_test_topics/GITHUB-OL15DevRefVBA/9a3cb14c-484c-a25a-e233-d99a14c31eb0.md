
# OlkDateControl.ShowNoneButton 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，指定是否在日期控件中显示 **无**按钮。 读/写。


## 语法

 _表达式_. **ShowNoneButton**

 _表达式_ 一个代表 **OlkDateControl** 对象的变量。


## 说明

默认值为 **True** 。
