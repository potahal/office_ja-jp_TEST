
# Explorer.PreviewPane 属性 (Outlook)

[PreviewPane](fd4f497b-7085-6e0f-018b-17845f4dfe61.md)对象显示的方式显示仅预览窗格视图"窗格中的单个模式"中的内容。在 Office 2016 中引入。


## 语法

 _表达式_. **PreviewPane**

 _表达式_ 一个表示 **Explorer**对象的变量。
