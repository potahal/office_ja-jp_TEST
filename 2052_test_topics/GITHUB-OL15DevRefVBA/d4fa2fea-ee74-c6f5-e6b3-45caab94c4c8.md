
# AccountRuleCondition.Account 属性 (Outlook)

返回或设置一个代表用于评估规则条件的帐户的  **[Account](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** 对象。可读/写。


## 语法

 _表达式_. **Account**

 _表达式_ 一个代表 **AccountRuleCondition** 对象的变量。


## 说明

如果帐户不代表一个初始化的帐户，则返回错误。
