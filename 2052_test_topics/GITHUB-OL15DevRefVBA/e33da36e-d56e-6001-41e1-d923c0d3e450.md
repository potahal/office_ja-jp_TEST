
# CardView.AllowInCellEditing 属性 (Outlook)

返回或设置一个 **布尔** 值，确定是否在 **[卡片视图-](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** 对象允许单元格内编辑。读/写。


## 语法

 _表达式_. **AllowInCellEditing**

 _表达式_ 一个代表 **CardView** 对象的变量。
