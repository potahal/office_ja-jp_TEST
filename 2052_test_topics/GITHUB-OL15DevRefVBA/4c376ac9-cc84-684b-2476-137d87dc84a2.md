
# OlkOptionButton.Caption 属性 (Outlook)

返回或设置用于标识或描述该对象上显示的 **字符串** 。 读/写。


## 语法

 _表达式_. **Caption**

 _表达式_ 一个代表 **OlkOptionButton** 对象的变量。


## 说明

默认值基于控件的名称。标题相对于复选框的位置由  **[Alignment](0d6d5ac4-d084-ea7d-80e6-854315f94b90.md)** 属性决定。
