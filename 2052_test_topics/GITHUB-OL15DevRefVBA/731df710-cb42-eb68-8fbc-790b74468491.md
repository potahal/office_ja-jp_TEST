
# NameSpace.CurrentProfileName 属性 (Outlook)

返回一个 **字符串** ，表示当前配置文件的名称。只读的。


## 语法

 _表达式_. **CurrentProfileName**

 _表达式_ 一个表示 **NameSpace** 对象的变量。


## 说明

在创建配置文件时指定配置文件名称。

 **CurrentProfileName** 到 **PidTagProfileName** 的 MAPI 属性映射。
