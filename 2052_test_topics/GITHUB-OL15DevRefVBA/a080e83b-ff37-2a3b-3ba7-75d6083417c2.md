
# PeopleView.GoToDate 方法 (Outlook)
更改当前视图用来显示信息的日期。

## 版本信息

添加的版本： Outlook 2013


## 语法

 _表达式_. **GoToDate** _(Date)_

 _表达式_ 一个表示 **PeopleView**对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
|||||
| _Date_|必需|**Date**|应当将视图更改到的日期。|
