
# ContactItem.BeforeAutoSave 事件 (Outlook)

Outlook 自动保存项目前发生。


## 语法

 _表达式_. **BeforeAutoSave**( ** _Cancel_**, )

 _表达式_ 一个表示 **ContactItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Cancel_|必需|**Boolean**|设置为 **True** 以取消操作;否则，设置为 **False** 以允许 **[联系人](8e32093c-a678-f1fd-3f35-c2d8994d166f.md)** 保存。|
