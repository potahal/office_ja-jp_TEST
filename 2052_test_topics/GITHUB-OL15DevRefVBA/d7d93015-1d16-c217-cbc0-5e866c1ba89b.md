
# ReportItem.Attachments 属性 (Outlook)

返回一个表示指定项的所有附件 **[附件](4cc96a5f-a822-8ad5-6f61-e996bee8ba22.md)** 对象。只读的。


## 语法

 _表达式_. **Attachments**

 _表达式_ 一个代表 **ReportItem** 对象的变量。
