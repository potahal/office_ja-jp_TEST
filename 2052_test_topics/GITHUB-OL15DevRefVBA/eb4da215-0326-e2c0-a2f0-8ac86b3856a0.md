
# ContactsModule.NavigationGroups 属性 (Outlook)

返回一个  **[NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** 对象，该对象包含一组与 **[ContactsModule](fb183bd5-c72f-b38f-97e3-209a2a463d24.md)** 对象关联的导航组。只读。


## 语法

 _表达式_. **NavigationGroups**

 _表达式_ 一个代表 **ContactsModule** 对象的变量。
