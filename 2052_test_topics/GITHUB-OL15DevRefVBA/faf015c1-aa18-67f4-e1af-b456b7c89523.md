
# SharingItem.CustomPropertyChange 事件 (Outlook)

当项目（父对象的一个实例）的自定义属性更改时发生。


## 语法

 _表达式_. **CustomPropertyChange**( ** _Name_** )

 _表达式_ 一个返回 **SharingItem** 对象的表达式。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Name_|必需|**String**|被更改的自定义属性的名称。|

## 说明

属性名传递给过程，以便于用户确定哪个自定义属性被更改。
