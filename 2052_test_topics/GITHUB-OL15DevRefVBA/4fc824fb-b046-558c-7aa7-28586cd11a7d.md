
# AppointmentItem.Recipients 属性 (Outlook)

返回表示 Outlook 项目的所有收件人的 **[收件人](774f56b7-4de8-9584-60cd-4fbf361f4c85.md)** 集合。只读的。


## 语法

 _表达式_. **Recipients**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。


## 注解

可以通过代表收件人的显示名称、别名或完整的 SMTP 电子邮件地址的字符串指定收件人。
