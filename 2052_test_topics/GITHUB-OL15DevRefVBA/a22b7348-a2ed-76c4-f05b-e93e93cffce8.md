
# NotesModule.NavigationModuleType 属性 (Outlook)

返回一个  **[olNavigationModuleType](2140a094-6bee-aba1-03cd-71fa2c55842e.md)** 常量，该常量代表对象的导航模块类型。只读。


## 语法

 _表达式_. **NavigationModuleType**

 _表达式_ 一个返回 **NotesModule** 对象的表达式。


## 说明

此属性返回 **olModuleNotes** **[NotesModule](cdbdde08-0773-a78d-3809-a3811975bcc1.md)** 对象。
