
# Attachment.Size 属性 (Outlook)

返回 **Long** 类型，表示附件的大小 (以字节为单位)。只读的。


## 语法

 _表达式_. **Size**

 _表达式_ 一个代表 **Attachment** 对象的变量。


## 注解

并非总是可以提供附件的此信息。例如，在 S/MIME 邮件中，实际附件大小在提取附件之前是未知的。在无法确定附件大小的情况下，此属性返回 0。
