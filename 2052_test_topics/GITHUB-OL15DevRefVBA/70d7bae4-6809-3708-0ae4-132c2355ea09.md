
# SharingItem.DeleteAfterSubmit 属性 (Outlook)

返回或设置一个 **布尔** 值，如果一项的副本并不保存发送，和 **假** 如果副本保存为 **真** 。读/写。


## 语法

 _表达式_. **DeleteAfterSubmit**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
