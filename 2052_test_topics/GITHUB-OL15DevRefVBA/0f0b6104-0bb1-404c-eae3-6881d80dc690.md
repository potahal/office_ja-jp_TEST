
# RuleConditions.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **RuleConditions** 对象的变量。


## 说明

 **[RuleConditions](e8e9a05a-b36b-add2-b294-8cdc5a97e119.md)** 对象的父对象是 **[Rule](ea2ddbcc-fd65-a636-c6da-79950033f385.md)** 对象。
