
# Recipient.TrackingStatus 属性 (Outlook)

返回或设置一个  **[OlTrackingStatus](a2253862-b1a1-6d99-81ad-1984ba615919.md)** 常量，该常量指示收件人的跟踪状态。可读/写。


## 语法

 _表达式_. **TrackingStatus**

 _表达式_ 一个代表 **Recipient** 对象的变量。
