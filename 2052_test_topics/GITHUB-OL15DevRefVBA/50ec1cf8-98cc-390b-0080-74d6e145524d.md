
# MeetingItem.AttachmentRead 事件 (Outlook)

当打开父对象实例中的附件进行阅读时发生。


## 语法

 _表达式_. **AttachmentRead**( ** _Attachment_** )

 _表达式_ 一个代表 **MeetingItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Attachment_|必需|**[Attachment](3e11582b-ac90-0948-bc37-506570bb287b.md)**|打开该 **附件** 。|
