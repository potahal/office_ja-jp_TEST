
# Items.GetLast 方法 （Outlook）

返回集合中的最后一个对象。


## 语法

 _表达式_. **GetLast**

 _表达式_ 一个代表 **Items** 对象的变量。


### 返回值

一个 **Object**值，该值代表集合中包含的最后一个对象。


## 注解

返回 **Nothing** 如果没有最后一个对象存在，例如，如果集合为空。为了确保 **[GetFirst](142a6174-118e-6256-0511-8ae9e142e555.md)** 、 **进入** 、 **[GetNext](01c49c21-d9f9-37c4-8c64-ff8e2b1f9462.md)** 和 **[方法在大型集合](5dde47f8-2bd8-fdbe-d6e7-b1381e8a97a6.md)** 的正确操作，在该集合上调用 **GetNext** 之前调用 **GetFirst** 并调用 **变量** 之前调用 **进入** 。要确保您始终在同一集合下生成的调用，请创建引用在进入循环前该集合的显式变量。
