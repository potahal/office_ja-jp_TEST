
# OlAccountType 枚举 （Outlook）

指定  **[Account](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** 的类型。



|**名称**|**值**|**说明**|
|:-----|:-----|:-----|
|**olEas**|4|在移动设备使用 Exchange ActiveSync (EAS) 的帐户。|
|**olExchange**|0|Exchange 帐户。|
|**olHttp**|3|HTTP 帐户。|
|**olImap**|1|IMAP 帐户。|
|**olOtherAccount**|5|其他帐户或未知帐户。|
|**olPop3**|2|POP3 帐户。|
