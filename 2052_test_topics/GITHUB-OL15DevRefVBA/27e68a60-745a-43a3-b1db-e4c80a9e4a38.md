
# JournalItem.Conflicts 属性 (Outlook)

返回表示正在发生冲突的任何 Outlook 项目对象的项的 **[冲突](c4e1c060-519a-a6d1-8fb2-c7dfa1e3e66f.md)** 对象。 只读的。


## 语法

 _表达式_. **Conflicts**

 _表达式_ 一个代表 **JournalItem** 对象的变量。
