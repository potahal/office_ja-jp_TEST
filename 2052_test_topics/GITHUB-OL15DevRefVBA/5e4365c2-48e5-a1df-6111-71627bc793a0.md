
# JournalModule.Name 属性 (Outlook)

返回一个 **字符串** 值，表示本地化的显示名称 **[JournalModule](5a696d10-8a10-c01d-cf65-f8a65718f120.md)** 对象。只读的。


## 语法

 _表达式_. **Name**

 _表达式_ 一个代表 **JournalModule** 对象的变量。
