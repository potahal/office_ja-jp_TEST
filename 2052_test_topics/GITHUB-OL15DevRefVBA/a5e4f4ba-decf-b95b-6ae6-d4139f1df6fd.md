
# TaskRequestUpdateItem.Size 属性 (Outlook)

返回一个 **Long** ，该值指示在 Outlook 项的大小 (以字节为单位)。只读的。


## 语法

 _表达式_. **Size**

 _表达式_ 一个表示 **TaskRequestUpdateItem** 对象的变量。
