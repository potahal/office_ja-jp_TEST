
# OlRemoteStatus 枚举 （Outlook）

指示项目由远程用户接收后的状态。



|**名称**|**值**|**说明**|
|:-----|:-----|:-----|
|**olMarkedForCopy**|3|**Item**被标记为复制。|
|**olMarkedForDelete**|4|**Item**已标记为删除。|
|**olMarkedForDownload**|2|**Item**标记为要下载。|
|**olRemoteStatusNone**|0|未设置远程状态。|
|**olUnMarked**|1|未标记为 **Item** 。|

## 注解

请参阅  **MarkForDownLoad** 属性。
