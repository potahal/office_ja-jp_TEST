
# TaskItem.PropertyChange 事件 (Outlook)

当父对象的实例的显式内置属性（例如  **[Subject](9f487fbc-48ab-e01d-c1a4-5b67fcb1a118.md)** ）更改时发生。


## 语法

 _表达式_. **PropertyChange**( ** _Name_** )

 _表达式_ 一个代表 **TaskItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Name_|必需|**String**|已更改的属性的名称。|

## 注解

属性名被传递给事件，这样用户就可以确定哪个属性被更改。
