
# Action.CopyLike 属性 (Outlook)

返回或设置一个  **[OlActionCopyLike](f566bbb1-4906-c1c2-1f8e-9f1564e6c072.md)** 常量，该常量指示要在操作中使用的属性继承样式。可读/写。


## 语法

 _表达式_. **CopyLike**

 _表达式_ 一个代表 **Action** 对象的变量。


## 注解

执行操作时，使用继承样式控制如何将属性复制到由操作创建的新项目中。
