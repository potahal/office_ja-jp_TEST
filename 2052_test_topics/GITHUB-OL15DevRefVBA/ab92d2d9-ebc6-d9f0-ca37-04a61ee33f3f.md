
# PreviewPane.Parent 属性 (Outlook)

返回[PreviewPane](fd4f497b-7085-6e0f-018b-17845f4dfe61.md)对象的父属性。只读。在 Office 2016 中引入。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个表示 **PreviewPane**对象的变量。
