
# Application.DefaultProfileName 属性 (Outlook)

返回一个表示的默认配置文件名称的 **字符串** 。只读的。


## 语法

 _表达式_. **DefaultProfileName**

 _表达式_ 一个表示 **Application** 对象的变量。


## 说明

如果没有为 Outlook 配置配置文件或者 Outlook 没有默认的配置文件，将返回空字符串。
