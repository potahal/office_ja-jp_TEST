
# RecurrencePattern.DayOfMonth 属性 (Outlook)

返回或设置 **Long** 类型，表示发生定期约会或任务的月份的日期。读/写。


## 语法

 _表达式_. **DayOfMonth**

 _表达式_ 一个代表 **RecurrencePattern** 对象的变量。
