
# TaskItem.ActualWork 属性 (Outlook)

返回或设置 **Long** 值，表明实际花费的时间在任务上。读/写。


## 语法

 _表达式_. **ActualWork**

 _表达式_ 一个代表 **TaskItem** 对象的变量。


## 注解

 **ActualWork** 是以分钟为单位存储中。标准任务窗体上的 **ActualWork** 字段绑定到您的 **ActualWork** 属性。默认情况下该字段假定一个工作日 8 小时和每周 40 小时。
