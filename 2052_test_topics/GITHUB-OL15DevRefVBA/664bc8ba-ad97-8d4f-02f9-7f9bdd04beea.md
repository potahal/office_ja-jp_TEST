
# Application.Quit 方法 （Outlook）

关闭当前所有打开的窗口。


## 语法

 _表达式_. **Quit**

 _表达式_ 一个返回 **Application** 对象的表达式。


## 注解

彻底关闭相关的 Outlook 会话；使用户从邮件系统中注销，并丢弃对项目所做的所有未保存的更改。
