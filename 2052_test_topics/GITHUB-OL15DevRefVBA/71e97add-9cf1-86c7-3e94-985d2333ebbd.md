
# ExchangeUser.YomiDisplayName 属性 (Outlook)

返回一个 **字符串** ，表示 **[ExchangeUser](6ec117d1-7fdb-aa36-b567-1242f8238df0.md)** Exchange 显示名称的日文拼音呈现 (yomigana)。读/写。


## 语法

 _表达式_. **YomiDisplayName**

 _表达式_ 一个代表 **ExchangeUser** 对象的变量。
