
# NavigationFolder.Folder 属性 (Outlook)

返回一个  **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** 对象，该对象代表与导航文件夹关联的共享或链接文件夹。只读。


## 语法

 _表达式_. **Folder**

 _表达式_ 一个代表 **NavigationFolder** 对象的变量。
