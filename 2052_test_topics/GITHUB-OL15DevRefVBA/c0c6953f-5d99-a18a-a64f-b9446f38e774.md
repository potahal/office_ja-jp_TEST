
# AddressList.ID 属性 (Outlook)

返回一个表示此对象的唯一标识符的 **字符串** 。只读的。


## 语法

 _表达式_. **ID**

 _表达式_ 一个代表 **AddressList** 对象的变量。


## 注解

传输提供程序在创建单个成员对象时分配永久的、 唯一的字符串 **ID** 属性。这些标识符不变化从一个会话到另一个。
