
# DocumentItem.Move 方法 （Outlook）

将 Microsoft Outlook项目移至新文件夹中。


## 语法

 _表达式_. **Move**( ** _DestFldr_** )

 _表达式_ 一个代表 **DocumentItem** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _DestFldr_|必需|**[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)**|目标文件夹。|

### 返回值

表示已被移动到指定文件夹的项的 **对象** 。
