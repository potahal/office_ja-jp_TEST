
# StorageItem.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **StorageItem** 对象的变量。


## 说明

StorageItem 对象的父对象是  **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** 对象。
