
# Rules.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **Rules** 对象的变量。


## 说明

 **Rules**对象的父项是 **[存储](1eb22fe9-8849-7476-5388-2515b48591b9.md)** 对象。
