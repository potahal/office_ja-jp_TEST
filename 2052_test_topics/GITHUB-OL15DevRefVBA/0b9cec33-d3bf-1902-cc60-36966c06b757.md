
# AppointmentItem.Body 属性 (Outlook)

返回或设置一个 **字符串** ，表示纯文本正文的 Outlook 项目。读/写。


## 语法

 _表达式_. **Body**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
