
# ContactItem.Email2DisplayName 属性 (Outlook)

返回一个 **字符串** ，表示联系人显示名称的第二个电子邮件条目。读/写。


## 语法

 _表达式_. **Email2DisplayName**

 _表达式_ 一个代表 **ContactItem** 对象的变量。


## 注解

默认情况下，此属性设置为 **[FullName](3036dc57-31fb-45ad-f51e-49336206581d.md)** 属性的值。
