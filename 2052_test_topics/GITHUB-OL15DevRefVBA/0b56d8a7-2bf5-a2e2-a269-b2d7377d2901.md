
# SimpleItems.Item 方法 （Outlook）

返回  **[SimpleItems](b929ae28-fe5f-607e-37b5-ed6a304d4896.md)** 集合中的一个项目。


## 语法

 _表达式_. **Item**( ** _Index_** )

 _表达式_ 一个代表 **SimpleItems** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Index_|必需|**Variant**|**SimpleItems** 集合中的对象的从零开始的索引号。|

### 返回值

一个 **对象** ，表示 **SimpleItems** 集合中的某个 Outlook 项目。


## 说明

如果此方法不能作为 _索引_ 参数指定的集合中返回一个对象，该方法返回 **Null** ( **不** 在 Visual Basic 中)。
