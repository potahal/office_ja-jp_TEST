
# ViewFont.Bold 属性 (Outlook)

返回或设置一个 **布尔** 值，该值指示字体是否为粗体 **[ViewFont](cbd7c6ce-f49a-1627-0ad9-a019911fb47b.md)** 对象中。读/写。


## 语法

 _表达式_. **Bold**

 _表达式_ 一个代表 **ViewFont** 对象的变量。
