
# NameSpace.Stores 属性 (Outlook)

返回一个  **[Stores](8915a8e4-9c22-21d5-c492-051d393ce5f7.md)** 集合对象，该集合对象代表当前配置文件中的所有 **[Store](1eb22fe9-8849-7476-5388-2515b48591b9.md)** 对象。只读。


## 语法

 _表达式_. **Stores**

 _表达式_ 一个表示 **NameSpace** 对象的变量。
