
# OlkComboBox.TopIndex 属性 (Outlook)

返回或设置一个 **Long** ，代表在组合框列表的显示部分的顶端项目的索引。读/写。


## 语法

 _表达式_. **TopIndex**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

当列表滚动时，列表顶端的项目会更改，此属性的值也会随之更改以反映当前显示在列表顶端的项目。

索引值是从零开始的。默认值为 -1，表示不应用特殊的排序。
