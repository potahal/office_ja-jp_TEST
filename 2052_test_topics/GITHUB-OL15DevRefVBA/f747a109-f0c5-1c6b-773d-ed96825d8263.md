
# SharingItem.ExpiryTime 属性 (Outlook)

返回或设置一个指示的日期和时间的 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 变为无效，并可被删除的 **日期** 。读/写。


## 语法

 _表达式_. **ExpiryTime**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
