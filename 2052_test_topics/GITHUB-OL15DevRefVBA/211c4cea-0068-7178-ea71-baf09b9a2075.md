
# Inspector.Deactivate 事件 (Outlook)

由于用户操作或程序代码而使检查器不再是活动窗口时发生。


## 语法

 _表达式_. **Deactivate**

 _表达式_ 一个代表 **Inspector** 对象的变量。


## 注解

此事件在 Microsoft Visual Basic Scripting Edition (VBScript) 中不可用。
