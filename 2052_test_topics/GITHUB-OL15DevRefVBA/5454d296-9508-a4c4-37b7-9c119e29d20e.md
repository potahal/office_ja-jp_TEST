
# OlkTimeControl.AfterUpdate 事件 (Outlook)

当控件中的数据已通过用户界面更改后发生。


## 语法

 _表达式_. **AfterUpdate**

 _表达式_ 一个代表 **OlkTimeControl** 对象的变量。


## 说明

 **更新** 和 **更新后** 会在该控件中的数据保存到项目的任何时候。 典型的涉及 **更新后** ，此控件的事件序列如下所示:


1. 用户将焦点放在控件上
    
2.  **BeforeUpdate**
    
3. 控件数据被更新
    
4.  ** AfterUpdate**
    
5.  **退出**: 用户移动焦点离开控件
