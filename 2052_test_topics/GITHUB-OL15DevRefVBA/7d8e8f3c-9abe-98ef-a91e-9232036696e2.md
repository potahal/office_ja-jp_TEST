
# OlkListBox.Enabled 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，该值指示是否允许控件运行。 读/写。


## 语法

 _表达式_. **Enabled**

 _表达式_ 一个代表 **OlkListBox** 对象的变量。


## 说明

默认值为 **True** 。
