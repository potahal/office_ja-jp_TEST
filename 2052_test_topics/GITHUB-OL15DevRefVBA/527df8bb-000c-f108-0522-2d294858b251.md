
# OlMousePointer 枚举 （Outlook）

指定鼠标放在控件上但尚未执行拖放操作时显示的鼠标指针。



|**名称**|**值**|**说明**|
|:-----|:-----|:-----|
|**olMousePointerAppStarting**|13|指针是箭头和沙漏形状，代表应用程序正在启动。|
|**olMousePointerArrow**|1|指针是标准箭头。|
|**olMousePointerCross**|2|指针是十字。|
|**olMousePointerCustom**|99|指针由自定义的指针位图代表。|
|**olMousePointerDefault**|0|默认指针。|
|**olMousePointerHelp**|14|指针是"这是什么"帮助指针。|
|**olMousePointerHourGlass**|11|指针是沙漏形状，代表应用程序忙。|
|**olMousePointerIBeam**|3|指针是 I-beam 文本输入指针。|
|**olMousePointerNoDrop**|12|指针是不允许放下指针。|
|**olMousePointerSizeAll**|15|在所有方向调整指针大小。|
|**olMousePointerSizeNESW**|6|由东北向西南方向调整指针大小。|
|**olMousePointerSizeNS**|7|由北向南方向调整指针大小。|
|**olMousePointerSizeNWSE**|8|由西北向东南方向调整指针大小。|
|**olMousePointerSizeWE**|9|由西向东方向调整指针大小。|
|**olMousePointerUpArrow**|10|指针是垂直箭头。|
