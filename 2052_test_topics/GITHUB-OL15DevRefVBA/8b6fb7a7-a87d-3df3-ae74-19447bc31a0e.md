
# Folder.PropertyAccessor 属性 (Outlook)

返回一个  **[PropertyAccessor](2fc91e13-703c-3ec9-9066-ffee7144306c.md)** 对象，该对象支持创建、获取、设置和删除父 **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** 对象的属性。只读。


## 语法

 _表达式_. **PropertyAccessor**

 _表达式_ 一个代表 **Folder** 对象的变量。
