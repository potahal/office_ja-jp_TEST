
# PlaySoundRuleAction.FilePath 属性 (Outlook)

返回或设置一个 **字符串** ，表示声音 (.wav) 文件的完整文件路径。读/写。


## 语法

 _表达式_. **FilePath**

 _表达式_ 一个代表 **PlaySoundRuleAction** 对象的变量。


## 说明

如果路径无效，则返回错误。
