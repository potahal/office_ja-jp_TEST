
# OlkComboBox.Paste 方法 （Outlook）

将剪贴板的内容粘贴到控件中。


## 语法

 _表达式_. **Paste**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

以无格式文本格式粘贴剪贴板的内容。

 **OlComboBoxStyleListBox** 控件的 **[样式](442acfef-795c-f41a-a19e-197a3b1ca12b.md)** 时，此方法 does 粘贴任何内容。
