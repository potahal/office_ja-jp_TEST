
# OlkOptionButton.Value 属性 (Outlook)

返回或设置一个 **variant 类型的值** ，该值表示控件的状态。读/写。


## 语法

 _表达式_. **Value**

 _表达式_ 一个代表 **OlkOptionButton** 对象的变量。


## 说明

有关选项按钮控件，值为 **True** 或 **False** ，表示是否已被选择按钮，或不。默认值为 **False** 。
