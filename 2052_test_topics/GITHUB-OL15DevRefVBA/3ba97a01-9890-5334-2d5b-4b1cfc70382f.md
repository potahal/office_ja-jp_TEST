
# SharingItem.Submitted 属性 (Outlook)

返回一个 **布尔** 值，如果已提交 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 为 **True** 。只读的。


## 语法

 _表达式_. **Submitted**

 _表达式_ 一个表示 **SharingItem** 对象的变量。


## 说明

邮件总是在文件夹（通常为 **"发件箱"**）中创建和提交。
