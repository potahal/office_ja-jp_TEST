
# PeopleView.LockUserChanges 属性 (Outlook)
返回或设置一个  **Boolean** 值，该值指示用户能否修改视图的设置。可读/写。

## 版本信息

添加的版本： Outlook 2013


## 语法

 _表达式_. **LockUserChanges**

 _表达式_ 一个表示 **PeopleView**对象的变量。


## 属性值

 **BOOL**
