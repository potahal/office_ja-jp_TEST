
# NavigationGroups.Class 属性 (Outlook)

返回一个指示对象的类的  **[OlObjectClass](33d724b3-df3c-2a7f-a80f-93b66d96f588.md)** 常量。只读。


## 语法

 _表达式_. **Class**

 _表达式_ 一个代表 **NavigationGroups** 对象的变量。


## 说明

此属性返回的 **[NavigationGroups](07206203-36a9-7467-3a89-24fa2a7c2b1f.md)** 对象的 **olNavigationGroups** 。
