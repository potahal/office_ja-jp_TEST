
# FormDescription.Hidden 属性 (Outlook)

返回一个 **布尔** 值 **True** 以使指定的自定义窗体被隐藏。 读/写。


## 语法

 _表达式_. **Hidden**

 _表达式_ 一个代表 **FormDescription** 对象的变量。


## 注解

自定义窗体不会显示在菜单或 **选择窗体**对话框中，并被指定为其他自定义窗体的响应窗体时才会使用。默认值为 **False** 。
