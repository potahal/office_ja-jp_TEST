
# TableView.GroupByFields 属性 (Outlook)

返回一个  **[OrderFields](e115fb80-352d-fd2e-c1c3-d266776fe122.md)** 对象，该对象代表一组作为 **[TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** 对象中所显示项目的分组依据的字段。只读。


## 语法

 _表达式_. **GroupByFields**

 _表达式_ 一个代表 **TableView** 对象的变量。
