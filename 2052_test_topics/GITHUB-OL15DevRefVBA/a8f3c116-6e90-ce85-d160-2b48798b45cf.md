
# Inspector.NewFormRegion 方法 （Outlook）

在新窗体区域的检查器中，在设计模式下打开新页面。


## 语法

 _表达式_. **NewFormRegion**

 _表达式_ 一个表示 **Inspector** 对象的变量。


### 返回值

代表在检查器中显示该窗体区域的页面 **Object** 。


## 说明

如果检查器不在设计模式中，  **NewFormRegion** 会将其置于设计模式。

此方法仅在设计模式下打开新窗体区域的页面。此方法不应当用于在运行时创建窗体区域。
