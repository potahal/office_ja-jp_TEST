
# TimelineView.Copy 方法 （Outlook）

基于现有的  **[TimelineView](fb14c1a1-f542-fa1e-f30f-c5ee3d2f0206.md)** 对象创建一个新的 **[View](41c8d149-9912-1685-4c8b-3c849cc6f1ed.md)** 对象。


## 语法

 _表达式_. **Copy**( ** _Name_**, ** _SaveOption_** )

 _表达式_ 一个代表 **TimelineView** 对象的变量。


### 参数



|**名称**|**必需/可选**|**数据类型**|**说明**|
|:-----|:-----|:-----|:-----|
| _Name_|必需|**String**|新视图的名称。|
| _SaveOption_|可选|**[OlViewSaveOption](c08bab4d-ecdd-a2ac-1cdc-fa910f9585e0.md)**|新视图的保存选项。|

### 返回值

表示新视图的 **视图** 对象。
