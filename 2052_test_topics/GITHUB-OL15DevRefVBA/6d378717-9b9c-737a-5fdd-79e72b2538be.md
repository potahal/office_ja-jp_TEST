
# OlkComboBox.HideSelection 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，它指定是显示还是隐藏控件在控件失去焦点时的选择。 读/写。


## 语法

 _表达式_. **HideSelection**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

此属性的默认值为 **True** 。
