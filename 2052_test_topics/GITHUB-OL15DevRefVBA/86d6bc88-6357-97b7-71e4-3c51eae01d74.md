
# Account.Parent 属性 (Outlook)

返回指定对象的 **对象** 的父级。只读的。


## 语法

 _表达式_. **Parent**

 _表达式_ 一个代表 **Account** 对象的变量。


## 说明

 **[Account](f624438c-4e45-2822-18b6-bfe8074a33c0.md)** 对象的父对象是 **[Accounts](2510b7d7-5062-8ea3-dda4-b544d2882a2b.md)** 对象。
