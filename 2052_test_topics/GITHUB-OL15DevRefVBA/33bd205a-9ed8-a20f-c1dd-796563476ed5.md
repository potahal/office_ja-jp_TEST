
# OlkListBox.MultiSelect 属性 (Outlook)

返回或设置一个  **[OlMultiSelect](33d61773-515f-e267-d60a-a3d91f3be9d7.md)** 常量，该常量指定在列表框控件中可以选择的项目的范围。可读/写。


## 语法

 _表达式_. **MultiSelect**

 _表达式_ 一个代表 **OlkListBox** 对象的变量。
