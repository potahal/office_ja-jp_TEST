
# TimeZone.ID 属性 (Outlook)

返回一个 **字符串** ，用于唯一标识时区。 只读的。


## 语法

 _表达式_. **ID**

 _表达式_ 一个代表 **TimeZone** 对象的变量。


## 说明

时区 **ID** 是全球对于该时区相同。 它是包含时区信息的 Windows 注册表项的名称。与 **[名称](e75bf2af-ae74-4d8f-7246-99a0daa17cb1.md)** 属性中，不同 **ID** 的值是未本地化。
