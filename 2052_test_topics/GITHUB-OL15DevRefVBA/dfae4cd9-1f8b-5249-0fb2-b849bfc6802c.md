
# OlkLabel.WordWrap 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，指定控件的内容是否自动换行的末尾。 读/写。


## 语法

 _表达式_. **WordWrap**

 _表达式_ 一个代表 **OlkLabel** 对象的变量。


## 说明

默认值为 **True** 。
