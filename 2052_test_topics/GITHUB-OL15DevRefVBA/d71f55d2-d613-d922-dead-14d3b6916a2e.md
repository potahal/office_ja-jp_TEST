
# SharingItem.IsMarkedAsTask 属性 (Outlook)

返回一个 **布尔** 值，该值指示是否将 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 标记为一项任务。只读的。


## 语法

 _表达式_. **IsMarkedAsTask**

 _表达式_ 一个返回 **SharingItem** 对象的表达式。
