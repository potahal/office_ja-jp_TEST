
# UserDefinedProperties.Refresh 方法 （Outlook）

刷新存储区中集合的内容，从而检索在远程浏览器中执行的所有更改。


## 语法

 _表达式_. **Refresh**

 _表达式_ 一个代表 **UserDefinedProperties** 对象的变量。
