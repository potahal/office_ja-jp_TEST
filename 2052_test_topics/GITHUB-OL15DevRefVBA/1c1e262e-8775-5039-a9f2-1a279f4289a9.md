
# CalendarModule.Name 属性 (Outlook)

返回一个 **字符串** 值，表示本地化的显示名称 **[CalendarModule](9203024d-9cef-75e0-600f-f3899e24761a.md)** 对象。只读的。


## 语法

 _表达式_. **Name**

 _表达式_ 一个代表 **CalendarModule** 对象的变量。
