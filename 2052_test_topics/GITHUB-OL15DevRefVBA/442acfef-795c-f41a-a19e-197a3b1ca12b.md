
# OlkComboBox.Style 属性 (Outlook)

返回或设置一个  **[OlComboBoxStyle](8aaeceb7-3928-5691-2f4b-9c27c88ca1c8.md)** 常量，用于指定用户可以选择或设置控件值的方式。可读/写。


## 语法

 _表达式_. **Style**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

默认值是 **olComboBoxStyleCombo** 。
