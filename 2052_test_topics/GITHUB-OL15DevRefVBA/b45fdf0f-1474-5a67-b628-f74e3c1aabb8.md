
# ContactItem.InternetFreeBusyAddress 属性 (Outlook)

返回或设置联系人的 **详细信息**选项卡上的 **地址**框中为对应的 **字符串** 。 读/写。


## 语法

 _表达式_. **InternetFreeBusyAddress**

 _表达式_ 一个代表 **ContactItem** 对象的变量。


## 注解

 **"详细信息"**选项卡上的 **"地址"**框可包含以"电子名片闲-忙"标准格式表示的用户忙/闲信息的 URL 位置。
