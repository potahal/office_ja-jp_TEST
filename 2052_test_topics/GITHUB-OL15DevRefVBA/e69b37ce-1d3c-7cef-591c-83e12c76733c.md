
# ContactItem.PropertyAccessor 属性 (Outlook)

返回一个 **[PropertyAccessor](2fc91e13-703c-3ec9-9066-ffee7144306c.md)** 对象，该对象支持创建、 获取、 设置，以及删除父 **创建 ContactItem** 对象的属性。只读的。


## 语法

 _表达式_. **PropertyAccessor**

 _表达式_ 一个表示 **ContactItem** 对象的变量。
