
# ReportItem.Actions 属性 (Outlook)

返回表示项的所有可用操作的 **[操作](b0903aa4-9b75-5311-d0a5-5ff4a5e29c79.md)** 集合。只读的。


## 语法

 _表达式_. **Actions**

 _表达式_ 一个代表 **ReportItem** 对象的变量。
