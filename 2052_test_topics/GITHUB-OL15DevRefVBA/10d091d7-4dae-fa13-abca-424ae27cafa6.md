
# OlkOptionButton.GroupName 属性 (Outlook)

返回或设置一个 **字符串** ，它标识一组互斥的选项按钮控件。 读/写。


## 语法

 _表达式_. **GroupName**

 _表达式_ 一个代表 **OlkOptionButton** 对象的变量。


## 说明

一次只能选择组中的一个成员。选择组中的新成员会自动删除以前选择的任何其他组成员。

默认值为空字符串。
