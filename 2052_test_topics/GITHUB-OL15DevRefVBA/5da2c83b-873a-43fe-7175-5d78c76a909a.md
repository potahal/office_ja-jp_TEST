
# CalendarSharing.Folder 属性 (Outlook)

返回包含了将由  **[CalendarSharing](37a8a15e-51c2-b1a0-7db6-cf2a1f4e8405.md)** 对象共享的日历项目的 **[Folder](3cf6cda8-6d70-666e-2643-9d9c5b9cacfc.md)** 。只读。


## 语法

 _表达式_. **Folder**

 _表达式_ 一个返回 **CalendarSharing** 对象的表达式。


### 返回值

该 **文件夹** 包含日历项目共享。
