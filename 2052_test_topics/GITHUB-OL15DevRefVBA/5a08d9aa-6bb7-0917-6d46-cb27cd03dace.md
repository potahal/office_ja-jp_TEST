
# Folder.DefaultItemType 属性 (Outlook)

从  **[OlItemType](d42959b8-6c91-4d9e-98db-7226780f9995.md)** 枚举中返回一个指示文件夹中包含的默认 Outlook 项目类型的常量。只读。


## 语法

 _表达式_. **DefaultItemType**

 _表达式_ 一个代表 **Folder** 对象的变量。
