
# OlkComboBox.SelectionMargin 属性 (Outlook)

返回或设置一个 **boolean 类型的值** ，指定用户是否可以通过在文本左侧的区域单击来选择一行文本。 读/写。


## 语法

 _表达式_. **SelectionMargin**

 _表达式_ 一个代表 **OlkComboBox** 对象的变量。


## 说明

如果 **True** ，在页边单击会选择文本。 如果是 **False** ，此操作不会选择任何文本。默认值为 **False** 。
