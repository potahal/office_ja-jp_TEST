
# SharingItem.ConversationTopic 属性 (Outlook)

返回一个 **字符串** ，表示 **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** 话题的主题。只读的。


## 语法

 _表达式_. **ConversationTopic**

 _表达式_ 一个代表 **SharingItem** 对象的变量。
