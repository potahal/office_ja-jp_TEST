
# AppointmentItem.OutlookVersion 属性 (Outlook)

返回一个 **字符串** ，表示 Outlook 项的 Outlook 应用程序主版本号和次版本号。只读的。


## 语法

 _表达式_. **OutlookVersion**

 _表达式_ 一个代表 **AppointmentItem** 对象的变量。
