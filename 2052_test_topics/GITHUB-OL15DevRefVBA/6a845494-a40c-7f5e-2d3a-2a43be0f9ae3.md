
# StorageItem.Body 属性 (Outlook)

返回或设置一个 **字符串** ，表示纯文本正文的 Outlook 项目。读/写。


## 语法

 _表达式_. **Body**

 _表达式_ 一个代表 **StorageItem** 对象的变量。


## 说明

 **[StorageItem](41776bc3-b838-2755-fd6b-3b5012fb9ae5.md)** 对象只支持文本格式的项目正文。它们不支持 HTML 或 RTF 正文流。
