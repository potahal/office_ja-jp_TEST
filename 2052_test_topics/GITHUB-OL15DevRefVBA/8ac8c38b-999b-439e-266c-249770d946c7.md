
# CardView.BodyFont 属性 (Outlook)

返回一个  **[ViewFont](cbd7c6ce-f49a-1627-0ad9-a019911fb47b.md)** 对象，该对象代表在 **[CardView](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** 对象中显示每张卡片的正文时使用的字体。只读。


## 语法

 _表达式_. **BodyFont**

 _表达式_ 一个代表 **CardView** 对象的变量。
