
# TaskItem.ResponseState 属性 (Outlook)

返回一个  **[OlTaskResponse](7616cbdc-fc9c-abbe-fd07-ebdadc13ede2.md)** 常量，该常量指示对指定任务要求的响应的总体状态。只读。


## 语法

 _表达式_. **ResponseState**

 _表达式_ 一个代表 **TaskItem** 对象的变量。
