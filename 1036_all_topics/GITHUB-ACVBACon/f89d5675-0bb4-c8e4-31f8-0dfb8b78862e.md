
# Vous avez essayé d'affecter la valeur Null à une variable qui n'est pas du type de données Variant. (Erreur 3162)

Vous avez essayé d'attribuer une valeur  **Null** à une variable dont le type de données n'est pas **Variant**. À l'aide de l'instruction **Dim**, déclarez la variable en tant que **Variant**, puis tentez à nouveau l'opération.
