
# ODBC - L'appel a échoué. (Erreur 3146)

Au moyen d'une connexion ODBC, vous avez essayé d'effectuer une opération sur les données d'une source de données ODBC. Cette erreur peut survenir lorsque la source de données ODBC réside sur un lecteur réseau et que vous n'êtes pas connecté au réseau. Assurez-vous que le réseau est disponible, puis tentez à nouveau l'opération.
